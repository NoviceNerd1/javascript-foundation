# Quantifiers: +, \*, ? and {n} Summary

Quantifiers specify how many times a character, class, or group must appear in a match.

## Quantity `{n}`

| Syntax  | Meaning           |
| ------- | ----------------- |
| `{n}`   | Exactly n times   |
| `{n,m}` | From n to m times |
| `{n,}`  | n or more times   |

```javascript
// Exactly 5 digits
console.log("I'm 12345 years old".match(/\d{5}/)); // ['12345']

// 3 to 5 digits
console.log("I'm 1234 years old".match(/\d{3,5}/)); // ['1234']

// 3 or more digits
console.log("I'm 345678 years old".match(/\d{3,}/)); // ['345678']

// Find all numbers from phone
let str = "+7(903)-123-45-67";
console.log(str.match(/\d{1,}/g)); // ['7', '903', '123', '45', '67']
```

## Shorthand Quantifiers

| Symbol | Meaning      | Equivalent |
| ------ | ------------ | ---------- |
| `+`    | One or more  | `{1,}`     |
| `?`    | Zero or one  | `{0,1}`    |
| `*`    | Zero or more | `{0,}`     |

### `+` (One or More)

```javascript
let str = "+7(903)-123-45-67";
console.log(str.match(/\d+/g)); // ['7', '903', '123', '45', '67']
```

### `?` (Zero or One)

```javascript
// Matches both "color" and "colour"
console.log("color colour".match(/colou?r/g)); // ['color', 'colour']

// Optional character
console.log("abc ac".match(/ab?c/g)); // ['abc', 'ac']
```

### `*` (Zero or More)

```javascript
// Digit followed by any number of zeros
console.log("100 10 1".match(/\d0*/g)); // ['100', '10', '1']

// Compare with + (requires at least one zero)
console.log("100 10 1".match(/\d0+/g)); // ['100', '10']
```

## Practical Examples

### Decimal Fractions

```javascript
let str = "0 1 12.345 7890";
console.log(str.match(/\d+\.\d+/g)); // ['12.345']
```

### HTML Tags (Simple)

```javascript
// Opening tag
console.log("<body>...</body>".match(/<[a-z]+>/gi)); // ['<body>']

// With digits allowed after first char (like <h1>)
console.log("<h1>Hi!</h1>".match(/<[a-z][a-z0-9]*>/gi)); // ['<h1>']

// Both opening and closing
console.log("<h1>Hi!</h1>".match(/<\/?[a-z][a-z0-9]*>/gi)); // ['<h1>', '</h1>']
```

### Find All Words

```javascript
let text = "Hello, world! How are you?";
console.log(text.match(/[a-zA-Z]+/g));
// ['Hello', 'world', 'How', 'are', 'you']
```

### Validate Phone Number Format

```javascript
function isPhoneNumber(str) {
  return /^\+\d{1,3}\(\d{3}\)\d{3}-\d{2}-\d{2}$/.test(str);
}

console.log(isPhoneNumber("+7(903)-123-45-67")); // true
console.log(isPhoneNumber("+7(903)-123-45-6")); // false
```

## Common Patterns

| Pattern     | Matches                               |
| ----------- | ------------------------------------- |
| `\d+`       | One or more digits                    |
| `\w+`       | One or more word characters           |
| `\s*`       | Any number of spaces (including none) |
| `colou?r`   | color or colour                       |
| `https?://` | http:// or https://                   |
| `<[^>]+>`   | HTML tag (simplified)                 |

## Greedy vs Lazy (Brief Note)

Quantifiers are **greedy** by default—they match as many characters as possible:

```javascript
// Greedy: matches "12345" (as many digits as possible)
console.log("12345".match(/\d{2,4}/)); // ['1234']

// To make lazy, add '?'
console.log("12345".match(/\d{2,4}?/)); // ['12']
```

## Tasks

### Task 1: Find Ellipsis "..."

```javascript
let regexp = /\.{3,}/g;
let str = "Hello!... How goes?.....";
console.log(str.match(regexp)); // ['...', '.....']
```

### Task 2: HTML Colors (#ABCDEF)

```javascript
let regexp = /#[0-9A-Fa-f]{6}\b/g;
let str =
  "color:#121212; background-color:#AA00ef bad-colors:f#fddee #fd2 #12345678";
console.log(str.match(regexp)); // ['#121212', '#AA00ef']
```

## Summary Table

| Quantifier | Meaning   | Example   | Matches           |
| ---------- | --------- | --------- | ----------------- |
| `{n}`      | Exactly n | `\d{3}`   | 3 digits          |
| `{n,m}`    | n to m    | `\d{2,4}` | 2, 3, or 4 digits |
| `{n,}`     | n or more | `\d{3,}`  | 3+ digits         |
| `+`        | 1 or more | `\d+`     | 1+ digits         |
| `?`        | 0 or 1    | `colou?r` | color, colour     |
| `*`        | 0 or more | `\d0*`    | digit + any zeros |

**Key Points:**

- Quantifiers specify repetition count
- `+`, `?`, `*` are shorthands for common cases
- Quantifiers apply to preceding character/class/group
- Use `\b` boundaries to prevent matching inside longer numbers
- Quantifiers are greedy by default (match as many as possible)
