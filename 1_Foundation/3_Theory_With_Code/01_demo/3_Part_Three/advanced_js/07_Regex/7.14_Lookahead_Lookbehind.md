# Lookahead and Lookbehind Summary

Lookahead and lookbehind (lookaround) allow matching patterns only if followed or preceded by another pattern, without including that pattern in the match.

## Lookahead

### Positive Lookahead: `X(?=Y)`

Matches X only if followed by Y (Y not included in result).

```javascript
let str = "1 turkey costs 30€";
console.log(str.match(/\d+(?=€)/)); // ["30"] (€ not included)
```

### Negative Lookahead: `X(?!Y)`

Matches X only if **not** followed by Y.

```javascript
let str = "2 turkeys cost 60€";
console.log(str.match(/\d+(?!€)/g)); // ["2"] (price skipped)
```

## Lookbehind

⚠️ **Browser compatibility**: Lookbehind not supported in Safari, Internet Explorer (works in Chrome, Firefox, Edge).

### Positive Lookbehind: `(?<=Y)X`

Matches X only if preceded by Y.

```javascript
let str = "1 turkey costs $30";
console.log(str.match(/(?<=\$)\d+/)); // ["30"]
```

### Negative Lookbehind: `(?<!Y)X`

Matches X only if **not** preceded by Y.

```javascript
let str = "2 turkeys cost $60";
console.log(str.match(/(?<!\$)\b\d+/g)); // ["2"]
```

## Capturing in Lookaround

Wrap parts in extra parentheses to capture them:

```javascript
// Capture currency symbol with lookahead
let str = "1 turkey costs 30€";
let regexp = /\d+(?=(€|kr))/;
console.log(str.match(regexp)); // ["30", "€"]

// Capture currency symbol with lookbehind
let str2 = "1 turkey costs $30";
let regexp2 = /(?<=(\$|£))\d+/;
console.log(str2.match(regexp2)); // ["30", "$"]
```

## Practical Examples

### Find Price with Currency

```javascript
let str = "Item: $19.99, €24.50, £15.00";
let prices = str.match(/(?<=\$)\d+\.?\d*|(?<=€)\d+\.?\d*|(?<=£)\d+\.?\d*/g);
console.log(prices); // ["19.99", "24.50", "15.00"]
```

### Find Words Before Colon

```javascript
let str = "name: John, age: 30, city: London";
let labels = str.match(/\b\w+(?=:)/g);
console.log(labels); // ["name", "age", "city"]
```

### Find Numbers Not Followed by %

```javascript
let str = "Discount: 20% off, price: $100";
let numbers = str.match(/\d+(?!%)/g);
console.log(numbers); // ["100"] (skips "20")
```

### Validate Number Not Preceded by Minus

```javascript
let str = "10 -5 20 -3 30";
let positives = str.match(/(?<!-)\b\d+/g);
console.log(positives); // ["10", "20", "30"]
```

## Complex Example: Email Validation with Lookaround

```javascript
function isValidEmail(email) {
  // Must have @, must not have spaces before @
  let regexp = /^(?![ ])[^\s@]+@[^\s@]+\.[^\s@]+$/;
  return regexp.test(email);
}

console.log(isValidEmail("user@example.com")); // true
console.log(isValidEmail(" user@example.com")); // false
```

## Lookaround Reference

| Pattern   | Type                | Description            |
| --------- | ------------------- | ---------------------- |
| `X(?=Y)`  | Positive lookahead  | X if followed by Y     |
| `X(?!Y)`  | Negative lookahead  | X if not followed by Y |
| `(?<=Y)X` | Positive lookbehind | X if preceded by Y     |
| `(?<!Y)X` | Negative lookbehind | X if not preceded by Y |

## Tasks

### Task 1: Find Non-Negative Integers

```javascript
let regexp = /(?<![-\d])\b\d+\b/g;
let str = "0 12 -5 123 -18";
console.log(str.match(regexp)); // ["0", "12", "123"]
```

**Alternative:** `/\b\d+\b(?![-\d])/g` also works.

### Task 2: Insert After `<body>` Tag

```javascript
let regexp = /(?<=<body.*?>)/;
let str = `
<html>
  <body style="height: 200px">
  ...
  </body>
</html>
`;

str = str.replace(regexp, `<h1>Hello</h1>`);
console.log(str);
// <body style="height: 200px"><h1>Hello</h1>
```

## Summary

| Feature        | Purpose                                      |
| -------------- | -------------------------------------------- |
| **Lookahead**  | Check what follows                           |
| **Lookbehind** | Check what precedes                          |
| **Positive**   | Must exist                                   |
| **Negative**   | Must not exist                               |
| **Zero-width** | Tests only, doesn't consume characters       |
| **Capturing**  | Add extra `()` to capture lookaround content |

**Key Points:**

- Lookaround tests conditions without including them in the match
- Positive = `(?=Y)` / `(?<=Y)`
- Negative = `(?!Y)` / `(?<!Y)`
- Useful for validation and context-based matching
- Lookbehind has limited browser support (not in Safari/IE)
- Can capture by adding extra parentheses around the lookaround content
