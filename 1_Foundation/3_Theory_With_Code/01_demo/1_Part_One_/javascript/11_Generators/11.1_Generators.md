# Generators

## Overview

Generators are functions that can return ("yield") multiple values, one after another, on demand. They work great with iterables.

## Generator Functions

Define a generator with `function*`:

```javascript
function* generateSequence() {
  yield 1;
  yield 2;
  return 3;
}

let generator = generateSequence();
alert(generator); // [object Generator]
```

## The `next()` Method

Calling `.next()` runs execution until the nearest `yield`:

```javascript
let generator = generateSequence();

let one = generator.next();
alert(JSON.stringify(one)); // {value: 1, done: false}

let two = generator.next();
alert(JSON.stringify(two)); // {value: 2, done: false}

let three = generator.next();
alert(JSON.stringify(three)); // {value: 3, done: true}
```

## Generators Are Iterable

```javascript
function* generateSequence() {
  yield 1;
  yield 2;
  yield 3; // Use yield, not return, for for..of
}

for (let value of generateSequence()) {
  alert(value); // 1, 2, 3
}

// Spread syntax works too
let sequence = [0, ...generateSequence()];
alert(sequence); // 0, 1, 2, 3
```

## Using Generators for Iterables

Compact implementation of iterable objects:

```javascript
let range = {
  from: 1,
  to: 5,

  *[Symbol.iterator]() {
    for (let value = this.from; value <= this.to; value++) {
      yield value;
    }
  },
};

alert([...range]); // 1, 2, 3, 4, 5
```

## Generator Composition (`yield*`)

Embed one generator into another:

```javascript
function* generateSequence(start, end) {
  for (let i = start; i <= end; i++) yield i;
}

function* generatePasswordCodes() {
  yield* generateSequence(48, 57); // 0-9
  yield* generateSequence(65, 90); // A-Z
  yield* generateSequence(97, 122); // a-z
}

let str = "";
for (let code of generatePasswordCodes()) {
  str += String.fromCharCode(code);
}
alert(str); // 0-9A-Za-z
```

## Two-Way Communication with `yield`

`yield` can receive values from the outside via `next(arg)`:

```javascript
function* gen() {
  let result = yield "2 + 2 = ?"; // Waits for answer
  alert(result); // 4
}

let generator = gen();
let question = generator.next().value; // "2 + 2 = ?"
generator.next(4); // Pass answer back
```

### Multiple Exchanges

```javascript
function* gen() {
  let ask1 = yield "2 + 2 = ?";
  alert(ask1); // 4

  let ask2 = yield "3 * 3 = ?";
  alert(ask2); // 9
}

let generator = gen();
alert(generator.next().value); // "2 + 2 = ?"
alert(generator.next(4).value); // "3 * 3 = ?"
alert(generator.next(9).done); // true
```

## Error Handling with `generator.throw()`

```javascript
function* gen() {
  try {
    let result = yield "2 + 2 = ?";
  } catch (e) {
    alert(e); // Error: Answer not found
  }
}

let generator = gen();
generator.next();
generator.throw(new Error("Answer not found"));
```

## `generator.return()`

```javascript
function* gen() {
  yield 1;
  yield 2;
  yield 3;
}

let g = gen();
g.next(); // {value: 1, done: false}
g.return("foo"); // {value: "foo", done: true}
g.next(); // {value: undefined, done: true}
```

## Summary

| Feature              | Description                                |
| -------------------- | ------------------------------------------ |
| `function*`          | Declares a generator function              |
| `yield`              | Pauses execution and returns a value       |
| `next()`             | Resumes execution, returns `{value, done}` |
| `yield*`             | Delegates to another generator             |
| `generator.throw()`  | Throws error into generator                |
| `generator.return()` | Finishes generator with value              |

**Key Points:**

- Generators produce values lazily, on demand
- Generators are iterable (works with `for..of`, spread `...`)
- `yield` is two-way – can both send and receive values
- Use `yield*` to compose generators
- Great for creating custom iterables and handling streams

## Task: Pseudo-Random Generator

```javascript
function* pseudoRandom(seed) {
  let value = seed;
  while (true) {
    value = (value * 16807) % 2147483647;
    yield value;
  }
}

let generator = pseudoRandom(1);
alert(generator.next().value); // 16807
alert(generator.next().value); // 282475249
alert(generator.next().value); // 1622650073
```
