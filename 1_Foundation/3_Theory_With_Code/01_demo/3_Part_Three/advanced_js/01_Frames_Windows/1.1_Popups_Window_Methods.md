# Popup Windows & Window Methods Summary

Popup windows are an ancient browser feature for displaying additional content in separate windows. While mostly replaced by modern alternatives like dynamic `<div>` content, they're still used for OAuth authorization due to security benefits.

## Popup Blocking

Modern browsers block popups unless triggered by user actions:

```javascript
// Blocked - not in user event
window.open("https://example.com");

// Allowed - user click triggers
button.onclick = () => {
  window.open("https://example.com");
};
```

## window.open() Syntax

```javascript
let popup = window.open(url, name, params);
```

### Parameters Configuration

```javascript
let params = "width=600,height=300,left=100,top=100,menubar=no,toolbar=no";
let popup = window.open("/page", "windowName", params);
```

**Common params settings:**

- `left/top` - window position on screen
- `width/height` - window dimensions
- `menubar/toolbar/location/status` - show/hide browser UI elements
- `resizable/scrollbars` - enable/disable resizing

## Working with Popups

### Access and Control

```javascript
let popup = window.open("about:blank", "example", "width=300,height=300");

// Modify content after load
popup.onload = function () {
  popup.document.body.innerHTML = "<h1>Hello!</h1>";
};

// Check if closed
popup.closed; // true/false

// Close popup
popup.close();
```

### Bidirectional Access

```javascript
// Popup accessing main window
popup.document.write(`
  <script>window.opener.document.body.style.background = 'red';<\/script>
`);
```

## Window Methods

### Move & Resize (popups only)

```javascript
popup.moveBy(100, 50); // move relative
popup.moveTo(200, 200); // move to absolute position
popup.resizeBy(50, 50); // resize relative
popup.resizeTo(400, 400); // resize to absolute dimensions
```

### Scrolling

```javascript
popup.scrollBy(0, 100); // scroll down 100px
popup.scrollTo(0, 0); // scroll to top
element.scrollIntoView(); // scroll to make element visible
```

### Focus Events

```javascript
popup.focus(); // bring to foreground

window.onfocus = () => console.log("Window active");
window.onblur = () => console.log("Window lost focus");
```

## Key Limitations

- **Same Origin Policy**: Windows can only freely access each other if same protocol://domain:port
- **Mobile devices**: Popups work poorly or not at all
- **Move/resize methods**: Only work reliably on popups opened by your code
- **Focus methods**: Heavily restricted by browsers to prevent abuse
