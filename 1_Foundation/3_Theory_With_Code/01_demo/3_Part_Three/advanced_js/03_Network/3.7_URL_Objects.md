# URL Objects Summary

The built-in `URL` class provides convenient interface for creating and parsing URLs with automatic encoding.

## Creating URL Objects

```javascript
// Full URL
let url = new URL("https://javascript.info/profile/admin");

// Relative URL with base
let url2 = new URL("/profile/admin", "https://javascript.info");

// From existing URL
let url3 = new URL("tester", url); // https://javascript.info/profile/tester
```

## URL Components

```javascript
let url = new URL("https://javascript.info:8080/path/page?q=value#hash");

console.log(url.protocol); // https:
console.log(url.host); // javascript.info:8080
console.log(url.hostname); // javascript.info
console.log(url.port); // 8080
console.log(url.pathname); // /path/page
console.log(url.search); // ?q=value
console.log(url.hash); // #hash
console.log(url.href); // full URL
```

## URLSearchParams

Manages query parameters with automatic encoding.

```javascript
let url = new URL("https://google.com/search");

// Add/Set parameters
url.searchParams.set("q", "test me!");
url.searchParams.append("lang", "en");
url.searchParams.set("tbs", "qdr:y");

console.log(url.toString());
// https://google.com/search?q=test+me%21&lang=en&tbs=qdr%3Ay

// Get parameters
console.log(url.searchParams.get("q")); // test me!
console.log(url.searchParams.getAll("lang")); // ['en']

// Check existence
console.log(url.searchParams.has("q")); // true

// Delete
url.searchParams.delete("lang");

// Iterate
for (let [name, value] of url.searchParams) {
  console.log(`${name}=${value}`);
}
// q=test me!
// tbs=qdr:y
```

### URLSearchParams Methods

| Method                | Description             |
| --------------------- | ----------------------- |
| `append(name, value)` | Add parameter           |
| `delete(name)`        | Remove parameter        |
| `get(name)`           | Get first value         |
| `getAll(name)`        | Get all values          |
| `has(name)`           | Check existence         |
| `set(name, value)`    | Set/replace parameter   |
| `sort()`              | Sort parameters by name |

## URL with Network Requests

```javascript
let url = new URL(
  "https://api.github.com/repos/javascript-tutorial/en.javascript.info/commits",
);
url.searchParams.set("per_page", "100");

// Use URL object directly in fetch
let response = await fetch(url);
let commits = await response.json();
```

## Automatic Encoding

URL objects automatically encode special characters.

```javascript
// Cyrillic characters
let url = new URL("https://ru.wikipedia.org/wiki/Тест");
url.searchParams.set("key", "ъ");

console.log(url);
// https://ru.wikipedia.org/wiki/%D0%A2%D0%B5%D1%81%D1%82?key=%D1%8A
```

## Manual Encoding Functions

When using strings instead of URL objects:

| Function               | Purpose                                                      |
| ---------------------- | ------------------------------------------------------------ |
| `encodeURI()`          | Encode entire URL (preserves `:`, `?`, `=`, `&`, `#`)        |
| `decodeURI()`          | Decode entire URL                                            |
| `encodeURIComponent()` | Encode URL component (encodes `:`, `?`, `=`, `&`, `#`, etc.) |
| `decodeURIComponent()` | Decode URL component                                         |

### When to Use Which

```javascript
// encodeURI for full URL
let url = encodeURI("http://site.com/привет");
console.log(url); // http://site.com/%D0%BF%D1%80%D0%B8%D0%B2%D0%B5%D1%82

// encodeURIComponent for parameters
let music = encodeURIComponent("Rock&Roll");
let searchUrl = `https://google.com/search?q=${music}`;
console.log(searchUrl); // https://google.com/search?q=Rock%26Roll

// Wrong - & breaks parameter
let badMusic = encodeURI("Rock&Roll");
let badUrl = `https://google.com/search?q=${badMusic}`;
console.log(badUrl); // https://google.com/search?q=Rock&Roll (incorrect!)
```

## Building URLs with Parameters

### Using URL Object (Recommended)

```javascript
function buildUrl(base, params) {
  let url = new URL(base);
  for (let [key, value] of Object.entries(params)) {
    url.searchParams.set(key, value);
  }
  return url;
}

let url = buildUrl("https://api.example.com/search", {
  q: "javascript tutorial",
  page: "1",
  sort: "desc",
});

console.log(url);
// https://api.example.com/search?q=javascript+tutorial&page=1&sort=desc
```

### Using String + encodeURIComponent

```javascript
function buildUrlString(base, params) {
  let query = Object.entries(params)
    .map(
      ([key, value]) =>
        `${encodeURIComponent(key)}=${encodeURIComponent(value)}`,
    )
    .join("&");

  return `${base}?${query}`;
}

let url = buildUrlString("https://api.example.com/search", {
  q: "javascript tutorial",
  page: "1",
  sort: "desc",
});
```

## Parsing URL Parameters

```javascript
function parseParams(urlString) {
  let url = new URL(urlString);
  let params = {};

  for (let [key, value] of url.searchParams) {
    params[key] = value;
  }

  return params;
}

let params = parseParams("https://example.com?name=John&age=25&city=NYC");
console.log(params); // { name: 'John', age: '25', city: 'NYC' }
```

## Working with Relative URLs

```javascript
let base = new URL("https://javascript.info/guide");
let guides = [
  new URL("html", base), // https://javascript.info/guide/html
  new URL("css", base), // https://javascript.info/guide/css
  new URL("js", base), // https://javascript.info/guide/js
];
```

## URL vs Encoding Functions

| Feature            | URL Object          | encodeURIComponent        |
| ------------------ | ------------------- | ------------------------- |
| Automatic encoding | ✅ Yes              | ❌ Manual                 |
| Parameter handling | ✅ searchParams API | ❌ Manual string building |
| IPv6 support       | ✅ Correct          | ❌ May encode brackets    |
| Readability        | ✅ High             | ⚠️ Moderate               |

## Summary

- **URL objects** handle automatic encoding and provide easy parameter management
- Use `url.searchParams` for query string operations
- **encodeURIComponent()** for encoding individual parameter values when building strings
- **encodeURI()** for encoding full URLs (preserves URL structure characters)
- URL objects can be passed directly to `fetch()` and other network methods
- The main advantage is automatic encoding and built-in parameter management
