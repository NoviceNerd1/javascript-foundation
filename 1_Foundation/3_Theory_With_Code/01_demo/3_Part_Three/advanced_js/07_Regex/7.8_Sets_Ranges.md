# Sets and Ranges `[...]` Summary

Square brackets `[...]` define a set of characters to match. They match exactly **one** character from the set.

## Basic Sets

```javascript
// Match 'a', 'e', or 'o'
console.log("Mop top".match(/[tm]op/gi)); // ['Mop', 'top']

// Match 'V' + [o or i] + 'la'
console.log("Voila".match(/V[oi]la/)); // null (would match "Vola" or "Vila")
```

## Ranges

```javascript
// Hexadecimal digits
console.log("Exception 0xAF".match(/x[0-9A-F][0-9A-F]/g)); // ['xAF']

// Lowercase letters a-z
console.log("abc123".match(/[a-z]+/g)); // ['abc']

// Digits 0-5
console.log("0123456789".match(/[0-5]/g)); // ['0','1','2','3','4','5']
```

## Combining Ranges and Sets

```javascript
// Letters and digits
console.log("Hello 123".match(/[a-zA-Z0-9]+/g)); // ['Hello', '123']

// Word characters + hyphen
console.log("hello-world_123".match(/[\w-]+/g)); // ['hello-world_123']

// Space or digit
console.log("a 1 b 2".match(/[\s\d]/g)); // [' ', '1', ' ', '2']
```

## Character Class Shorthands

| Shorthand | Equivalent Set  |
| --------- | --------------- |
| `\d`      | `[0-9]`         |
| `\w`      | `[a-zA-Z0-9_]`  |
| `\s`      | `[\t\n\v\f\r ]` |

## Negated Sets `[^...]`

Caret `^` at the start excludes characters:

```javascript
// Any character except a, e, y, o
console.log("hello".match(/[^aeyo]/g)); // ['h', 'l', 'l']

// Any non-digit (same as \D)
console.log("a1b2".match(/[^0-9]/g)); // ['a', 'b']

// Any non-space (same as \S)
console.log("a b".match(/[^\s]/g)); // ['a', 'b']
```

### Find Non-Alphanumeric

```javascript
let email = "alice15@gmail.com";
console.log(email.match(/[^\d\sA-Z]/gi)); // ['@', '.']
```

## Escaping in Brackets

Most special characters don't need escaping inside `[...]`:

| Character | Inside `[...]`           | Example                        |
| --------- | ------------------------ | ------------------------------ |
| `. + ( )` | No escape needed         | `[.,]` matches `.` or `,`      |
| `-`       | Only at start/end        | `[-a-z]` or `[a-z-]`           |
| `^`       | Only at start (negation) | `[^abc]` or `[\^]` for literal |
| `]`       | Always escape            | `[\[\]]` matches `[` or `]`    |

```javascript
// No escaping needed
console.log("1 + 2 - 3".match(/[-().^+]/g)); // ['+', '-']

// Escaping works too
console.log("1 + 2 - 3".match(/[\-\(\)\.\^\+]/g)); // ['+', '-']
```

## Unicode and Flag `u`

Without `u` flag, surrogate pairs break:

```javascript
// ❌ Without 'u' - broken
console.log("𝒳".match(/[𝒳𝒴]/)); // Returns half-character

// ✅ With 'u' - correct
console.log("𝒳".match(/[𝒳𝒴]/u)); // ['𝒳']

// Range with surrogate pairs requires 'u'
console.log("𝒴".match(/[𝒳-𝒵]/u)); // ['𝒴']
```

## Multi-Language Word Characters

`\w` only works with Latin. For Unicode letters:

```javascript
// Unicode property set for any word character
let regexp = /[\p{Alpha}\p{M}\p{Nd}\p{Pc}\p{Join_C}]/gu;

let str = `Hi 你好 12`;
console.log(str.match(regexp)); // ['H','i','你','好','1','2']
```

## Practical Examples

### Validate Hexadecimal

```javascript
let hex = /^[0-9A-Fa-f]+$/;
console.log(hex.test("1A3F")); // true
console.log(hex.test("1G3F")); // false
```

### Find Vowels

```javascript
let text = "Hello World";
let vowels = text.match(/[aeiou]/gi);
console.log(vowels); // ['e', 'o', 'o']
```

### Match Punctuation

```javascript
let text = "Hello! How are you?";
let punct = text.match(/[!?.]/g);
console.log(punct); // ['!', '?']
```

### Validate Username

```javascript
function isValidUsername(name) {
  return /^[a-zA-Z0-9_]{3,16}$/.test(name);
}

console.log(isValidUsername("john_123")); // true
console.log(isValidUsername("john@123")); // false
```

### Task 2: Find Time (hh:mm or hh-mm)

```javascript
let regexp = /\d\d[-:]\d\d/g;
let str = "Breakfast at 09:00. Dinner at 21-30";
console.log(str.match(regexp)); // ['09:00', '21-30']
```

## Summary

| Syntax   | Description              | Example      |
| -------- | ------------------------ | ------------ |
| `[abc]`  | Any of a, b, or c        | `/[aeiou]/`  |
| `[a-z]`  | Range from a to z        | `/[0-9A-F]/` |
| `[^abc]` | Any except a, b, c       | `/[^0-9]/`   |
| `[a-z-]` | Hyphen at end is literal | `/[a-z-]/`   |
| `[\w-]`  | Word char or hyphen      | `/[\w-]+/`   |

**Key Points:**

- `[...]` matches exactly one character from the set
- Use `-` for ranges (escape or place at start/end for literal)
- Use `^` at start for negation (escape for literal)
- Most special characters don't need escaping inside brackets
- Use `u` flag for Unicode surrogate pairs
- `\w` is Latin-only; use `\p{Alpha}` with `u` for Unicode letters
