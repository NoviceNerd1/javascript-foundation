# JSON Methods, toJSON

## Overview

JSON (JavaScript Object Notation) is a lightweight data format for exchanging data between systems. JavaScript provides two methods for JSON handling.

## JSON.stringify – Convert to JSON

```javascript
let student = {
  name: "John",
  age: 30,
  isAdmin: false,
  courses: ["html", "css", "js"],
  spouse: null,
};

let json = JSON.stringify(student);
alert(typeof json); // string

/* JSON output:
{
  "name": "John",
  "age": 30,
  "isAdmin": false,
  "courses": ["html", "css", "js"],
  "spouse": null
}
*/
```

### JSON Format Rules

- Strings use **double quotes** only
- Object property names are **double-quoted**
- No functions, symbols, or `undefined` (they're skipped)

```javascript
JSON.stringify(1); // "1"
JSON.stringify("test"); // "\"test\""
JSON.stringify(true); // "true"
JSON.stringify([1, 2, 3]); // "[1,2,3]"
```

### What Gets Skipped

```javascript
let user = {
  sayHi() {}, // ignored (method)
  [Symbol("id")]: 123, // ignored (symbol)
  something: undefined, // ignored
};
JSON.stringify(user); // "{}"
```

### Circular References Cause Error

```javascript
let room = { number: 23 };
let meetup = { title: "Conference", place: room };
room.occupiedBy = meetup; // Circular reference

JSON.stringify(meetup); // Error: Converting circular structure to JSON
```

## JSON.stringify Parameters

```javascript
JSON.stringify(value[, replacer, space])
```

### Replacer (Array) – Include Only Specified Properties

```javascript
let meetup = {
  title: "Conference",
  participants: [{ name: "John" }, { name: "Alice" }],
  place: { number: 23 },
};

JSON.stringify(meetup, ["title", "participants"]);
// {"title":"Conference","participants":[{},{}]}  (name missing)
```

### Replacer (Function) – Custom Transformation

```javascript
let room = { number: 23 };
let meetup = {
  title: "Conference",
  place: room,
};
room.occupiedBy = meetup; // Circular reference

let json = JSON.stringify(meetup, function replacer(key, value) {
  return key == "occupiedBy" ? undefined : value;
});
// Works! occupiedBy excluded
```

### Space – Pretty Formatting

```javascript
let user = { name: "John", age: 25, roles: { isAdmin: false } };

JSON.stringify(user, null, 2);
/* Output with 2-space indents:
{
  "name": "John",
  "age": 25,
  "roles": {
    "isAdmin": false
  }
}
*/
```

## Custom toJSON Method

If an object has `toJSON`, it's called automatically during serialization.

```javascript
let room = {
  number: 23,
  toJSON() {
    return this.number; // Return what to serialize
  },
};

JSON.stringify(room); // "23"

let meetup = { title: "Conference", room };
JSON.stringify(meetup);
// {"title":"Conference","room":23}
```

## JSON.parse – Parse JSON to Object

```javascript
let numbers = "[0, 1, 2, 3]";
numbers = JSON.parse(numbers);
alert(numbers[1]); // 1

let userData = '{ "name": "John", "age": 35, "friends": [0,1,2,3] }';
let user = JSON.parse(userData);
alert(user.friends[1]); // 1
```

### Common JSON Mistakes

```javascript
let json = `{
  name: "John",           // Error: key must be quoted
  "surname": 'Smith',     // Error: single quotes not allowed
  'isAdmin': false,       // Error: key quoted with single quotes
  "birthday": new Date(2000, 2, 3)  // Error: no functions allowed
}`;
```

### Reviver Function – Transform Values During Parsing

```javascript
let str = '{"title":"Conference","date":"2017-11-30T12:00:00.000Z"}';

let meetup = JSON.parse(str, function (key, value) {
  if (key == "date") return new Date(value);
  return value;
});

alert(meetup.date.getDate()); // Works! Date object

// Works with nested objects too
let schedule = `{
  "meetups": [
    {"title":"Conference","date":"2017-11-30T12:00:00.000Z"},
    {"title":"Birthday","date":"2017-04-18T12:00:00.000Z"}
  ]
}`;

schedule = JSON.parse(schedule, function (key, value) {
  if (key == "date") return new Date(value);
  return value;
});

alert(schedule.meetups[1].date.getDate()); // Works!
```

## Supported Data Types

| Type        | JSON Support        |
| ----------- | ------------------- |
| Objects     | ✅                  |
| Arrays      | ✅                  |
| Strings     | ✅ (double quotes)  |
| Numbers     | ✅                  |
| Booleans    | ✅ (`true`/`false`) |
| `null`      | ✅                  |
| Functions   | ❌ (skipped)        |
| Symbols     | ❌ (skipped)        |
| `undefined` | ❌ (skipped)        |

## Summary

| Method                                   | Purpose       | Parameters                                          |
| ---------------------------------------- | ------------- | --------------------------------------------------- |
| `JSON.stringify(value, replacer, space)` | Object → JSON | `replacer`: array or function, `space`: indentation |
| `JSON.parse(str, reviver)`               | JSON → Object | `reviver`: function to transform values             |

**Key Points:**

- JSON uses **double quotes** only
- No functions, symbols, or `undefined` in JSON
- Circular references cause errors
- Use `toJSON` for custom serialization
- Use `reviver` to transform during parsing (e.g., convert date strings to Date objects)

## Tasks

### 1. Turn Object into JSON and Back

```javascript
let user = { name: "John Smith", age: 35 };
let json = JSON.stringify(user);
let newUser = JSON.parse(json);
// newUser is a copy of user
```

### 2. Exclude Backreferences

```javascript
let room = { number: 23 };
let meetup = {
  title: "Conference",
  occupiedBy: [{ name: "John" }, { name: "Alice" }],
  place: room,
};

room.occupiedBy = meetup;
meetup.self = meetup;

let json = JSON.stringify(meetup, function replacer(key, value) {
  return key != "" && value === meetup ? undefined : value;
});

/* Result:
{
  "title":"Conference",
  "occupiedBy":[{"name":"John"},{"name":"Alice"}],
  "place":{"number":23}
}
*/
```
