# Regular Expressions: Patterns and Flags Summary

Regular expressions (regexp) are patterns used to search and replace text in strings.

## Creating Regular Expressions

### Two Syntaxes

```javascript
// Long syntax - dynamic pattern
let regexp = new RegExp("pattern", "flags");

// Short syntax - static pattern (slashes)
let regexp = /pattern/; // no flags
let regexp = /pattern/gim; // with flags
```

**Difference:** Slashes are static, `new RegExp` allows dynamic strings:

```javascript
let tag = prompt("What tag to find?", "h2");
let regexp = new RegExp(`<${tag}>`); // Dynamic pattern
```

## Flags

| Flag | Name             | Description                           |
| ---- | ---------------- | ------------------------------------- |
| `i`  | Case-insensitive | No difference between A and a         |
| `g`  | Global           | Finds all matches (not just first)    |
| `m`  | Multiline        | `^` and `$` work across line breaks   |
| `s`  | Dotall           | `.` matches newline `\n`              |
| `u`  | Unicode          | Full Unicode support, surrogate pairs |
| `y`  | Sticky           | Search at exact position              |

## String Methods with Regexp

### str.match(regexp)

```javascript
let str = "We will, we will rock you";

// With 'g' flag - returns array of all matches
console.log(str.match(/we/gi)); // ['We', 'we']

// Without 'g' flag - returns first match with details
let result = str.match(/we/i);
console.log(result[0]); // 'We'
console.log(result.index); // 0 (position)
console.log(result.input); // full string

// No matches - returns null
console.log("JavaScript".match(/HTML/)); // null
```

**Important:** Always handle `null` case:

```javascript
let matches = "JavaScript".match(/HTML/) || [];
if (!matches.length) {
  console.log("No matches"); // Works safely
}
```

### str.replace(regexp, replacement)

```javascript
// Without 'g' - only first match
console.log("We will, we will".replace(/we/i, "I"));
// "I will, we will"

// With 'g' - all matches
console.log("We will, we will".replace(/we/gi, "I"));
// "I will, I will"
```

**Special replacement patterns:**

| Symbol  | Action                      |
| ------- | --------------------------- |
| `$&`    | Inserts whole match         |
| ``$` `` | Inserts text before match   |
| `$'`    | Inserts text after match    |
| `$$`    | Inserts `$` character       |
| `$n`    | Inserts nth capturing group |

```javascript
console.log("I love HTML".replace(/HTML/, "$& and JavaScript"));
// "I love HTML and JavaScript"

console.log("Hello world".replace(/world/, "$` before"));
// "Hello Hello  before"
```

### regexp.test(str)

```javascript
let str = "I love JavaScript";
let regexp = /LOVE/i;

console.log(regexp.test(str)); // true
```

## Additional String Methods

```javascript
// str.search(regexp) - returns index of first match
console.log("I love JS".search(/love/i)); // 2

// str.split(regexp) - split by regexp
console.log("Hello, world! Goodbye".split(/[\s,]+/));
// ['Hello', 'world!', 'Goodbye']
```

## Common Examples

### Case-Insensitive Search

```javascript
let text = "Hello HELLO hello";
let matches = text.match(/hello/gi);
console.log(matches); // ['Hello', 'HELLO', 'hello']
```

### Global Replace

```javascript
let text = "The cat and the cat";
let result = text.replace(/cat/g, "dog");
console.log(result); // "The dog and the dog"
```

### Testing for Pattern

```javascript
function isValidEmail(email) {
  return /^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email);
}

console.log(isValidEmail("test@example.com")); // true
console.log(isValidEmail("invalid")); // false
```

## Regexp Object Methods

```javascript
// regexp.exec(str) - detailed match info
let regexp = /we/gi;
let str = "We will, we will";

let match1 = regexp.exec(str);
console.log(match1[0]); // "We"
console.log(match1.index); // 0

let match2 = regexp.exec(str);
console.log(match2[0]); // "we"
console.log(match2.index); // 9

// With 'g' flag, exec advances position
```

## Summary Table

| Method                      | Description             | Returns             |
| --------------------------- | ----------------------- | ------------------- |
| `str.match(regexp)`         | Find matches            | Array or null       |
| `str.matchAll(regexp)`      | All matches with groups | Iterator            |
| `str.replace(regexp, repl)` | Replace matches         | String              |
| `str.search(regexp)`        | Find first match index  | Number (-1 if none) |
| `str.split(regexp)`         | Split by regexp         | Array               |
| `regexp.test(str)`          | Check for match         | Boolean             |
| `regexp.exec(str)`          | Find with details       | Array or null       |

## Key Points

1. **Two syntaxes**: `/pattern/flags` (static) or `new RegExp("pattern", "flags")` (dynamic)
2. **Flags**: `i` (case-insensitive), `g` (global), `m` (multiline), `s` (dotall), `u` (unicode), `y` (sticky)
3. **str.match()** returns `null` for no matches - always handle this case
4. **str.replace()** with `g` flag replaces all matches
5. **regexp.test()** is simplest for checking existence
6. **Special characters** in replacement: `$&`, ``$` ``, `$'`, `$$`, `$n`
