# Fetch API: Complete Options Summary

Complete reference for all `fetch()` options beyond basic usage.

## Full fetch() Syntax

```javascript
let promise = fetch(url, {
  method: "GET",
  headers: {},
  body: undefined,
  referrer: "about:client",
  referrerPolicy: "strict-origin-when-cross-origin",
  mode: "cors",
  credentials: "same-origin",
  cache: "default",
  redirect: "follow",
  integrity: "",
  keepalive: false,
  signal: undefined,
  window: window,
});
```

## referrer & referrerPolicy

Control the `Referer` header behavior.

### referrer Option

```javascript
// Send no Referer header
fetch("/page", { referrer: "" });

// Set specific URL (same origin only)
fetch("/page", {
  referrer: "https://javascript.info/anotherpage",
});
```

### referrerPolicy Values

| Value                                         | Same Origin | Other Origin | HTTPS→HTTP |
| --------------------------------------------- | ----------- | ------------ | ---------- |
| `"no-referrer"`                               | -           | -            | -          |
| `"no-referrer-when-downgrade"`                | full        | full         | -          |
| `"origin"`                                    | origin      | origin       | origin     |
| `"origin-when-cross-origin"`                  | full        | origin       | origin     |
| `"same-origin"`                               | full        | -            | -          |
| `"strict-origin"`                             | origin      | origin       | -          |
| `"strict-origin-when-cross-origin"` (default) | full        | origin       | -          |
| `"unsafe-url"`                                | full        | full         | full       |

```javascript
// Send only origin to cross-origin sites
fetch("https://another.com/page", {
  referrerPolicy: "origin-when-cross-origin",
  // Referer: https://javascript.info (no path)
});
```

## mode

Controls cross-origin request behavior.

```javascript
// Default - allows CORS requests
fetch(url, { mode: "cors" });

// Block cross-origin requests entirely
fetch(url, { mode: "same-origin" });

// Allow only safe cross-origin requests (GET, POST with safe headers)
fetch(url, { mode: "no-cors" });
```

## credentials

Controls sending cookies and HTTP authentication.

```javascript
// Default - cookies sent only to same origin
fetch(url, { credentials: "same-origin" });

// Always send credentials (requires server CORS setup)
fetch(url, {
  credentials: "include",
  // Server must respond with: Access-Control-Allow-Credentials: true
});

// Never send credentials
fetch(url, { credentials: "omit" });
```

## cache

Controls HTTP cache behavior.

| Value              | Behavior                                          |
| ------------------ | ------------------------------------------------- |
| `"default"`        | Standard HTTP cache rules                         |
| `"no-store"`       | Ignore HTTP cache entirely                        |
| `"reload"`         | Don't use cache, but populate it with response    |
| `"no-cache"`       | Conditional request if cached, normal otherwise   |
| `"force-cache"`    | Use cached even if stale, request if none         |
| `"only-if-cached"` | Use cached only, error if none (same-origin only) |

```javascript
// Force fresh request, don't use cache
fetch(url, { cache: "reload" });

// Use cache only, error if not cached
fetch(url, {
  cache: "only-if-cached",
  mode: "same-origin", // Required for only-if-cached
});
```

## redirect

Controls handling of HTTP redirects (301, 302, etc.).

```javascript
// Default - follow redirects automatically
fetch(url, { redirect: "follow" });

// Throw error on redirect
fetch(url, { redirect: "error" });

// Manual handling
fetch(url, { redirect: "manual" }).then((response) => {
  if (response.type === "opaqueredirect") {
    console.log("Redirect detected");
    // Response has zeroed/empty status and properties
  }
});
```

## integrity

Verify response matches expected checksum.

```javascript
fetch("http://site.com/file", {
  integrity: "sha256-abcdef1234567890...", // SHA-256, SHA-384, or SHA-512
}).catch((err) => {
  console.log("Checksum mismatch!"); // Integrity check failed
});
```

## keepalive

Allow requests to continue after page unload.

```javascript
// Send analytics when user leaves page
window.onunload = function () {
  fetch("/analytics", {
    method: "POST",
    body: JSON.stringify(stats),
    keepalive: true, // Request continues after page closes
  });
};
```

**Limitations:**

- Maximum body size: 64KB total across all keepalive requests
- Cannot process response (page already unloaded)

## Complete Example with Multiple Options

```javascript
async function secureFetch(url, data) {
  let controller = new AbortController();

  // Timeout after 5 seconds
  setTimeout(() => controller.abort(), 5000);

  try {
    let response = await fetch(url, {
      method: "POST",
      headers: {
        "Content-Type": "application/json",
        "X-API-Version": "2.0",
      },
      body: JSON.stringify(data),
      credentials: "include",
      mode: "cors",
      cache: "no-cache",
      redirect: "error",
      referrerPolicy: "strict-origin-when-cross-origin",
      keepalive: false,
      signal: controller.signal,
    });

    if (!response.ok) {
      throw new Error(`HTTP ${response.status}`);
    }

    return await response.json();
  } catch (err) {
    if (err.name === "AbortError") {
      console.log("Request timed out");
    }
    throw err;
  }
}
```

## Option Quick Reference

| Option           | Default                             | Purpose                |
| ---------------- | ----------------------------------- | ---------------------- |
| `method`         | `"GET"`                             | HTTP method            |
| `headers`        | `{}`                                | Request headers        |
| `body`           | `undefined`                         | Request body           |
| `referrer`       | `"about:client"`                    | Referer header value   |
| `referrerPolicy` | `"strict-origin-when-cross-origin"` | Referer rules          |
| `mode`           | `"cors"`                            | CORS behavior          |
| `credentials`    | `"same-origin"`                     | Send cookies/auth      |
| `cache`          | `"default"`                         | Cache behavior         |
| `redirect`       | `"follow"`                          | Redirect handling      |
| `integrity`      | `""`                                | Checksum verification  |
| `keepalive`      | `false`                             | Survive page unload    |
| `signal`         | `undefined`                         | AbortController signal |
