# Class Inheritance

## Overview

Class inheritance allows one class to extend another, reusing and extending functionality.

## The `extends` Keyword

```javascript
class Animal {
  constructor(name) {
    this.speed = 0;
    this.name = name;
  }
  run(speed) {
    this.speed = speed;
    alert(`${this.name} runs with speed ${this.speed}.`);
  }
  stop() {
    this.speed = 0;
    alert(`${this.name} stands still.`);
  }
}

class Rabbit extends Animal {
  hide() {
    alert(`${this.name} hides!`);
  }
}

let rabbit = new Rabbit("White Rabbit");
rabbit.run(5); // White Rabbit runs with speed 5.
rabbit.hide(); // White Rabbit hides!
```

**How it works:** `Rabbit.prototype.[[Prototype]]` is set to `Animal.prototype`.

## Any Expression After `extends`

```javascript
function f(phrase) {
  return class {
    sayHi() {
      alert(phrase);
    }
  };
}

class User extends f("Hello") {}
new User().sayHi(); // Hello
```

## Overriding Methods with `super`

Use `super.method()` to call parent method:

```javascript
class Rabbit extends Animal {
  stop() {
    super.stop(); // Call parent stop
    this.hide(); // Then hide
  }

  hide() {
    alert(`${this.name} hides!`);
  }
}

let rabbit = new Rabbit("White Rabbit");
rabbit.run(5); // White Rabbit runs with speed 5.
rabbit.stop(); // White Rabbit stands still. White Rabbit hides!
```

## Overriding Constructor

**Rule:** In a derived class, must call `super()` before using `this`.

```javascript
class Animal {
  constructor(name) {
    this.speed = 0;
    this.name = name;
  }
}

class Rabbit extends Animal {
  constructor(name, earLength) {
    super(name); // Call parent constructor first
    this.earLength = earLength; // Then use this
  }
}

let rabbit = new Rabbit("White Rabbit", 10);
alert(rabbit.name); // White Rabbit
alert(rabbit.earLength); // 10
```

If no constructor is written, this is automatically generated:

```javascript
constructor(...args) {
  super(...args);
}
```

## Class Fields Override Behavior (Important Note)

Parent constructor uses **its own field values**, not overridden ones:

```javascript
class Animal {
  name = "animal";
  constructor() {
    alert(this.name); // Always "animal"
  }
}

class Rabbit extends Animal {
  name = "rabbit";
}

new Animal(); // animal
new Rabbit(); // animal (not rabbit!)
```

**Reason:** Field initialization order:

- Base class: fields initialized before constructor
- Derived class: fields initialized after `super()`

When parent constructor runs, derived class fields don't exist yet.

## `super` Internals: `[[HomeObject]]`

Methods have an internal `[[HomeObject]]` property that remembers the object where they were created. `super` uses this to find the parent prototype.

**Important:** This makes methods with `super` **not freely copyable** between objects:

```javascript
let animal = {
  sayHi() {
    alert("I'm an animal");
  },
};

let rabbit = {
  __proto__: animal,
  sayHi() {
    super.sayHi();
  },
};

let tree = {
  __proto__: animal,
  sayHi: rabbit.sayHi, // Copying method
};

tree.sayHi(); // "I'm an animal" (wrong - uses rabbit's [[HomeObject]])
```

**Rule:** For `[[HomeObject]]` to work, methods must be defined as `method()` not `method: function()`.

## Summary

| Feature                        | Syntax                                   |
| ------------------------------ | ---------------------------------------- |
| Inherit from class             | `class Child extends Parent`             |
| Call parent constructor        | `super()` (must be first in constructor) |
| Call parent method             | `super.method()`                         |
| Any expression after `extends` | `class Child extends expression`         |

**Key Points:**

- Derived classes must call `super()` before using `this`
- `super` works via internal `[[HomeObject]]` property
- Methods with `super` cannot be safely copied between objects
- Parent constructor uses its own field values, not overridden ones

class Animal {
  constructor(name) {
    this.name = name;
  }
}

class Rabbit extends Animal {
  constructor(name) {

    this.created = Date.now();
  }
}

let rabbit = new Rabbit("White Rabbit");
alert(rabbit.name); // White Rabbit

### 2. Extended Clock

```javascript
class ExtendedClock extends Clock {
  constructor(options) {
    super(options);
    let { precision = 1000 } = options;
    this.precision = precision;
  }

  start() {
    this.render();
    this.timer = setInterval(() => this.render(), this.precision);
  }
}

let clock = new ExtendedClock({ template: "h:m:s", precision: 2000 });
clock.start();
```
