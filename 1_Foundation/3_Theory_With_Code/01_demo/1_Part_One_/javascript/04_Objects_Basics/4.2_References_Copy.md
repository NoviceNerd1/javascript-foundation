# Object References and Copying

## Overview

Objects are stored and copied **by reference**, while primitives are copied **by value**.

## Primitives (Copy by Value)

```javascript
let message = "Hello!";
let phrase = message; // Creates an independent copy
// Both variables hold separate "Hello!" values
```

## Objects (Copy by Reference)

```javascript
let user = { name: "John" };
let admin = user; // Copies reference, not the object

admin.name = "Pete";
alert(user.name); // Pete (same object modified)
```

A variable stores an **address/reference** to the object, not the object itself.

## Comparison by Reference

```javascript
let a = {};
let b = a; // Same reference
alert(a == b); // true
alert(a === b); // true

let c = {};
let d = {}; // Different references
alert(c == d); // false (different objects)
```

## `const` Objects Can Be Modified

```javascript
const user = { name: "John" };
user.name = "Pete"; // Allowed (modifying object)
// user = {};        // Error (reassigning const variable)
```

## Cloning Objects

### Manual Shallow Clone

```javascript
let user = { name: "John", age: 30 };
let clone = {};

for (let key in user) {
  clone[key] = user[key];
}

clone.name = "Pete";
alert(user.name); // John (independent)
```

### `Object.assign()` – Shallow Clone

```javascript
let user = { name: "John", age: 30 };
let clone = Object.assign({}, user);

// Also merges multiple objects
let permissions = { canView: true };
let settings = { theme: "dark" };
Object.assign(user, permissions, settings);
// user = { name: "John", age: 30, canView: true, theme: "dark" }

// Overwrites existing properties
Object.assign(user, { name: "Pete" });
```

### Shallow Clone Limitation – Nested Objects

```javascript
let user = {
  name: "John",
  sizes: { height: 182, width: 50 },
};

let clone = Object.assign({}, user);
clone.sizes.height = 200;

alert(user.sizes.height); // 200 (same object! not cloned)
```

## Deep Cloning

### `structuredClone()` – Built-in Deep Clone

```javascript
let user = {
  name: "John",
  sizes: { height: 182, width: 50 },
};

let clone = structuredClone(user);
clone.sizes.height = 200;

alert(user.sizes.height); // 182 (independent)

// Handles circular references
let user = {};
user.me = user;
let clone = structuredClone(user);
alert(clone.me === clone); // true
```

**Limitations:** Cannot clone functions, DOM elements, etc.

### Lodash `_.cloneDeep()`

For complex cases (functions, etc.):

```javascript
let clone = _.cloneDeep(obj); // Requires lodash library
```

## Summary

| Aspect            | Behavior                                          |
| ----------------- | ------------------------------------------------- |
| **Primitives**    | Copied by value (independent)                     |
| **Objects**       | Copied by reference (shared)                      |
| **Shallow clone** | `Object.assign({}, obj)` or spread `{...obj}`     |
| **Deep clone**    | `structuredClone(obj)` or `_.cloneDeep(obj)`      |
| **Comparison**    | Same object → `true`; Different objects → `false` |
