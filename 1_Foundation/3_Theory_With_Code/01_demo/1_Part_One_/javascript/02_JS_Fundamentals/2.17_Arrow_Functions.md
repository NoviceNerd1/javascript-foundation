# Arrow Functions

## Overview

Arrow functions provide a concise syntax for creating functions, using `=>` (arrow).

## Basic Syntax

```javascript
let func = (arg1, arg2, ..., argN) => expression;
// Returns the expression result
```

Equivalent to:

```javascript
let func = function(arg1, arg2, ..., argN) {
  return expression;
};
```

## Examples

### Multiple Arguments

```javascript
let sum = (a, b) => a + b;
alert(sum(1, 2)); // 3
```

### Single Argument (parentheses optional)

```javascript
let double = (n) => n * 2;
alert(double(3)); // 6
```

### No Arguments (parentheses required)

```javascript
let sayHi = () => alert("Hello!");
sayHi();
```

## Dynamic Creation

Arrow functions work like Function Expressions:

```javascript
let age = prompt("What is your age?", 18);

let welcome = age < 18 ? () => alert("Hello!") : () => alert("Greetings!");

welcome();
```

## Multiline Arrow Functions

Use curly braces `{}` for multiple statements. **Requires explicit `return`**:

```javascript
let sum = (a, b) => {
  let result = a + b;
  return result; // Explicit return needed
};

alert(sum(1, 2)); // 3
```

## Summary

| Syntax                 | Use Case            | Return                   |
| ---------------------- | ------------------- | ------------------------ |
| `(args) => expression` | Single expression   | Implicit                 |
| `(args) => { body }`   | Multiple statements | Explicit `return` needed |

**Parentheses rules:**

- No arguments: `() => ...`
- One argument: `n => ...` (parentheses optional)
- Multiple arguments: `(a, b) => ...`

## Task: Rewrite with Arrow Functions

```javascript
function ask(question, yes, no) {
  if (confirm(question)) yes();
  else no();
}

ask(
  "Do you agree?",
  () => alert("You agreed."),
  () => alert("You canceled the execution."),
);
```
