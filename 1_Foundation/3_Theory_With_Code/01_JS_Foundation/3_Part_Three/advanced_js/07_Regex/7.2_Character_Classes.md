# Character Classes Summary

Character classes are special notations that match any symbol from a specific set.

## Basic Character Classes

| Class | Description                                                 | Example Match      |
| ----- | ----------------------------------------------------------- | ------------------ |
| `\d`  | Digit (0-9)                                                 | `7`, `3`, `0`      |
| `\s`  | Space, tab `\t`, newline `\n`, etc.                         | ` `, `\t`          |
| `\w`  | Word character: Latin letter, digit, underscore `_`         | `a`, `Z`, `5`, `_` |
| `.`   | Any character except newline (with `s` flag: any character) | `a`, ` `, `@`      |

## Usage Examples

```javascript
// Find all digits
let str = "+7(903)-123-45-67";
let digits = str.match(/\d/g);
console.log(digits); // ['7','9','0','3','1','2','3','4','5','6','7']
console.log(digits.join("")); // "79031234567"
```

```javascript
// Find word + digit pattern
console.log("I love HTML5".match(/\s\w\w\w\w\d/)); // [' HTML5']
```

## Inverse Classes (Uppercase)

| Class | Description        | Example       |
| ----- | ------------------ | ------------- |
| `\D`  | Non-digit          | `a`, `@`, ` ` |
| `\S`  | Non-space          | `a`, `1`, `@` |
| `\W`  | Non-word character | `@`, ` `, `-` |

### Remove Non-Digits

```javascript
let str = "+7(903)-123-45-67";
console.log(str.replace(/\D/g, "")); // "79031234567"
```

## Dot `.` (Any Character)

```javascript
// Dot matches any character except newline
console.log("CSS4".match(/CS.4/)); // CSS4
console.log("CS-4".match(/CS.4/)); // CS-4
console.log("CS 4".match(/CS.4/)); // CS 4 (space)
console.log("CS4".match(/CS.4/)); // null (no character for dot)

// Dot does NOT match newline by default
console.log("A\nB".match(/A.B/)); // null

// With 's' flag, dot matches newline
console.log("A\nB".match(/A.B/s)); // A\nB
```

### Alternative for Any Character (No `s` Flag)

```javascript
// [\s\S] = space OR non-space = any character
console.log("A\nB".match(/A[\s\S]B/)); // A\nB

// Other options: [\d\D], [\w\W], [^]
```

## Word Character Limitations

`\w` only matches **Latin letters**, not Cyrillic, Arabic, etc.:

```javascript
// Latin - matches
console.log("hello".match(/\w/g)); // ['h','e','l','l','o']

// Cyrillic - does NOT match
console.log("привет".match(/\w/g)); // null
```

For Unicode letters, use `\p{L}` with `u` flag (covered in Unicode chapter).

## Spaces Matter

```javascript
let str = "1 - 5";

// No spaces - no match
console.log(str.match(/\d-\d/)); // null

// With spaces - matches
console.log(str.match(/\d - \d/)); // "1 - 5"
console.log(str.match(/\d\s-\s\d/)); // "1 - 5"
```

## Practical Examples

### Extract All Numbers

```javascript
let text = "Price: $19.99, discount: 5%";
let numbers = text.match(/\d+(?:\.\d+)?/g);
console.log(numbers); // ['19.99', '5']
```

### Validate Phone Format

```javascript
let phone = "+1 (555) 123-4567";
let isValid = /^\+\d\s\(\d{3}\)\s\d{3}-\d{4}$/.test(phone);
console.log(isValid); // true
```

### Remove Whitespace

```javascript
let messy = "  Hello   world  ";
let clean = messy.replace(/\s+/g, " ").trim();
console.log(clean); // "Hello world"
```

### Count Words

```javascript
let text = "The quick brown fox";
let wordCount = (text.match(/\w+/g) || []).length;
console.log(wordCount); // 4
```

## Summary Table

| Class | Matches                         | Inverse                   |
| ----- | ------------------------------- | ------------------------- |
| `\d`  | Digit (0-9)                     | `\D` (non-digit)          |
| `\s`  | Space, tab, newline             | `\S` (non-space)          |
| `\w`  | Latin letter, digit, underscore | `\W` (non-word)           |
| `.`   | Any character (except newline)  | `[^]` or `[\s\S]` for any |

## Key Points

1. **Character classes** match specific sets of characters
2. **Inverse classes** (uppercase) match everything except the class
3. **Dot `.`** matches any character except newline (use `s` flag or `[\s\S]` for newline)
4. **`\w`** only matches Latin letters, not Unicode
5. **Spaces in regexp** must match spaces in string exactly
6. Use `\D` to remove non-digits, `\S` to find non-whitespace, etc.
