# Promise API

## Overview

The `Promise` class provides 6 static methods for working with multiple promises.

## Promise.all

Waits for **all** promises to resolve. Returns array of results. Rejects immediately if any promise rejects.

```javascript
Promise.all([
  new Promise((resolve) => setTimeout(() => resolve(1), 3000)),
  new Promise((resolve) => setTimeout(() => resolve(2), 2000)),
  new Promise((resolve) => setTimeout(() => resolve(3), 1000)),
]).then(alert); // [1, 2, 3] after 3 seconds
```

### Practical Example: Fetch Multiple URLs

```javascript
let urls = [
  "https://api.github.com/users/iliakan",
  "https://api.github.com/users/remy",
  "https://api.github.com/users/jeresig",
];

let requests = urls.map((url) => fetch(url));

Promise.all(requests)
  .then((responses) => responses.forEach((r) => alert(`${r.url}: ${r.status}`)))
  .then(() => Promise.all(responses.map((r) => r.json())))
  .then((users) => users.forEach((user) => alert(user.name)));
```

### Error Behavior

If any promise rejects, `Promise.all` immediately rejects:

```javascript
Promise.all([
  new Promise((resolve) => setTimeout(() => resolve(1), 1000)),
  new Promise((resolve, reject) =>
    setTimeout(() => reject(new Error("Whoops!")), 2000),
  ),
  new Promise((resolve) => setTimeout(() => resolve(3), 3000)),
]).catch(alert); // Error: Whoops! (after 2 seconds)
```

**Note:** Other promises continue executing but their results are ignored.

### Non-promise Values

Regular values are passed through as-is:

```javascript
Promise.all([
  new Promise((resolve) => setTimeout(() => resolve(1), 1000)),
  2,
  3,
]).then(alert); // [1, 2, 3]
```

## Promise.allSettled

Waits for **all** promises to settle (fulfill or reject). Never rejects.

```javascript
let urls = [
  "https://api.github.com/users/iliakan",
  "https://api.github.com/users/remy",
  "https://no-such-url",
];

Promise.allSettled(urls.map((url) => fetch(url))).then((results) => {
  results.forEach((result, num) => {
    if (result.status == "fulfilled") {
      alert(`${urls[num]}: ${result.value.status}`);
    }
    if (result.status == "rejected") {
      alert(`${urls[num]}: ${result.reason}`);
    }
  });
});
```

**Result format:**

- `{status:"fulfilled", value: result}` for success
- `{status:"rejected", reason: error}` for failure

### Polyfill

```javascript
if (!Promise.allSettled) {
  Promise.allSettled = function (promises) {
    return Promise.all(
      promises.map((p) =>
        Promise.resolve(p).then(
          (value) => ({ status: "fulfilled", value }),
          (reason) => ({ status: "rejected", reason }),
        ),
      ),
    );
  };
}
```

## Promise.race

Waits for the **first settled** promise (fulfilled or rejected). Returns its result/error.

```javascript
Promise.race([
  new Promise((resolve, reject) => setTimeout(() => resolve(1), 1000)),
  new Promise((resolve, reject) =>
    setTimeout(() => reject(new Error("Whoops!")), 2000),
  ),
  new Promise((resolve, reject) => setTimeout(() => resolve(3), 3000)),
]).then(alert); // 1 (after 1 second)
```

## Promise.any

Waits for the **first fulfilled** promise. Ignores rejections. If all reject, returns `AggregateError`.

```javascript
Promise.any([
  new Promise((resolve, reject) =>
    setTimeout(() => reject(new Error("Ouch!")), 1000),
  ),
  new Promise((resolve, reject) => setTimeout(() => resolve(1), 2000)),
  new Promise((resolve, reject) => setTimeout(() => resolve(3), 3000)),
]).then(alert); // 1 (after 2 seconds)
```

### All Rejected Case

```javascript
Promise.any([
  new Promise((resolve, reject) =>
    setTimeout(() => reject(new Error("Ouch!")), 1000),
  ),
  new Promise((resolve, reject) =>
    setTimeout(() => reject(new Error("Error!")), 2000),
  ),
]).catch((error) => {
  alert(error.constructor.name); // AggregateError
  alert(error.errors[0]); // Error: Ouch!
  alert(error.errors[1]); // Error: Error!
});
```

## Promise.resolve / Promise.reject

### Promise.resolve

Creates a resolved promise with the given value.

```javascript
let cache = new Map();

function loadCached(url) {
  if (cache.has(url)) {
    return Promise.resolve(cache.get(url)); // Returns a promise
  }
  return fetch(url)
    .then((response) => response.text())
    .then((text) => {
      cache.set(url, text);
      return text;
    });
}

loadCached(url).then(alert); // Always works (returns promise)
```

### Promise.reject

Creates a rejected promise with the given error (rarely used in modern code).

```javascript
let promise = Promise.reject(new Error("Whoops!"));
// Same as: new Promise((resolve, reject) => reject(error))
```

## Summary

| Method               | Waits for       | Returns                 | Rejects if            |
| -------------------- | --------------- | ----------------------- | --------------------- |
| `Promise.all`        | All promises    | Array of results        | Any promise rejects   |
| `Promise.allSettled` | All promises    | Array of status objects | Never                 |
| `Promise.race`       | First settled   | Result/error            | First promise rejects |
| `Promise.any`        | First fulfilled | Result                  | All promises reject   |
| `Promise.resolve`    | -               | Resolved promise        | -                     |
| `Promise.reject`     | -               | Rejected promise        | -                     |

**Most common:** `Promise.all` for parallel execution with "all or nothing" semantics.
