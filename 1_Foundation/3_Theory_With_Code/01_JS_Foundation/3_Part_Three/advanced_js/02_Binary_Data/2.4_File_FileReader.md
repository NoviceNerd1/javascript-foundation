# File and FileReader Summary

`File` objects inherit from `Blob` and add filesystem capabilities like file names and modification dates.

## File Object

### Constructor

```javascript
new File(fileParts, fileName, [options]);
```

- **fileParts** - array of `Blob`/`BufferSource`/`String`
- **fileName** - string filename
- **options.lastModified** - timestamp of last modification (optional)

### Getting Files from Input

```javascript
<input type="file" onchange="showFile(this)">

<script>
function showFile(input) {
  let file = input.files[0];

  console.log(`File name: ${file.name}`);
  console.log(`Last modified: ${file.lastModified}`);
  console.log(`Size: ${file.size} bytes`);
  console.log(`Type: ${file.type}`);
}
</script>
```

**Note:** For multiple file selection, `input.files` is an array-like object.

### File Properties

| Property       | Description                         |
| -------------- | ----------------------------------- |
| `name`         | File name string                    |
| `lastModified` | Timestamp of last modification      |
| `size`         | Size in bytes (inherited from Blob) |
| `type`         | MIME type (inherited from Blob)     |

## FileReader

Reads data from `Blob`/`File` objects asynchronously.

### Constructor & Methods

```javascript
let reader = new FileReader();

reader.readAsArrayBuffer(blob); // → ArrayBuffer (binary)
reader.readAsText(blob, [encoding]); // → string (utf-8 default)
reader.readAsDataURL(blob); // → base64 data URL
reader.abort(); // cancel operation
```

### Events

| Event       | Description                   |
| ----------- | ----------------------------- |
| `loadstart` | Loading started               |
| `progress`  | During reading                |
| `load`      | Completed successfully        |
| `error`     | Error occurred                |
| `abort`     | `abort()` called              |
| `loadend`   | Finished (success or failure) |

### Reading Text File

```javascript
<input type="file" onchange="readFile(this)">

<script>
function readFile(input) {
  let file = input.files[0];
  let reader = new FileReader();

  reader.readAsText(file);

  reader.onload = function() {
    console.log(reader.result);  // file contents as string
  };

  reader.onerror = function() {
    console.log(reader.error);
  };
}
</script>
```

### Reading as ArrayBuffer

```javascript
reader.readAsArrayBuffer(file);
reader.onload = function () {
  let buffer = reader.result;
  let bytes = new Uint8Array(buffer);
  console.log(bytes);
};
```

### Reading as Data URL (Base64)

```javascript
reader.readAsDataURL(file);
reader.onload = function () {
  let dataUrl = reader.result;
  document.getElementById("preview").src = dataUrl;
};
```

## Alternative: URL.createObjectURL()

Instead of reading file contents, create a direct URL:

```javascript
let url = URL.createObjectURL(file);
img.src = url;
// Later: URL.revokeObjectURL(url);
```

**Comparison:**

| Method                           | Use Case                                          |
| -------------------------------- | ------------------------------------------------- |
| `URL.createObjectURL()`          | Display images, videos, or provide download links |
| `FileReader.readAsText()`        | Need to process text content                      |
| `FileReader.readAsArrayBuffer()` | Need binary manipulation                          |
| `FileReader.readAsDataURL()`     | Need inline data (e.g., canvas, CSS)              |

## Complete Examples

### Image Preview

```javascript
<input type="file" accept="image/*" onchange="previewImage(this)">
<img id="preview">

<script>
function previewImage(input) {
  let file = input.files[0];
  if (!file) return;

  let reader = new FileReader();
  reader.readAsDataURL(file);

  reader.onload = function() {
    document.getElementById('preview').src = reader.result;
  };
}
</script>
```

### File Upload

```javascript
async function uploadFile(file) {
  let formData = new FormData();
  formData.append("file", file);

  let response = await fetch("/upload", {
    method: "POST",
    body: formData,
  });

  return await response.json();
}
```

### Reading Multiple Files

```javascript
<input type="file" multiple onchange="readMultiple(this)">

<script>
function readMultiple(input) {
  Array.from(input.files).forEach(file => {
    let reader = new FileReader();
    reader.readAsText(file);
    reader.onload = () => {
      console.log(`${file.name}: ${reader.result.length} chars`);
    };
  });
}
</script>
```

## FileReaderSync (Web Workers Only)

Synchronous version available in Web Workers:

```javascript
// Inside a Web Worker
let reader = new FileReaderSync();
let arrayBuffer = reader.readAsArrayBuffer(blob);
let text = reader.readAsText(blob);
let dataUrl = reader.readAsDataURL(blob);
```

## Summary

| Object                    | Purpose                                                   |
| ------------------------- | --------------------------------------------------------- |
| **File**                  | Blob with name and lastModified (from user input)         |
| **FileReader**            | Read File/Blob as text, ArrayBuffer, or data URL          |
| **URL.createObjectURL()** | Create temporary URL for display/download without reading |

**Best Practice:** Use `URL.createObjectURL()` for displaying images/videos. Use `FileReader` only when you need to process the actual content (text manipulation, binary processing).
