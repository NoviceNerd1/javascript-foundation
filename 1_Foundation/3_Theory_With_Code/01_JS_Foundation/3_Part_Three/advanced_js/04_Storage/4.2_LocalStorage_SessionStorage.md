# LocalStorage & SessionStorage Summary

Web storage objects provide persistent key/value storage in the browser that is not sent to the server with each request.

## Key Differences

| Feature         | localStorage                                    | sessionStorage                           |
| --------------- | ----------------------------------------------- | ---------------------------------------- |
| **Scope**       | Shared across all tabs/windows with same origin | Per browser tab only                     |
| **Persistence** | Survives browser restart                        | Survives page refresh, lost on tab close |
| **Capacity**    | ~5MB (browser dependent)                        | ~5MB (browser dependent)                 |

## Common Methods

```javascript
// Store data
localStorage.setItem("key", "value");

// Retrieve data
let value = localStorage.getItem("key"); // 'value'

// Remove item
localStorage.removeItem("key");

// Clear all
localStorage.clear();

// Get key by index
let key = localStorage.key(0);

// Get number of items
let count = localStorage.length; // 0
```

## Object-like Access (Not Recommended)

```javascript
// Works but has issues
localStorage.test = 2;
console.log(localStorage.test); // 2
delete localStorage.test;

// Problem: conflicts with built-in methods
let key = "length";
localStorage[key] = 5; // Error!
```

## Looping Over Keys

```javascript
// Method 1: Using index
for (let i = 0; i < localStorage.length; i++) {
  let key = localStorage.key(i);
  console.log(`${key}: ${localStorage.getItem(key)}`);
}

// Method 2: Object.keys (recommended)
let keys = Object.keys(localStorage);
for (let key of keys) {
  console.log(`${key}: ${localStorage.getItem(key)}`);
}

// Method 3: with hasOwnProperty
for (let key in localStorage) {
  if (localStorage.hasOwnProperty(key)) {
    console.log(`${key}: ${localStorage.getItem(key)}`);
  }
}
```

## Storing Objects (Use JSON)

```javascript
// Store object
let user = { name: "John", age: 30 };
localStorage.setItem("user", JSON.stringify(user));

// Retrieve object
let stored = JSON.parse(localStorage.getItem("user"));
console.log(stored.name); // John
```

## Storage Event

Triggers when storage changes in other windows/tabs:

```javascript
window.onstorage = (event) => {
  console.log("Key changed:", event.key);
  console.log("Old value:", event.oldValue);
  console.log("New value:", event.newValue);
  console.log("URL:", event.url);
  console.log("Storage area:", event.storageArea);
};

// This triggers the event in other windows/tabs
localStorage.setItem("now", Date.now());
```

**Event properties:**

- `key` - changed key (null if `clear()`)
- `oldValue` - previous value
- `newValue` - new value
- `url` - page where update occurred
- `storageArea` - localStorage or sessionStorage object

## sessionStorage Example

```javascript
// Tab-specific storage
sessionStorage.setItem("temp", "data");

// Survives refresh
console.log(sessionStorage.getItem("temp")); // 'data'

// Not visible in other tabs
// Each tab has its own sessionStorage
```

## localStorage vs sessionStorage vs Cookies

| Feature           | localStorage | sessionStorage | Cookies            |
| ----------------- | ------------ | -------------- | ------------------ |
| Capacity          | ~5MB+        | ~5MB+          | 4KB                |
| Sent to server    | No           | No             | Yes                |
| Expiration        | Never        | Tab close      | Manual             |
| JavaScript access | Yes          | Yes            | Yes                |
| Server access     | No           | No             | Yes (HTTP headers) |

## Task: Autosave Form Field

```html
<textarea id="editor" rows="10" cols="50"></textarea>

<script>
  const editor = document.getElementById("editor");

  // Load saved content
  editor.value = localStorage.getItem("editorContent") || "";

  // Save on every change
  editor.addEventListener("input", () => {
    localStorage.setItem("editorContent", editor.value);
  });

  // Optional: Save on page unload as backup
  window.addEventListener("beforeunload", () => {
    localStorage.setItem("editorContent", editor.value);
  });
</script>
```

## Additional Examples

### Session-based Shopping Cart

```javascript
// Add to cart (per tab)
function addToCart(item) {
  let cart = JSON.parse(sessionStorage.getItem("cart") || "[]");
  cart.push(item);
  sessionStorage.setItem("cart", JSON.stringify(cart));
}

// Get cart
function getCart() {
  return JSON.parse(sessionStorage.getItem("cart") || "[]");
}
```

### Cross-Tab Communication

```javascript
// Tab 1: Send message
localStorage.setItem(
  "message",
  JSON.stringify({
    text: "Hello from tab 1!",
    timestamp: Date.now(),
  }),
);

// Tab 2: Listen for messages
window.addEventListener("storage", (event) => {
  if (event.key === "message") {
    let message = JSON.parse(event.newValue);
    console.log("Received:", message.text);
    localStorage.removeItem("message"); // Clean up
  }
});
```

### Clear Expired Data

```javascript
function setWithExpiry(key, value, ttlMs) {
  const item = {
    value: value,
    expiry: Date.now() + ttlMs,
  };
  localStorage.setItem(key, JSON.stringify(item));
}

function getWithExpiry(key) {
  const item = JSON.parse(localStorage.getItem(key));
  if (!item) return null;

  if (Date.now() > item.expiry) {
    localStorage.removeItem(key);
    return null;
  }

  return item.value;
}

// Usage
setWithExpiry("temp", "data", 60000); // Expires in 1 minute
```

## Summary

1. **localStorage**: Shared across tabs, survives browser restart
2. **sessionStorage**: Per tab only, survives refresh only
3. **Methods**: `setItem()`, `getItem()`, `removeItem()`, `clear()`, `key()`, `length`
4. **Data type**: Strings only (use JSON for objects)
5. **Storage event**: Triggers on changes from other windows
6. **Capacity**: ~5MB (much larger than cookies)
7. **Security**: Not sent to server, origin-bound
