# Numbers in JavaScript

## Overview

JavaScript has two number types:

- **Regular numbers**: 64-bit IEEE-754 format ("double precision floating point")
- **BigInt**: Arbitrary length integers (for numbers beyond 2⁵³-1)

## Writing Numbers

### Scientific Notation (`e`)

```javascript
let billion = 1e9; // 1,000,000,000
let mcs = 1e-6; // 0.000001
let num = 7.3e9; // 7,300,000,000
```

### Underscores as Separators

```javascript
let billion = 1_000_000_000; // More readable
```

### Different Numeral Systems

```javascript
alert(0xff); // 255 (hex)
alert(0b11111111); // 255 (binary)
alert(0o377); // 255 (octal)
```

### `toString(base)`

```javascript
let num = 255;
alert(num.toString(16)); // "ff"
alert(num.toString(2)); // "11111111"
alert(num.toString(36)); // "73" (max base, uses A-Z)
```

> **Note:** Two dots needed for direct method call: `123456..toString(36)` or `(123456).toString(36)`

## Rounding Functions

| Function     | Description                   | 3.1 | 3.5 | 3.6 | -1.1 | -1.5 | -1.6 |
| ------------ | ----------------------------- | --- | --- | --- | ---- | ---- | ---- |
| `Math.floor` | Rounds down                   | 3   | 3   | 3   | -2   | -2   | -2   |
| `Math.ceil`  | Rounds up                     | 4   | 4   | 4   | -1   | -1   | -1   |
| `Math.round` | To nearest integer            | 3   | 4   | 4   | -1   | -1   | -2   |
| `Math.trunc` | Removes decimal (no rounding) | 3   | 3   | 3   | -1   | -1   | -1   |

### Rounding to n Digits

```javascript
// Multiply and divide
let num = 1.23456;
alert(Math.round(num * 100) / 100); // 1.23

// toFixed(n) - returns string
let num = 12.34;
alert(num.toFixed(1)); // "12.3"
alert(num.toFixed(5)); // "12.34000"
alert(+num.toFixed(2)); // 12.34 (convert to number)
```

## Precision Issues

### The 0.1 + 0.2 Problem

```javascript
alert(0.1 + 0.2 == 0.3); // false
alert(0.1 + 0.2); // 0.30000000000000004
```

**Why?** Numbers are stored in binary; fractions like 0.1 are repeating binary fractions and cannot be stored exactly.

**Solutions:**

```javascript
let sum = 0.1 + 0.2;
alert(sum.toFixed(2)); // "0.30"
alert(+sum.toFixed(2)); // 0.3
alert((0.1 * 10 + 0.2 * 10) / 10); // 0.3 (reduces but doesn't eliminate error)
```

### Large Number Precision Loss

```javascript
alert(9999999999999999); // 10000000000000000 (bits overflow)
```

## Special Values: NaN, Infinity, -0

### Checking NaN

```javascript
alert(isNaN(NaN)); // true
alert(isNaN("str")); // true (converted to NaN)
alert(NaN === NaN); // false (NaN is not equal to itself)

// Strict version (no type conversion)
alert(Number.isNaN("str")); // false (string, not number)
alert(Number.isNaN(NaN)); // true
```

### Checking Finite Numbers

```javascript
alert(isFinite("15")); // true
alert(isFinite(Infinity)); // false
alert(isFinite("str")); // false

// Strict version
alert(Number.isFinite("123")); // false (string)
alert(Number.isFinite(123)); // true
```

### Zero Sign

```javascript
alert(0 === -0); // true (operators treat as same)

// Object.is distinguishes them
alert(Object.is(0, -0)); // false
alert(Object.is(NaN, NaN)); // true (unlike ===)
```

## Parsing Numbers

### `parseInt` and `parseFloat`

Read numbers from strings until they can't continue:

```javascript
alert(parseInt("100px")); // 100
alert(parseFloat("12.5em")); // 12.5
alert(parseInt("12.3")); // 12 (only integer part)
alert(parseFloat("12.3.4")); // 12.3 (stops at second dot)
alert(parseInt("a123")); // NaN (no digits read)
```

### Radix Parameter (base)

```javascript
alert(parseInt("0xff", 16)); // 255
alert(parseInt("ff", 16)); // 255 (0x optional)
alert(parseInt("2n9c", 36)); // 123456
```

## Math Object

```javascript
// Random number 0 to 1 (not including 1)
Math.random();

// Max and min
Math.max(3, 5, -10, 0, 1);   // 5
Math.min(1, 2);               // 1

// Power
Math.pow(2, 10);              // 1024

// Trigonometry, constants, etc.
Math.PI, Math.sin, Math.cos, Math.sqrt, etc.
```

## Tasks

### 1. Sum Numbers from Visitor

```javascript
let a = +prompt("Enter first number:", "");
let b = +prompt("Enter second number:", "");
alert(a + b);
```

### 2. Why 6.35.toFixed(1) == 6.3?

```javascript
alert((6.35).toFixed(1)); // 6.3 (due to binary precision)
// Fix:
alert(Math.round(6.35 * 10) / 10); // 6.4
```

### 3. Repeat Until Input is a Number

```javascript
function readNumber() {
  let num;
  do {
    num = prompt("Enter a number:", 0);
    if (num === null || num === "") return null;
  } while (isNaN(num));
  return +num;
}
```

### 4. Occasional Infinite Loop

```javascript
let i = 0;
while (i != 10) {
  i += 0.2; // Never exactly 10 due to precision loss
}
// Fix: while (i < 10)
```

### 5. Random Number from min to max

```javascript
function random(min, max) {
  return min + Math.random() * (max - min);
}
```

### 6. Random Integer from min to max

```javascript
function randomInteger(min, max) {
  return Math.floor(min + Math.random() * (max - min + 1));
}
```
