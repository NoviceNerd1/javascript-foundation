# Hello, World!

This part covers core JavaScript (the language itself). We'll use the browser as a working environment, keeping browser-specific commands minimal.

## The `<script>` Tag

JavaScript programs can be inserted almost anywhere in HTML using the `<script>` tag.

```html
<!DOCTYPE html>
<html>
  <body>
    <p>Before the script...</p>

    <script>
      alert("Hello, world!");
    </script>

    <p>...After the script.</p>
  </body>
</html>
```

The `<script>` tag contains JavaScript code that automatically executes when the browser processes the tag.

## Modern Markup (Old Attributes)

| Attribute                | Old Usage            | Modern Status                    |
| ------------------------ | -------------------- | -------------------------------- |
| `type="text/javascript"` | HTML4 requirement    | Not required anymore             |
| `language="..."`         | Show script language | No longer needed (JS is default) |

### Ancient Comments (for identification of really old code)

```html
<script type="text/javascript">
  <!--
      ...
  //-->
</script>
```

This hides code from ancient browsers that didn't understand `<script>`. Not used in modern JavaScript.

## External Scripts

For larger JavaScript code, use separate files:

```html
<!-- Absolute path -->
<script src="/path/to/script.js"></script>

<!-- Relative path -->
<script src="script.js"></script>

<!-- Full URL -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/lodash.js/4.17.11/lodash.js"></script>

<!-- Multiple scripts -->
<script src="/js/script1.js"></script>
<script src="/js/script2.js"></script>
```

**Benefits of external files:**

- Browser downloads and caches the file
- Other pages reference the same script from cache (downloaded once)
- Reduces traffic and speeds up pages

## Important Rule

**If `src` is set, script content is ignored.**

```html
<!-- This won't work - content ignored -->
<script src="file.js">
  alert(1);  // ignored because src is set
</script>

<!-- Correct way - split into two scripts -->
<script src="file.js"></script>
<script>
  alert(1);
</script>
```

## Summary

- Use `<script>` tag to add JavaScript to a page
- `type` and `language` attributes are not required
- External scripts use `<script src="path/to/script.js"></script>`
- Only simplest scripts go directly in HTML
