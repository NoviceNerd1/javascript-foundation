# Date and Time

## Overview

The `Date` object stores date and time, providing methods for date/time management.

## Creating Dates

### `new Date()` – Current Date/Time

```javascript
let now = new Date();
alert(now); // Current date and time
```

### `new Date(milliseconds)` – From Timestamp

Milliseconds since Jan 1, 1970 UTC+0 (Unix epoch).

```javascript
let Jan01_1970 = new Date(0);
alert(Jan01_1970); // 01.01.1970

let Jan02_1970 = new Date(24 * 3600 * 1000); // +24 hours
alert(Jan02_1970); // 02.01.1970

// Negative timestamps for earlier dates
let Dec31_1969 = new Date(-24 * 3600 * 1000);
```

### `new Date(datestring)` – From String

```javascript
let date = new Date("2017-01-26");
alert(date); // Parsed automatically
```

### `new Date(year, month, date, hours, minutes, seconds, ms)`

```javascript
// Month: 0 = January, 11 = December
new Date(2011, 0, 1, 0, 0, 0, 0); // 1 Jan 2011, 00:00:00
new Date(2011, 0, 1); // Same (hours default 0)
```

## Accessing Date Components

| Method                                                            | Description                          |
| ----------------------------------------------------------------- | ------------------------------------ |
| `getFullYear()`                                                   | Year (4 digits)                      |
| `getMonth()`                                                      | Month (0-11)                         |
| `getDate()`                                                       | Day of month (1-31)                  |
| `getHours()`, `getMinutes()`, `getSeconds()`, `getMilliseconds()` | Time components                      |
| `getDay()`                                                        | Day of week (0=Sunday to 6=Saturday) |
| `getTime()`                                                       | Timestamp (ms since epoch)           |
| `getTimezoneOffset()`                                             | UTC-local difference in minutes      |

### UTC Versions

Add `UTC` after `get`: `getUTCFullYear()`, `getUTCMonth()`, `getUTCDay()`, etc.

```javascript
let date = new Date();
alert(date.getHours()); // Local timezone hour
alert(date.getUTCHours()); // UTC hour
```

## Setting Date Components

| Method                               | Description                |
| ------------------------------------ | -------------------------- |
| `setFullYear(year, [month], [date])` | Set year                   |
| `setMonth(month, [date])`            | Set month                  |
| `setDate(date)`                      | Set day of month           |
| `setHours(hour, [min], [sec], [ms])` | Set hours                  |
| `setMinutes(min, [sec], [ms])`       | Set minutes                |
| `setSeconds(sec, [ms])`              | Set seconds                |
| `setMilliseconds(ms)`                | Set milliseconds           |
| `setTime(milliseconds)`              | Set full date by timestamp |

## Autocorrection

Out-of-range values auto-adjust:

```javascript
let date = new Date(2013, 0, 32); // 32 Jan → 1 Feb
alert(date); // 1 Feb 2013

// Add 2 days to Feb 28, 2016
let date = new Date(2016, 1, 28);
date.setDate(date.getDate() + 2);
alert(date); // 1 Mar 2016

// Set day 0 → last day of previous month
date.setDate(0);
alert(date); // 31 Dec 2015
```

## Date to Number (Timestamp)

```javascript
let date = new Date();
alert(+date); // Same as date.getTime()
alert(date.getTime());
```

### Date Difference

```javascript
let start = new Date();
// ... some operation ...
let end = new Date();
alert(`Took ${end - start} ms`);
```

## `Date.now()` – Fast Current Timestamp

```javascript
let start = Date.now(); // No object created, faster
// ... operation ...
let end = Date.now();
alert(`Took ${end - start} ms`);
```

## `Date.parse()` – Parse String to Timestamp

Format: `YYYY-MM-DDTHH:mm:ss.sssZ`

```javascript
let ms = Date.parse("2012-01-26T13:51:50.417-07:00");
alert(ms); // 1327611110417

let date = new Date(ms);
alert(date);
```

## Tasks

### 1. Create a Date

```javascript
let date = new Date(2012, 1, 20, 3, 12);
alert(date); // Feb 20, 2012, 3:12am (local time)
```

### 2. Show a Weekday

```javascript
function getWeekDay(date) {
  let days = ["SU", "MO", "TU", "WE", "TH", "FR", "SA"];
  return days[date.getDay()];
}
```

### 3. European Weekday

```javascript
function getLocalDay(date) {
  let day = date.getDay();
  return day === 0 ? 7 : day; // Convert Sunday (0) to 7
}
```

### 4. Which Day of Month Was Days Ago

```javascript
function getDateAgo(date, days) {
  let copy = new Date(date);
  copy.setDate(date.getDate() - days);
  return copy.getDate();
}
```

### 5. Last Day of Month

```javascript
function getLastDayOfMonth(year, month) {
  let date = new Date(year, month + 1, 0); // Day 0 = last day of previous month
  return date.getDate();
}
```

### 6. Seconds Passed Today

```javascript
function getSecondsToday() {
  let now = new Date();
  let today = new Date(now.getFullYear(), now.getMonth(), now.getDate());
  return Math.round((now - today) / 1000);
}
```

### 7. Seconds Till Tomorrow

```javascript
function getSecondsToTomorrow() {
  let now = new Date();
  let tomorrow = new Date(now.getFullYear(), now.getMonth(), now.getDate() + 1);
  return Math.round((tomorrow - now) / 1000);
}
```

### 8. Format Relative Date

```javascript
function formatDate(date) {
  let diff = new Date() - date;

  if (diff < 1000) return "right now";
  if (diff < 60 * 1000) return `${Math.floor(diff / 1000)} sec. ago`;
  if (diff < 3600 * 1000) return `${Math.floor(diff / 60000)} min. ago`;

  let day = date.getDate().toString().padStart(2, "0");
  let month = (date.getMonth() + 1).toString().padStart(2, "0");
  let year = date.getFullYear().toString().slice(-2);
  let hours = date.getHours().toString().padStart(2, "0");
  let minutes = date.getMinutes().toString().padStart(2, "0");

  return `${day}.${month}.${year} ${hours}:${minutes}`;
}
```

## Summary

| Operation         | Method                               |
| ----------------- | ------------------------------------ |
| Current date/time | `new Date()`                         |
| From timestamp    | `new Date(ms)`                       |
| From components   | `new Date(year, month, ...)`         |
| Get timestamp     | `date.getTime()` or `+date`          |
| Fast timestamp    | `Date.now()`                         |
| Parse string      | `Date.parse(str)`                    |
| Get component     | `getFullYear()`, `getMonth()`, etc.  |
| Set component     | `setFullYear()`, `setMonth()`, etc.  |
| Date difference   | Subtract `Date` objects              |
| Autocorrection    | Automatically adjusts invalid values |

**Important:**

- Months: 0-11 (January = 0)
- Days of week: 0-6 (Sunday = 0)
- Timestamps are in **milliseconds** (not seconds)
