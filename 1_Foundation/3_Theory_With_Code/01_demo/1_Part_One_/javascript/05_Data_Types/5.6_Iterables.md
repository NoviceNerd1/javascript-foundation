# Iterables

## Overview

Iterable objects can be used in `for..of` loops. Arrays and strings are built-in iterables.

## Making an Object Iterable

To make an object iterable, add the `Symbol.iterator` method that returns an **iterator** with a `next()` method.

```javascript
let range = {
  from: 1,
  to: 5,
};

range[Symbol.iterator] = function () {
  return {
    current: this.from,
    last: this.to,

    next() {
      if (this.current <= this.last) {
        return { done: false, value: this.current++ };
      } else {
        return { done: true };
      }
    },
  };
};

for (let num of range) {
  alert(num); // 1, 2, 3, 4, 5
}
```

### Simplified (Object as Its Own Iterator)

```javascript
let range = {
  from: 1,
  to: 5,

  [Symbol.iterator]() {
    this.current = this.from;
    return this;
  },

  next() {
    if (this.current <= this.to) {
      return { done: false, value: this.current++ };
    } else {
      return { done: true };
    }
  },
};
```

**Downside:** Cannot have two parallel `for..of` loops.

## Strings Are Iterable

```javascript
for (let char of "test") {
  alert(char); // t, e, s, t
}

// Works correctly with surrogate pairs
let str = "ЁЭТ│ЁЯШВ";
for (let char of str) {
  alert(char); // ЁЭТ│, then ЁЯШВ
}
```

## Calling Iterator Explicitly

```javascript
let str = "Hello";
let iterator = str[Symbol.iterator]();

while (true) {
  let result = iterator.next();
  if (result.done) break;
  alert(result.value); // H, e, l, l, o
}
```

## Iterables vs Array-likes

| Type           | Definition                          | Example                    |
| -------------- | ----------------------------------- | -------------------------- |
| **Iterable**   | Has `Symbol.iterator`               | `range` (above)            |
| **Array-like** | Has `length` and indexed properties | `{0:"a", 1:"b", length:2}` |

```javascript
let arrayLike = {
  0: "Hello",
  1: "World",
  length: 2,
};

// Not iterable (no Symbol.iterator)
for (let item of arrayLike) {
} // Error!
```

## Array.from тАУ Convert to Real Array

```javascript
// From array-like
let arrayLike = { 0: "Hello", 1: "World", length: 2 };
let arr = Array.from(arrayLike);
alert(arr.pop()); // "World" (array methods work)

// From iterable
let range = { from: 1, to: 5 };
range[Symbol.iterator] = function () {
  return {
    current: this.from,
    last: this.to,
    next() {
      if (this.current <= this.last) {
        return { done: false, value: this.current++ };
      } else {
        return { done: true };
      }
    },
  };
};

let arr = Array.from(range);
alert(arr); // [1, 2, 3, 4, 5]
```

### With Mapping Function

```javascript
let arr = Array.from(range, (num) => num * num);
alert(arr); // [1, 4, 9, 16, 25]
```

### String to Array (Surrogate-Aware)

```javascript
let str = "ЁЭТ│ЁЯШВ";
let chars = Array.from(str);
alert(chars[0]); // ЁЭТ│
alert(chars[1]); // ЁЯШВ
alert(chars.length); // 2

// Surrogate-aware slice
function slice(str, start, end) {
  return Array.from(str).slice(start, end).join("");
}

alert(slice("ЁЭТ│ЁЯШВЁй╖╢", 1, 3)); // ЁЯШВЁй╖╢ (works correctly)
alert("ЁЭТ│ЁЯШВЁй╖╢".slice(1, 3)); // Garbage (native slice fails)
```

## Summary

| Concept              | Requirements                                              | Example                               |
| -------------------- | --------------------------------------------------------- | ------------------------------------- |
| **Iterable**         | `Symbol.iterator` method returning iterator with `next()` | `range` object                        |
| **Iterator**         | `next()` method returning `{done, value}`                 | Return value from `Symbol.iterator`   |
| **Array-like**       | `length` and indexed properties                           | `{0:"a", length:1}`                   |
| **Convert to array** | `Array.from(iterableOrArrayLike)`                         | Creates real array with array methods |

**Key Points:**

- `for..of` works with iterables only
- Strings are both iterable and array-like
- `Array.from` converts iterables or array-likes to real arrays
- Iterators can be infinite (use `break` to stop)
