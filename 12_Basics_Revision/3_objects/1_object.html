What Are Objects?

Objects are key-value pairs that group related data and behavior.

const person = {
  name: "Alice",
  age: 25,
  greet() {
    return `Hi, I'm ${this.name}`;
  }
};

console.log(person.greet()); // "Hi, I'm Alice"

Properties & Methods

Properties store values, methods are functions on objects.

const car = {
  brand: "Tesla",
  speed: 0,
  accelerate(amount) {
    this.speed += amount;
    return this.speed;
  }
};

car.accelerate(50); // 50

Creating Objects

Constructor Function

function Book(title, author) {
  this.title = title;
  this.author = author;
}
const b = new Book("1984", "Orwell");

ES6 Class

class Animal {
  constructor(name) { this.name = name; }
  speak() { return `${this.name} makes a noise`; }
}
const dog = new Animal("Rex");
dog.speak(); // "Rex makes a noise"

OOP Concepts

Encapsulation

class BankAccount {
  #balance = 0; // private field
  deposit(amount) { this.#balance += amount; }
  getBalance() { return this.#balance; }
}

Inheritance

class Dog extends Animal {
  speak() { return `${this.name} barks`; }
}

Polymorphism

function makeItSpeak(animal) {
  console.log(animal.speak());
}
makeItSpeak(new Dog("Buddy"));
makeItSpeak(new Animal("Creature"));
