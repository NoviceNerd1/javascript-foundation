# JavaScript Specials

## Code Structure

- Statements separated by semicolons or line breaks (automatic semicolon insertion)
- Semicolons **not required** after code blocks `{}` (functions, loops, etc.)

```javascript
alert("Hello");
alert("World"); // Works
alert("Hello"); // Also works
alert("World");

// No semicolon needed after function
function f() {}
```

## Strict Mode

Place `"use strict"` at the top of scripts/functions to enable modern JavaScript features.

```javascript
"use strict";
// Modern JavaScript code here
```

## Variables

### Declaration

- `let` – mutable variable
- `const` – constant (cannot be reassigned)
- `var` – old style

### Naming

- Letters, digits, `$`, `_` (first character cannot be digit)
- Case-sensitive

### Dynamic Typing

```javascript
let x = 5; // number
x = "John"; // string - same variable, different type
```

### Data Types (8 total)

1. `number` – integers and floats
2. `bigint` – arbitrary length integers
3. `string` – text
4. `boolean` – `true`/`false`
5. `null` – empty/does not exist
6. `undefined` – not assigned
7. `object` – complex structures
8. `symbol` – unique identifiers

### `typeof` Exceptions

```javascript
typeof null == "object"; // Language quirk
typeof function () {} == "function"; // Functions treated specially
```

## User Interaction (Browser)

```javascript
let name = prompt("Your name?", "Alice"); // Returns input or null
let tea = confirm("Do you want tea?"); // Returns true/false
alert("Visitor: " + name); // Outputs message
```

All are **modal** (pause code execution until user responds).

## Operators

### Arithmetic

- Regular: `+`, `-`, `*`, `/`, `%` (remainder), `**` (power)

### String Concatenation

```javascript
alert("1" + 2); // '12' (string)
alert(1 + "2"); // '12' (string)
```

### Comparisons

| Operator | Behavior                             |
| -------- | ------------------------------------ |
| `==`     | Converts types, then compares        |
| `===`    | Strict equality (no type conversion) |

```javascript
alert(0 == false); // true
alert(0 == ""); // true
alert(0 === false); // false
```

### Logical Operators

- `&&` (AND), `||` (OR) – short-circuit evaluation, return original value
- `!` (NOT) – converts to boolean and inverts

### Nullish Coalescing

```javascript
let result = a ?? b; // a if defined (not null/undefined), else b
```

### Conditional (Ternary)

```javascript
let result = condition ? value1 : value2;
```

## Loops

### Three Types

```javascript
// while – condition before each iteration
while (condition) {}

// do...while – executes at least once
do {} while (condition);

// for – begin, condition, step
for (let i = 0; i < 10; i++) {}
```

### Control Directives

- `break` – exit loop
- `continue` – skip current iteration
- `labels` – break/continue nested loops

## switch Statement

Uses **strict equality** (`===`).

```javascript
let age = prompt("Your age?", 18);

switch (age) {
  case "18":
    alert("Works!"); // String matches
    break;
  case 18:
    alert("Won't work"); // Number doesn't match string
    break;
  default:
    alert("Other value");
}
```

## Functions

### Function Declaration

```javascript
function sum(a, b) {
  return a + b;
}
```

### Function Expression

```javascript
let sum = function (a, b) {
  return a + b;
};
```

### Arrow Functions

```javascript
// Single line – implicit return
let sum = (a, b) => a + b;

// Single argument – parentheses optional
let double = (n) => n * 2;

// No arguments – parentheses required
let sayHi = () => alert("Hello");

// Multiple lines – explicit return
let sum = (a, b) => {
  let result = a + b;
  return result;
};
```

### Function Features

- **Local variables** – visible only inside function
- **Default parameters** – `function sum(a = 1, b = 2) { }`
- **Return value** – `undefined` if no `return` statement
