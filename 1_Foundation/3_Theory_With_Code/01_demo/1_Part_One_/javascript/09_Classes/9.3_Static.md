# Static Properties and Methods

## Overview

Static methods and properties belong to the class itself, not to instances.

## Static Methods

```javascript
class User {
  static staticMethod() {
    alert(this === User);
  }
}

User.staticMethod(); // true (this = User class)

// Equivalent to:
User.staticMethod = function () {
  alert(this === User);
};
```

### Example: Comparison Method

```javascript
class Article {
  constructor(title, date) {
    this.title = title;
    this.date = date;
  }

  static compare(articleA, articleB) {
    return articleA.date - articleB.date;
  }
}

let articles = [
  new Article("HTML", new Date(2019, 1, 1)),
  new Article("CSS", new Date(2019, 0, 1)),
  new Article("JavaScript", new Date(2019, 11, 1)),
];

articles.sort(Article.compare);
alert(articles[0].title); // CSS
```

### Example: Factory Method

```javascript
class Article {
  constructor(title, date) {
    this.title = title;
    this.date = date;
  }

  static createTodays() {
    return new this("Today's digest", new Date());
  }
}

let article = Article.createTodays();
alert(article.title); // Today's digest
```

## Static Properties

```javascript
class Article {
  static publisher = "Ilya Kantor";
}

alert(Article.publisher); // Ilya Kantor

// Equivalent to:
Article.publisher = "Ilya Kantor";
```

## Static Methods Are Not Available on Instances

```javascript
class User {
  static staticMethod() {}
}

let user = new User();
user.staticMethod(); // Error: user.staticMethod is not a function
```

## Inheritance of Static Members

Static properties and methods are inherited.

```javascript
class Animal {
  static planet = "Earth";

  static compare(animalA, animalB) {
    return animalA.speed - animalB.speed;
  }
}

class Rabbit extends Animal {
  hide() {
    alert(`${this.name} hides!`);
  }
}

let rabbits = [new Rabbit("White Rabbit", 10), new Rabbit("Black Rabbit", 5)];

rabbits.sort(Rabbit.compare); // Inherited static method
alert(Rabbit.planet); // Earth (inherited static property)
```

### How Inheritance Works

```javascript
class Animal {}
class Rabbit extends Animal {}

// Statics: Rabbit.__proto__ = Animal
alert(Rabbit.__proto__ === Animal); // true

// Regular methods: Rabbit.prototype.__proto__ = Animal.prototype
alert(Rabbit.prototype.__proto__ === Animal.prototype); // true
```

## Summary

| Feature         | Syntax                | Call                             |
| --------------- | --------------------- | -------------------------------- |
| Static method   | `static method()`     | `ClassName.method()`             |
| Static property | `static prop = value` | `ClassName.prop`                 |
| Inheritance     | Yes                   | Child class inherits from parent |

**Key Points:**

- Static members belong to the class, not instances
- `this` inside static method refers to the class
- Static methods are useful for factory functions, comparisons, utility functions
- Static properties store class-level data
- Static members are inherited by child classes

## Task: Class extends Object?

```javascript
// Problem: Missing super() call
class Rabbit extends Object {
  constructor(name) {
    super(); // Must call super() first
    this.name = name;
  }
}

let rabbit = new Rabbit("Rab");
alert(rabbit.hasOwnProperty("name")); // true
```

**Difference:**

- `class Rabbit` – prototype chain: `Rabbit.prototype` → `Object.prototype`
- `class Rabbit extends Object` – requires `super()` call and also adds `Object` as parent class, but `Rabbit.__proto__` becomes `Object` (not default `Function.prototype`)
