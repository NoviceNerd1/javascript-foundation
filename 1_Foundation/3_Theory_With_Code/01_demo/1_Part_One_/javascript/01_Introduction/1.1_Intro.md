# JavaScript Introduction

## What is JavaScript?

JavaScript is a programming language initially created to "make web pages alive." Scripts run automatically as web pages load, written directly in HTML as plain text - no compilation needed.

## Why "JavaScript"?

Originally named "LiveScript," it was renamed to position alongside the popular Java language. Today, JavaScript is independent with its own specification (ECMAScript) and has no relation to Java.

## JavaScript Engines

Different browsers have different JavaScript engines:

- **V8** - Chrome, Opera, Edge
- **SpiderMonkey** - Firefox
- **Chakra** - Internet Explorer
- **JavaScriptCore/Nitro/SquirrelFish** - Safari

### How Engines Work:

1. Reads (parses) the script
2. Converts (compiles) script to machine code
3. Machine code runs
4. Engine applies optimizations throughout

## What Can In-Browser JavaScript Do?

- Add/modify HTML content and styles
- React to user actions (clicks, key presses, mouse movements)
- Send/receive network requests (AJAX, COMET)
- Get/set cookies, show messages, ask questions
- Store client-side data (local storage)

## What CAN'T In-Browser JavaScript Do? (Security Limitations)

- Cannot read/write arbitrary files on hard disk
- Cannot execute programs or access OS functions directly
- Cannot access other tabs/windows from different domains, protocols, or ports (Same Origin Policy)
- Cannot receive data from other sites/domains without explicit HTTP header agreement
- Can access camera/microphone only with user's explicit permission

_Note: These limitations don't apply to JavaScript outside browsers (e.g., Node.js)._

## What Makes JavaScript Unique?

1. Full integration with HTML/CSS
2. Simple things done simply
3. Supported by all major browsers (enabled by default)

_JavaScript is the only browser technology combining all three features._

## Languages That Transpile to JavaScript

These languages convert to JavaScript before running in browsers:

| Language     | Key Feature                         | Developer |
| ------------ | ----------------------------------- | --------- |
| CoffeeScript | Shorter syntax, "syntactic sugar"   | -         |
| TypeScript   | Strict data typing                  | Microsoft |
| Flow         | Data typing (different approach)    | Facebook  |
| Dart         | Standalone language with own engine | Google    |
| Brython      | Python transpiler to JS             | -         |
| Kotlin       | Modern, concise, safe language      | JetBrains |

## Summary

- Initially browser-only, now used in many environments
- Most widely-adopted browser language, fully integrated with HTML/CSS
- Many languages transpile to JavaScript - worth exploring after mastering JS
