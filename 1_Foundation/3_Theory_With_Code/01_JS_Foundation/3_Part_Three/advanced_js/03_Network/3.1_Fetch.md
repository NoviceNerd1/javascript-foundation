# Fetch API Summary

`fetch()` is a modern method for network requests, returning a Promise that resolves to a Response object.

## Basic Syntax

```javascript
let promise = fetch(url, [options]);
```

## Two-Stage Response Processing

### Stage 1: Headers Received

```javascript
let response = await fetch(url);

console.log(response.status); // HTTP status code (e.g., 200)
console.log(response.ok); // true if status 200-299
```

### Stage 2: Body Read

```javascript
let text = await response.text(); // as string
let json = await response.json(); // parse as JSON
let blob = await response.blob(); // as Blob
let buffer = await response.arrayBuffer(); // as ArrayBuffer
let formData = await response.formData(); // as FormData
```

**⚠️ Important:** Can only use ONE body reading method per response.

## Complete Example

```javascript
let response = await fetch(
  "https://api.github.com/repos/javascript-tutorial/en.javascript.info/commits",
);

if (response.ok) {
  let commits = await response.json();
  console.log(commits[0].author.login);
} else {
  alert("HTTP-Error: " + response.status);
}
```

## Headers

### Reading Response Headers

```javascript
let response = await fetch(url);

// Get single header
console.log(response.headers.get("Content-Type"));

// Iterate all headers
for (let [key, value] of response.headers) {
  console.log(`${key} = ${value}`);
}
```

### Setting Request Headers

```javascript
let response = await fetch(url, {
  headers: {
    Authentication: "secret",
    "Content-Type": "application/json",
  },
});
```

**Forbidden headers** (controlled by browser): `Cookie`, `Host`, `Content-Length`, `Origin`, etc.

## POST Requests

### Sending JSON

```javascript
let user = { name: "John", surname: "Smith" };

let response = await fetch("/api/user", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
  },
  body: JSON.stringify(user),
});

let result = await response.json();
```

### Sending Form Data

```javascript
let formData = new FormData();
formData.append("name", "John");
formData.append("file", fileBlob);

let response = await fetch("/upload", {
  method: "POST",
  body: formData, // Content-Type automatically set to multipart/form-data
});
```

### Sending Binary Data (Blob)

```javascript
// Canvas to blob
let blob = await new Promise((resolve) => canvas.toBlob(resolve, "image/png"));

let response = await fetch("/upload", {
  method: "POST",
  body: blob, // Content-Type from blob.type automatically used
});
```

### Sending URLSearchParams

```javascript
let params = new URLSearchParams({
  name: "John",
  age: 30,
});

let response = await fetch("/api", {
  method: "POST",
  body: params, // application/x-www-form-urlencoded
});
```

## Error Handling

```javascript
try {
  let response = await fetch("/bad-url");

  if (!response.ok) {
    throw new Error(`HTTP error! status: ${response.status}`);
  }

  let data = await response.json();
} catch (error) {
  console.log("Fetch failed:", error); // Network errors, invalid JSON, etc.
}
```

## Response Body as Stream

```javascript
let response = await fetch("/large-file");
let reader = response.body.getReader();

while (true) {
  let { done, value } = await reader.read();
  if (done) break;
  console.log("Chunk:", value); // Uint8Array
}
```

## Task: Fetch Users from GitHub

```javascript
async function getUsers(names) {
  let requests = names.map((name) =>
    fetch(`https://api.github.com/users/${name}`)
      .then((response) => (response.ok ? response.json() : null))
      .catch(() => null),
  );

  return Promise.all(requests);
}

// Usage
let users = await getUsers(["iliakan", "remy", "no-such-user"]);
console.log(users);
```

## Summary Table

| Method                                | Description             |
| ------------------------------------- | ----------------------- |
| `fetch(url)`                          | GET request             |
| `fetch(url, {method, headers, body})` | Request with options    |
| `response.text()`                     | Read as string          |
| `response.json()`                     | Parse as JSON           |
| `response.blob()`                     | Read as Blob            |
| `response.arrayBuffer()`              | Read as ArrayBuffer     |
| `response.formData()`                 | Read as FormData        |
| `response.headers`                    | Map of response headers |
| `response.status`                     | HTTP status code        |
| `response.ok`                         | Status 200-299?         |

## Common Options

```javascript
let options = {
  method: "POST", // GET, POST, PUT, DELETE, etc.
  headers: {}, // Custom headers
  body: JSON.stringify(), // Request body
  mode: "cors", // cors, no-cors, same-origin
  credentials: "include", // send cookies
  cache: "no-cache", // cache behavior
};
```
