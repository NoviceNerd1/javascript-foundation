# Object.keys, values, entries

## Overview

Plain objects have methods similar to Map/Set/Array, but with different syntax:

| Collection    | Method             | Returns    |
| ------------- | ------------------ | ---------- |
| Map/Set/Array | `map.keys()`       | Iterable   |
| Plain Object  | `Object.keys(obj)` | Real Array |

## Object Methods

| Method                | Returns                       |
| --------------------- | ----------------------------- |
| `Object.keys(obj)`    | Array of keys                 |
| `Object.values(obj)`  | Array of values               |
| `Object.entries(obj)` | Array of `[key, value]` pairs |

### Examples

```javascript
let user = {
  name: "John",
  age: 30,
};

Object.keys(user); // ["name", "age"]
Object.values(user); // ["John", 30]
Object.entries(user); // [["name","John"], ["age",30]]

// Loop over values
for (let value of Object.values(user)) {
  alert(value); // John, then 30
}
```

## Key Differences from Map

| Feature       | Map          | Object             |
| ------------- | ------------ | ------------------ |
| Call syntax   | `map.keys()` | `Object.keys(obj)` |
| Return type   | Iterable     | Real Array         |
| Symbolic keys | Included     | Ignored            |

## Transforming Objects

Objects don't have array methods like `map`, `filter`. Use `Object.entries` + `Object.fromEntries`:

```javascript
let prices = {
  banana: 1,
  orange: 2,
  meat: 4,
};

let doublePrices = Object.fromEntries(
  Object.entries(prices).map((entry) => [entry[0], entry[1] * 2]),
);

alert(doublePrices.meat); // 8
```

### Step-by-step:

1. `Object.entries(prices)` → `[["banana",1], ["orange",2], ["meat",4]]`
2. `map` transforms each pair → `[["banana",2], ["orange",4], ["meat",8]]`
3. `Object.fromEntries` converts back to object

## Additional Methods for Symbols

| Method                              | Returns                           |
| ----------------------------------- | --------------------------------- |
| `Object.getOwnPropertySymbols(obj)` | Array of symbolic keys only       |
| `Reflect.ownKeys(obj)`              | All keys (both string and symbol) |

## Tasks

### 1. Sum the Properties

```javascript
function sumSalaries(salaries) {
  let sum = 0;
  for (let salary of Object.values(salaries)) {
    sum += salary;
  }
  return sum;
}

let salaries = {
  John: 100,
  Pete: 300,
  Mary: 250,
};

alert(sumSalaries(salaries)); // 650
```

### 2. Count Properties

```javascript
function count(obj) {
  return Object.keys(obj).length;
}

let user = {
  name: "John",
  age: 30,
};

alert(count(user)); // 2
```

## Summary

| Task             | Solution                       |
| ---------------- | ------------------------------ |
| Get keys         | `Object.keys(obj)`             |
| Get values       | `Object.values(obj)`           |
| Get entries      | `Object.entries(obj)`          |
| Transform object | `Object.fromEntries(array)`    |
| Count properties | `Object.keys(obj).length`      |
| Sum values       | Loop over `Object.values(obj)` |
