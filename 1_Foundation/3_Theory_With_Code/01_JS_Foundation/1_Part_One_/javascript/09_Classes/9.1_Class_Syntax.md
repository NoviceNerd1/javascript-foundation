# Class Basic Syntax

## Overview

Classes are templates for creating objects with shared methods and properties. They provide a cleaner syntax than constructor functions.

## Basic Class Syntax

```javascript
class MyClass {
  constructor() { ... }      // Called automatically on new
  method1() { ... }          // Method added to prototype
  method2() { ... }
}

let obj = new MyClass();
```

### Example

```javascript
class User {
  constructor(name) {
    this.name = name;
  }

  sayHi() {
    alert(this.name);
  }
}

let user = new User("John");
user.sayHi(); // John
```

**Important:** No commas between class methods!

## What a Class Really Is

A class is a **function**:

```javascript
class User {
  constructor(name) {
    this.name = name;
  }
  sayHi() {
    alert(this.name);
  }
}

alert(typeof User); // function
alert(User === User.prototype.constructor); // true
alert(User.prototype.sayHi); // method code
```

### Class Is Not Just Syntactic Sugar

| Feature                | Class        | Manual Constructor |
| ---------------------- | ------------ | ------------------ |
| Must call with `new`   | ✅ Required  | ❌ Not enforced    |
| Methods non-enumerable | ✅           | ❌ (by default)    |
| Strict mode            | ✅ Automatic | ❌ Manual          |

```javascript
class User {
  constructor() {}
}
User(); // Error: must be called with new
```

## Class Expression

```javascript
let User = class {
  sayHi() {
    alert("Hello");
  }
};

// Named class expression (name visible only inside)
let User = class MyClass {
  sayHi() {
    alert(MyClass);
  }
};

// Dynamically created class
function makeClass(phrase) {
  return class {
    sayHi() {
      alert(phrase);
    }
  };
}
let User = makeClass("Hello");
new User().sayHi(); // Hello
```

## Getters and Setters

```javascript
class User {
  constructor(name) {
    this.name = name; // calls setter
  }

  get name() {
    return this._name;
  }

  set name(value) {
    if (value.length < 4) {
      alert("Name too short");
      return;
    }
    this._name = value;
  }
}

let user = new User("John");
alert(user.name); // John
```

## Computed Method Names

```javascript
class User {
  ["say" + "Hi"]() {
    alert("Hello");
  }
}

new User().sayHi(); // Hello
```

## Class Fields

Properties defined directly in the class body (not in constructor):

```javascript
class User {
  name = "John"; // Class field
  age = prompt("Age?", 30); // Can use expressions

  sayHi() {
    alert(`Hello, ${this.name}!`);
  }
}

let user = new User();
alert(user.name); // John
alert(User.prototype.name); // undefined (not on prototype)
```

### Bound Methods with Class Fields

Class fields create methods bound to the instance:

```javascript
class Button {
  constructor(value) {
    this.value = value;
  }

  // Regular method - loses this when passed
  click() {
    alert(this.value);
  }

  // Arrow function field - this is always the instance
  clickBound = () => {
    alert(this.value);
  };
}

let button = new Button("hello");
setTimeout(button.click, 1000); // undefined (this lost)
setTimeout(button.clickBound, 1000); // hello (works!)
```

## Summary

| Feature         | Syntax                     |
| --------------- | -------------------------- |
| Constructor     | `constructor() { }`        |
| Methods         | `method() { }`             |
| Getters/Setters | `get prop()`, `set prop()` |
| Class fields    | `prop = value`             |
| Computed names  | `[expression]() { }`       |
| Bound methods   | `method = () => { }`       |

**Key Points:**

- Classes are functions with special properties
- Must be called with `new`
- Methods are non-enumerable
- Code inside classes is strict mode
- Class fields are per-instance, not on prototype
- Arrow function fields create bound methods

## Task: Rewrite to Class

```javascript
// Original functional Clock
function Clock({ template }) {
  let timer;

  function render() {
    let date = new Date();
    let hours = date.getHours();
    let mins = date.getMinutes();
    let secs = date.getSeconds();
    let output = template
      .replace("h", hours)
      .replace("m", mins)
      .replace("s", secs);
    console.log(output);
  }

  this.stop = function () {
    clearInterval(timer);
  };

  this.start = function () {
    render();
    timer = setInterval(render, 1000);
  };
}

// Rewritten as class
class Clock {
  constructor({ template }) {
    this.template = template;
    this.timer = null;
  }

  render() {
    let date = new Date();
    let hours = date.getHours();
    let mins = date.getMinutes();
    let secs = date.getSeconds();
    let output = this.template
      .replace("h", hours)
      .replace("m", mins)
      .replace("s", secs);
    console.log(output);
  }

  stop() {
    clearInterval(this.timer);
  }

  start() {
    this.render();
    this.timer = setInterval(() => this.render(), 1000);
  }
}
```
