# Global Object

## Overview

The global object provides variables and functions available everywhere. Its name depends on the environment:

| Environment | Global Object |
| ----------- | ------------- |
| Browser     | `window`      |
| Node.js     | `global`      |
| Universal   | `globalThis`  |

## Accessing Global Properties

```javascript
alert("Hello");
// Same as:
window.alert("Hello");
```

## `var` vs `let/const` with Global Object

### `var` Creates Global Properties

```javascript
var gVar = 5;
alert(window.gVar); // 5 (becomes property of global object)

function sayHi() {
  alert("Hi");
}
window.sayHi(); // Works (function declarations also become global)
```

### `let/const` Do NOT Create Global Properties

```javascript
let gLet = 5;
alert(window.gLet); // undefined (doesn't become global property)

const gConst = 10;
alert(window.gConst); // undefined
```

## Using Global Object Directly

For important values, assign them explicitly:

```javascript
// Make globally available
window.currentUser = {
  name: "John",
};

// Access anywhere
alert(currentUser.name); // John
alert(window.currentUser.name); // John (safe if local variable exists)
```

## Polyfills with Global Object

Check for feature support and add missing functionality:

```javascript
if (!window.Promise) {
  alert("Your browser is really old!");
  window.Promise = customPromiseImplementation; // Add polyfill
}
```

## Best Practices

- **Minimize** global variables – prefer functions with inputs/outputs
- Use `window.property` explicitly for clarity
- Modern scripts use **modules** where global object pollution doesn't occur
- Use `globalThis` for cross-environment compatibility

## Summary

| Concept           | Description                            |
| ----------------- | -------------------------------------- |
| **`window`**      | Browser global object                  |
| **`global`**      | Node.js global object                  |
| **`globalThis`**  | Universal name (modern standard)       |
| **`var`**         | Creates global object properties       |
| **`let`/`const`** | Do NOT create global object properties |
| **Polyfills**     | Add missing features via global object |
