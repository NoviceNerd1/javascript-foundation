# TextDecoder & TextEncoder Summary

Convert between binary data and text strings.

## TextDecoder (Bytes → String)

Creates a decoder for converting binary data to text.

```javascript
let decoder = new TextDecoder([label], [options]);
let str = decoder.decode(input);
```

### Parameters

- **label** - encoding format (`'utf-8'` default, also `'big5'`, `'windows-1251'`, etc.)
- **options.fatal** - if `true`, throws error on invalid characters; if `false` (default), replaces them with `\uFFFD`
- **options.ignoreBOM** - if `true`, ignores Byte Order Mark

### Usage Examples

```javascript
// Basic decoding
let uint8Array = new Uint8Array([72, 101, 108, 108, 111]);
console.log(new TextDecoder().decode(uint8Array)); // "Hello"

// UTF-8 Chinese characters
let chineseBytes = new Uint8Array([228, 189, 160, 229, 165, 189]);
console.log(new TextDecoder().decode(chineseBytes)); // "你好"

// Decode part of buffer using subarray
let data = new Uint8Array([0, 72, 101, 108, 108, 111, 0]);
let middle = data.subarray(1, -1); // [72, 101, 108, 108, 111]
console.log(new TextDecoder().decode(middle)); // "Hello"
```

### Streaming Decode

Use `stream: true` when decoding data in chunks:

```javascript
let decoder = new TextDecoder();
let chunk1 = new Uint8Array([0xe2, 0x82]); // partial character
let chunk2 = new Uint8Array([0xac]); // rest of character

let partial = decoder.decode(chunk1, { stream: true });
console.log(partial); // "" (incomplete, stored internally)

let complete = decoder.decode(chunk2);
console.log(complete); // "€" (Euro symbol)
```

## TextEncoder (String → Bytes)

Converts strings to bytes (UTF-8 only).

```javascript
let encoder = new TextEncoder();
let uint8Array = encoder.encode(str);
```

### Methods

- **encode(str)** - returns `Uint8Array`
- **encodeInto(str, destination)** - encodes directly into existing `Uint8Array`

### Usage Examples

```javascript
let encoder = new TextEncoder();

// Basic encoding
let bytes = encoder.encode("Hello");
console.log(bytes); // Uint8Array [72, 101, 108, 108, 111]

// Encoding special characters
let euroBytes = encoder.encode("€");
console.log(euroBytes); // Uint8Array [226, 130, 172] (3 bytes)

// encodeInto example
let dest = new Uint8Array(10);
let result = encoder.encodeInto("Hello", dest);
console.log(dest); // Uint8Array [72, 101, 108, 108, 111, 0, 0, 0, 0, 0]
console.log(result.read); // 5 (characters written)
console.log(result.written); // 5 (bytes written)
```

## Common Use Cases

### File Reading

```javascript
async function readTextFile(file) {
  const arrayBuffer = await file.arrayBuffer();
  const bytes = new Uint8Array(arrayBuffer);
  return new TextDecoder("utf-8").decode(bytes);
}
```

### Fetch Response

```javascript
async function fetchText(url) {
  const response = await fetch(url);
  const buffer = await response.arrayBuffer();
  return new TextDecoder("utf-8").decode(buffer);
}
```

### Sending Text as Binary

```javascript
function sendText(str) {
  const encoder = new TextEncoder();
  const bytes = encoder.encode(str);
  // Send bytes via WebSocket, fetch, etc.
  socket.send(bytes);
}
```

## Summary

| Object        | Direction      | Encodings                       | Output     |
| ------------- | -------------- | ------------------------------- | ---------- |
| `TextDecoder` | Bytes → String | UTF-8, Big5, Windows-1251, etc. | String     |
| `TextEncoder` | String → Bytes | UTF-8 only                      | Uint8Array |
