# Fetch: Abort Summary

`AbortController` allows canceling ongoing `fetch()` requests and other asynchronous operations.

## AbortController Basics

```javascript
let controller = new AbortController();
let signal = controller.signal;

// Listen for abort event
signal.addEventListener("abort", () => {
  console.log("Operation aborted!");
  console.log(signal.aborted); // true
});

// Trigger abort
controller.abort();
```

**Properties:**

- `controller.abort()` - triggers the abort event
- `controller.signal` - the signal object to pass to cancelable operations
- `signal.aborted` - becomes `true` after `abort()` is called

## Aborting a Fetch Request

```javascript
let controller = new AbortController();

try {
  let response = await fetch("/api/data", {
    signal: controller.signal, // Pass the signal
  });
  let data = await response.json();
} catch (err) {
  if (err.name === "AbortError") {
    console.log("Fetch was aborted");
  } else {
    throw err;
  }
}

// Abort after 1 second
setTimeout(() => controller.abort(), 1000);
```

## Abort with Timeout

```javascript
async function fetchWithTimeout(url, timeoutMs) {
  let controller = new AbortController();

  // Abort after timeout
  let timeoutId = setTimeout(() => controller.abort(), timeoutMs);

  try {
    let response = await fetch(url, {
      signal: controller.signal,
    });
    clearTimeout(timeoutId);
    return await response.json();
  } catch (err) {
    if (err.name === "AbortError") {
      throw new Error("Request timed out");
    }
    throw err;
  }
}

// Usage
try {
  let data = await fetchWithTimeout("/api/slow-endpoint", 3000);
  console.log(data);
} catch (err) {
  console.log(err.message); // "Request timed out"
}
```

## Aborting Multiple Fetches

```javascript
let urls = [
  "https://api.github.com/users/iliakan",
  "https://api.github.com/users/remy",
  "https://api.github.com/users/john",
];

let controller = new AbortController();

// All fetches share the same signal
let fetchJobs = urls.map((url) =>
  fetch(url, { signal: controller.signal }).then((res) => res.json()),
);

// Abort all fetches at once
setTimeout(() => controller.abort(), 100);

try {
  let results = await Promise.all(fetchJobs);
  console.log(results);
} catch (err) {
  if (err.name === "AbortError") {
    console.log("All fetches aborted");
  }
}
```

## Combining with Custom Async Tasks

```javascript
let controller = new AbortController();

// Custom async task that listens for abort
let customTask = new Promise((resolve, reject) => {
  let timer = setTimeout(() => resolve("Done!"), 5000);

  controller.signal.addEventListener("abort", () => {
    clearTimeout(timer);
    reject(new Error("Task aborted"));
  });
});

// Fetch with same signal
let fetchTask = fetch("/api/data", {
  signal: controller.signal,
});

// Run both in parallel
try {
  let [customResult, fetchResponse] = await Promise.all([
    customTask,
    fetchTask,
  ]);
  console.log(customResult, await fetchResponse.json());
} catch (err) {
  if (err.name === "AbortError") {
    console.log("Both tasks aborted");
  }
}

// Abort everything
controller.abort();
```

## Cleanup Helper Function

```javascript
function makeCancelable(promise, signal) {
  return new Promise((resolve, reject) => {
    if (signal.aborted) {
      reject(new Error("Aborted"));
      return;
    }

    const abortHandler = () => reject(new Error("Aborted"));
    signal.addEventListener("abort", abortHandler);

    promise
      .then(resolve)
      .catch(reject)
      .finally(() => {
        signal.removeEventListener("abort", abortHandler);
      });
  });
}

// Usage
let controller = new AbortController();
let fetchPromise = fetch("/api/data").then((res) => res.json());

try {
  let result = await makeCancelable(fetchPromise, controller.signal);
  console.log(result);
} catch (err) {
  if (err.message === "Aborted") {
    console.log("Operation was cancelled");
  }
}

// Cancel if needed
controller.abort();
```

## AbortController with Race Condition

```javascript
// Abort the slower request
async function raceWithTimeout(promise, timeoutMs) {
  let controller = new AbortController();

  let timeoutPromise = new Promise((_, reject) => {
    setTimeout(() => {
      controller.abort();
      reject(new Error("Timeout"));
    }, timeoutMs);
  });

  // Make promise abort-aware if it supports signals
  let result = await Promise.race([promise, timeoutPromise]);
  return result;
}
```

## Summary

| Feature              | Description                                                          |
| -------------------- | -------------------------------------------------------------------- |
| **Controller**       | `new AbortController()` creates controller                           |
| **Signal**           | Pass `signal` to fetch or custom tasks                               |
| **Abort**            | `controller.abort()` cancels all operations using that signal        |
| **Error handling**   | Catch `AbortError` (err.name === 'AbortError')                       |
| **Multiple fetches** | One controller can abort many fetches simultaneously                 |
| **Custom tasks**     | Listen to `signal.addEventListener('abort')` for custom cancellation |

**Key Points:**

- `fetch()` integrates natively with `AbortController`
- The aborted promise rejects with `AbortError`
- Can abort multiple operations with a single controller
- Works with any async operation by listening to abort event
- Always handle abort errors to prevent unhandled rejections
