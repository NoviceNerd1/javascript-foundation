# Async Iteration and Generators

## Overview

Async iteration allows iterating over data that arrives asynchronously (e.g., network requests, delays). Async generators combine generator syntax with async/await.

## Async Iterables

To make an object asynchronously iterable:

- Use `Symbol.asyncIterator` instead of `Symbol.iterator`
- `next()` must return a promise
- Use `for await...of` to iterate

```javascript
let range = {
  from: 1,
  to: 5,

  [Symbol.asyncIterator]() {
    return {
      current: this.from,
      last: this.to,

      async next() {
        await new Promise((resolve) => setTimeout(resolve, 1000));

        if (this.current <= this.last) {
          return { done: false, value: this.current++ };
        } else {
          return { done: true };
        }
      },
    };
  },
};

(async () => {
  for await (let value of range) {
    alert(value); // 1,2,3,4,5 (1 second between)
  }
})();
```

### Comparison: Regular vs Async Iterators

| Feature          | Regular Iterator  | Async Iterator         |
| ---------------- | ----------------- | ---------------------- |
| Method           | `Symbol.iterator` | `Symbol.asyncIterator` |
| `next()` returns | value             | Promise                |
| Loop             | `for..of`         | `for await..of`        |

## Async Generators

Async generators combine `async` with `function*`:

```javascript
async function* generateSequence(start, end) {
  for (let i = start; i <= end; i++) {
    await new Promise((resolve) => setTimeout(resolve, 1000));
    yield i;
  }
}

(async () => {
  for await (let value of generateSequence(1, 5)) {
    alert(value); // 1,2,3,4,5 (with delays)
  }
})();
```

### Key Difference

| Feature          | Regular Generator | Async Generator          |
| ---------------- | ----------------- | ------------------------ |
| Declaration      | `function*`       | `async function*`        |
| `next()` returns | `{value, done}`   | `Promise<{value, done}>` |

## Async Generator as Symbol.asyncIterator

```javascript
let range = {
  from: 1,
  to: 5,

  async *[Symbol.asyncIterator]() {
    for (let value = this.from; value <= this.to; value++) {
      await new Promise((resolve) => setTimeout(resolve, 1000));
      yield value;
    }
  },
};

(async () => {
  for await (let value of range) {
    alert(value); // 1,2,3,4,5
  }
})();
```

## Real-Life Example: Paginated Data

Fetching paginated commits from GitHub:

```javascript
async function* fetchCommits(repo) {
  let url = `https://api.github.com/repos/${repo}/commits`;

  while (url) {
    const response = await fetch(url, {
      headers: { "User-Agent": "Our script" },
    });

    const body = await response.json();

    // Extract next page URL from Link header
    let nextPage = response.headers.get("Link")?.match(/<(.*?)>; rel="next"/);
    url = nextPage?.[1];

    for (let commit of body) {
      yield commit;
    }
  }
}

// Usage
(async () => {
  let count = 0;
  for await (const commit of fetchCommits(
    "javascript-tutorial/en.javascript.info",
  )) {
    console.log(commit.author.login);
    if (++count == 100) break;
  }
})();
```

## Summary

| Concept             | Regular           | Async                  |
| ------------------- | ----------------- | ---------------------- |
| **Iterable**        | `Symbol.iterator` | `Symbol.asyncIterator` |
| **Iterator next()** | returns value     | returns Promise        |
| **Loop**            | `for..of`         | `for await..of`        |
| **Generator**       | `function*`       | `async function*`      |
| **Spread (`...`)**  | âś… works          | âťŚ not supported       |

**Key Points:**

- Use `Symbol.asyncIterator` and `for await..of` for asynchronous iteration
- Async generators (`async function*`) allow `await` inside and yield promises
- Perfect for paginated API data, streams, or any chunked asynchronous data
- Cannot use spread operator (`...`) with async iterables
