# Conditional Branching in JavaScript

## `if` Statement

Executes a code block if condition is `true`.

```javascript
if (year == 2015) {
  alert("You are right!");
}
```

### Boolean Conversion

- **Falsy values**: `0`, `""`, `null`, `undefined`, `NaN` → treated as `false`
- **Truthy values**: everything else → treated as `true`

```javascript
if (0) {
} // never executes
if (1) {
} // always executes
```

## `else` and `else if`

```javascript
if (year < 2015) {
  alert("Too early");
} else if (year > 2015) {
  alert("Too late");
} else {
  alert("Exactly!");
}
```

## Conditional Operator `?` (Ternary)

Syntax: `let result = condition ? value1 : value2;`

```javascript
let accessAllowed = age > 18 ? true : false;
```

### Multiple Conditions

```javascript
let message =
  age < 3
    ? "Hi, baby!"
    : age < 18
      ? "Hello!"
      : age < 100
        ? "Greetings!"
        : "What an unusual age!";
```

## Tasks

### 1. String with Zero

```javascript
if ("0") {
  alert("Hello"); // Executes - non-empty string is truthy
}
```

### 2. Official JavaScript Name

```javascript
let name = prompt("What is the official name of JavaScript?", "");
if (name == "ECMAScript") {
  alert("Right!");
} else {
  alert("You don't know? ECMAScript!");
}
```

### 3. Show Sign

```javascript
let value = prompt("Enter a number:", 0);
if (value > 0) {
  alert(1);
} else if (value < 0) {
  alert(-1);
} else {
  alert(0);
}
```

### 4. Rewrite `if` to `?`

```javascript
let result = a + b < 4 ? "Below" : "Over";
```

### 5. Rewrite `if..else` to `?`

```javascript
let message =
  login == "Employee"
    ? "Hello"
    : login == "Director"
      ? "Greetings"
      : login == ""
        ? "No login"
        : "";
```

> **Note**: Use `?` only for returning values. Use `if` for executing different code branches.
