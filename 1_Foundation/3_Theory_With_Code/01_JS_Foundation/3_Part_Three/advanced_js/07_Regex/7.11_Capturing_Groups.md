# Capturing Groups Summary

Parentheses `(...)` create **capturing groups** that allow extracting matched substrings and applying quantifiers to groups.

## Basic Capturing Groups

```javascript
// Without parentheses: go+ matches "g" + one or more "o"
console.log("Gogogo".match(/go+/i)); // ["gogo"]

// With parentheses: (go)+ matches "go" repeated
console.log("Gogogo".match(/(go)+/i)); // ["Gogogo", "go"]
```

## Accessing Groups in Match

Without `g` flag, `match()` returns array with groups:

```javascript
let str = "<h1>Hello!</h1>";
let tag = str.match(/<(.*?)>/);

console.log(tag[0]); // "<h1>" (full match)
console.log(tag[1]); // "h1" (first group)
```

## Nested Groups

Groups are numbered left-to-right by opening parentheses:

```javascript
let str = '<span class="my">';
let regexp = /<(([a-z]+)\s*([^>]*))>/;
let result = str.match(regexp);

console.log(result[0]); // "<span class=\"my\">"
console.log(result[1]); // "span class=\"my\""
console.log(result[2]); // "span"
console.log(result[3]); // "class=\"my\""
```

## Named Groups `(?<name>...)`

```javascript
let dateRegexp = /(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/;
let str = "2019-04-30";
let match = str.match(dateRegexp);

console.log(match.groups.year); // "2019"
console.log(match.groups.month); // "04"
console.log(match.groups.day); // "30"
```

## matchAll for Global Search with Groups

```javascript
let str = "<h1> <h2>";
let results = str.matchAll(/<(.*?)>/gi);

// Convert to array or iterate
let matches = Array.from(results);
console.log(matches[0]); // ["<h1>", "h1"]
console.log(matches[1]); // ["<h2>", "h2"]

// Iterate directly
for (let match of str.matchAll(/<(.*?)>/gi)) {
  console.log(match[0]); // full tag
  console.log(match[1]); // tag content
}
```

## Groups in Replacement `$n` and `$<name>`

```javascript
// By number
let str = "John Bull";
console.log(str.replace(/(\w+) (\w+)/, "$2, $1")); // "Bull, John"

// By name
let dateRegexp = /(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/g;
let dateStr = "2019-10-30, 2020-01-01";
console.log(dateStr.replace(dateRegexp, "$<day>.$<month>.$<year>"));
// "30.10.2019, 01.01.2020"
```

## Non-Capturing Groups `(?:...)`

Use when you need grouping but don't want the result in the array:

```javascript
let str = "Gogogo John!";
let regexp = /(?:go)+ (\w+)/i;
let result = str.match(regexp);

console.log(result[0]); // "Gogogo John"
console.log(result[1]); // "John"
console.log(result.length); // 2 (no extra group for "go")
```

## Optional Groups

Optional groups appear as `undefined` when missing:

```javascript
let match = "a".match(/a(z)?(c)?/);
console.log(match); // ["a", undefined, undefined]

match = "ac".match(/a(z)?(c)?/);
console.log(match); // ["ac", undefined, "c"]
```

## Practical Examples

### Domain Name

```javascript
let regexp = /(\w+\.)+\w+/g;
console.log("site.com my.site.com".match(regexp));
// ["site.com", "my.site.com"]
```

### Email (Basic)

```javascript
let regexp = /[-.\w]+@([\w-]+\.)+[\w-]+/g;
console.log("my@mail.com his@site.com.uk".match(regexp));
// ["my@mail.com", "his@site.com.uk"]
```

## Tasks

### Task 1: Check MAC-Address

```javascript
let regexp = /^([0-9A-Fa-f]{2}:){5}[0-9A-Fa-f]{2}$/;

console.log(regexp.test("01:32:54:67:89:AB")); // true
console.log(regexp.test("0132546789AB")); // false
console.log(regexp.test("01:32:54:67:89")); // false
console.log(regexp.test("01:32:54:67:89:ZZ")); // false
```

### Task 2: Find Color (#abc or #abcdef)

```javascript
let regexp = /#([0-9A-Fa-f]{3}|[0-9A-Fa-f]{6})\b/g;
let str = "color: #3f3; background-color: #AA00ef; and: #abcd";

console.log(str.match(regexp)); // ["#3f3", "#AA00ef"]
```

### Task 3: Find All Numbers

```javascript
let regexp = /-?\d+(?:\.\d+)?/g;
let str = "-1.5 0 2 -123.4.";

console.log(str.match(regexp)); // ["-1.5", "0", "2", "-123.4"]
```

### Task 4: Parse Expression

```javascript
function parse(expr) {
  let regexp = /^\s*(-?\d+(?:\.\d+)?)\s*([+\-*/])\s*(-?\d+(?:\.\d+)?)\s*$/;
  let match = expr.match(regexp);
  if (!match) return null;
  return [parseFloat(match[1]), match[2], parseFloat(match[3])];
}

let [a, op, b] = parse("1.2 * 3.4");
console.log(a, op, b); // 1.2, "*", 3.4
```

## Summary Table

| Syntax         | Description               | Example                     |
| -------------- | ------------------------- | --------------------------- |
| `(...)`        | Capturing group           | `/(go)+/` captures "go"     |
| `(?<name>...)` | Named group               | `/(?<year>\d{4})/`          |
| `(?:...)`      | Non-capturing group       | `/(?:go)+/`                 |
| `$n`           | Replacement by number     | `'$2, $1'`                  |
| `$<name>`      | Replacement by name       | `'$<day>.$<month>'`         |
| `matchAll()`   | Global search with groups | Returns iterable of matches |

**Key Points:**

- Groups are numbered left-to-right by opening parentheses
- Named groups accessed via `match.groups`
- Use `matchAll()` for global search with groups
- Non-capturing groups `(?:...)` exclude from results
- Optional groups return `undefined` when absent
- Use `$n` and `$<name>` in `replace()` for group references
