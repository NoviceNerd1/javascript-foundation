# Property Flags and Descriptors

## Overview

Object properties have three hidden attributes (flags) beyond their value:

- **`writable`** – can the value be changed?
- **`enumerable`** – appears in loops?
- **`configurable`** – can the property be deleted or flags changed?

By default, all flags are `true` when creating properties normally.

## Getting Property Descriptor

```javascript
let user = { name: "John" };
let descriptor = Object.getOwnPropertyDescriptor(user, "name");

console.log(descriptor);
// { value: "John", writable: true, enumerable: true, configurable: true }
```

## Changing Flags with `defineProperty`

```javascript
Object.defineProperty(obj, propertyName, descriptor);
```

### Example: Non-writable

```javascript
let user = { name: "John" };
Object.defineProperty(user, "name", { writable: false });

user.name = "Pete"; // Error (strict mode)
```

### Example: Non-enumerable

```javascript
let user = {
  name: "John",
  toString() {
    return this.name;
  },
};

Object.defineProperty(user, "toString", { enumerable: false });

for (let key in user) alert(key); // name (toString hidden)
alert(Object.keys(user)); // ["name"]
```

### Example: Non-configurable

```javascript
let user = { name: "John" };
Object.defineProperty(user, "name", { configurable: false });

delete user.name; // Error
// Can't change flags either
Object.defineProperty(user, "name", { writable: false }); // Error

// But can still change value if writable is true
user.name = "Pete"; // Works
```

### Making a Sealed Constant

```javascript
let user = { name: "John" };
Object.defineProperty(user, "name", {
  writable: false,
  configurable: false,
});

// All these fail:
user.name = "Pete";
delete user.name;
Object.defineProperty(user, "name", { value: "Pete" });
```

## Creating Properties with Flags

When creating a new property with `defineProperty`, missing flags default to `false`:

```javascript
let user = {};
Object.defineProperty(user, "name", { value: "John" });
// writable, enumerable, configurable are all false

let descriptor = Object.getOwnPropertyDescriptor(user, "name");
// { value: "John", writable: false, enumerable: false, configurable: false }
```

## Defining Multiple Properties

```javascript
Object.defineProperties(user, {
  name: { value: "John", writable: false },
  surname: { value: "Smith", writable: false },
});
```

## Getting All Descriptors

```javascript
let descriptors = Object.getOwnPropertyDescriptors(user);
```

### Flags-Aware Cloning

```javascript
let clone = Object.defineProperties({}, Object.getOwnPropertyDescriptors(obj));
// Copies all properties WITH their flags
```

## Global Object Sealing

| Method                          | Effect                                                                 |
| ------------------------------- | ---------------------------------------------------------------------- |
| `Object.preventExtensions(obj)` | No new properties can be added                                         |
| `Object.seal(obj)`              | No add/remove; `configurable: false` for all properties                |
| `Object.freeze(obj)`            | No add/remove/change; `configurable: false`, `writable: false` for all |

### Testing Methods

| Method                     | Checks                       |
| -------------------------- | ---------------------------- |
| `Object.isExtensible(obj)` | Can new properties be added? |
| `Object.isSealed(obj)`     | Are add/remove forbidden?    |
| `Object.isFrozen(obj)`     | Are all changes forbidden?   |

## Summary

| Flag           | When `false`                                          |
| -------------- | ----------------------------------------------------- |
| `writable`     | Value cannot be changed                               |
| `enumerable`   | Property hidden from loops (`for..in`, `Object.keys`) |
| `configurable` | Property cannot be deleted, flags cannot be changed   |

**Key Points:**

- Use `Object.getOwnPropertyDescriptor` to read flags
- Use `Object.defineProperty` to modify flags
- New properties created with `defineProperty` have flags default to `false`
- `Object.freeze` makes an object fully immutable
- Flags-aware cloning preserves property descriptors
