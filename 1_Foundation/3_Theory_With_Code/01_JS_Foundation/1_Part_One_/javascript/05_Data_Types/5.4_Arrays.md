# Arrays in JavaScript

## Overview

Arrays store **ordered collections** of data. They are special objects with numeric indexes and a `length` property.

## Creating Arrays

```javascript
let arr = new Array(); // Constructor syntax (rare)
let arr = []; // Array literal syntax (preferred)

let fruits = ["Apple", "Orange", "Plum"];
```

## Accessing Elements

```javascript
let fruits = ["Apple", "Orange", "Plum"];

alert(fruits[0]); // Apple
alert(fruits.at(0)); // Apple
alert(fruits.at(-1)); // Plum (negative counts from end)
alert(fruits.length); // 3

// Modify
fruits[2] = "Pear"; // ["Apple", "Orange", "Pear"]
fruits[3] = "Lemon"; // ["Apple", "Orange", "Pear", "Lemon"]
```

## Array Methods (Queue/Stack Operations)

| Method          | Description                        | Example                   |
| --------------- | ---------------------------------- | ------------------------- |
| `push(item)`    | Adds to end                        | `fruits.push("Pear")`     |
| `pop()`         | Removes from end, returns it       | `fruits.pop()`            |
| `unshift(item)` | Adds to beginning                  | `fruits.unshift("Apple")` |
| `shift()`       | Removes from beginning, returns it | `fruits.shift()`          |

```javascript
let fruits = ["Apple", "Orange", "Pear"];

fruits.pop(); // Removes "Pear"
alert(fruits); // ["Apple", "Orange"]

fruits.push("Lemon"); // Adds "Lemon"
alert(fruits); // ["Apple", "Orange", "Lemon"]

fruits.shift(); // Removes "Apple"
alert(fruits); // ["Orange", "Lemon"]

fruits.unshift("Kiwi"); // Adds "Kiwi" to beginning
alert(fruits); // ["Kiwi", "Orange", "Lemon"]

// Multiple elements
fruits.push("Mango", "Banana");
fruits.unshift("Grape", "Melon");
```

### Performance

- `push`/`pop` – **fast** (operate at end)
- `unshift`/`shift` – **slow** (need to reindex all elements)

## Looping Over Arrays

```javascript
let arr = ["Apple", "Orange", "Pear"];

// Classic for loop (fastest)
for (let i = 0; i < arr.length; i++) {
  alert(arr[i]);
}

// for...of (values only, modern)
for (let fruit of arr) {
  alert(fruit);
}

// for...in (NOT recommended for arrays)
for (let key in arr) {
  // Avoid: includes non-numeric props, slower
  alert(arr[key]);
}
```

## Array Internals

Arrays are objects and are **copied by reference**:

```javascript
let fruits = ["Banana"];
let arr = fruits; // Copy reference

arr.push("Pear");
alert(fruits); // ["Banana", "Pear"] (same array)
```

### Arrays vs Regular Objects

Arrays should be used as **ordered collections**. Avoid:

```javascript
let arr = [];
arr[99999] = 5; // Creates holes (bad for performance)
arr.age = 25; // Non-numeric property (turns off optimizations)
```

## The `length` Property

`length` = greatest numeric index + 1 (not necessarily element count):

```javascript
let fruits = [];
fruits[123] = "Apple";
alert(fruits.length); // 124
```

### `length` is Writable

```javascript
let arr = [1, 2, 3, 4, 5];
arr.length = 2; // Truncates to [1, 2]
alert(arr); // [1, 2]

arr.length = 5; // Restores length, but values lost
alert(arr[3]); // undefined

arr.length = 0; // Clears the array
```

## `new Array()` Pitfall

```javascript
let arr = new Array(2); // Creates array of length 2, empty!
alert(arr[0]); // undefined
alert(arr.length); // 2

// Use array literal to avoid confusion
let arr = [2]; // Array with one element: 2
```

## Multidimensional Arrays

```javascript
let matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9],
];

alert(matrix[0][1]); // 2
```

## `toString()` for Arrays

```javascript
let arr = [1, 2, 3];
alert(arr); // "1,2,3"
alert(String(arr)); // "1,2,3"

alert([] + 1); // "1"
alert([1] + 1); // "11"
alert([1, 2] + 1); // "1,21"
```

## Comparing Arrays

**Never use `==` or `===` for arrays:**

```javascript
alert([] == []); // false (different objects)
alert([0] == [0]); // false

// Compare item-by-item manually instead
```

## Summary Table

| Operation             | Method                                                        |
| --------------------- | ------------------------------------------------------------- |
| Create                | `[]` or `new Array()`                                         |
| Access                | `arr[i]` or `arr.at(i)`                                       |
| Add to end            | `push(item)`                                                  |
| Remove from end       | `pop()`                                                       |
| Add to beginning      | `unshift(item)`                                               |
| Remove from beginning | `shift()`                                                     |
| Length                | `arr.length`                                                  |
| Loop                  | `for (let i=0; i<arr.length; i++)` or `for (let item of arr)` |

## Tasks

### 1. Is Array Copied?

```javascript
let fruits = ["Apples", "Pear", "Orange"];
let shoppingCart = fruits;
shoppingCart.push("Banana");
alert(fruits.length); // 4 (arrays are copied by reference)
```

### 2. Array Operations

```javascript
let styles = ["Jazz", "Blues"];
styles.push("Rock-n-Roll");
styles[Math.floor((styles.length - 1) / 2)] = "Classics";
alert(styles.shift());
styles.unshift("Rap", "Reggae");
// Result: ["Rap", "Reggae", "Classics", "Rock-n-Roll"]
```

### 3. Calling in Array Context

```javascript
let arr = ["a", "b"];
arr.push(function () {
  alert(this);
});
arr[2](); // Alerts the array itself (this = arr)
```

### 4. Sum Input Numbers

```javascript
function sumInput() {
  let numbers = [];
  while (true) {
    let value = prompt("Enter a number:", 0);
    if (value === "" || value === null || isNaN(value)) break;
    numbers.push(+value);
  }
  let sum = 0;
  for (let num of numbers) sum += num;
  return sum;
}
```

### 5. Maximal Subarray

```javascript
function getMaxSubSum(arr) {
  let maxSum = 0;
  let partialSum = 0;
  for (let item of arr) {
    partialSum += item;
    maxSum = Math.max(maxSum, partialSum);
    if (partialSum < 0) partialSum = 0;
  }
  return maxSum;
}
// Kadane's algorithm - O(n) complexity
```
