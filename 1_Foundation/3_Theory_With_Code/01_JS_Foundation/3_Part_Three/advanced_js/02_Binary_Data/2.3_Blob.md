# Blob Summary

`Blob` (Binary Large Object) represents binary data with a MIME type. It's a browser-specific object built on top of `ArrayBuffer` and views.

## Constructor

```javascript
new Blob(blobParts, options);
```

- **blobParts** - array of `Blob`/`BufferSource`/`String` values
- **options.type** - MIME type (e.g., `'image/png'`, `'text/plain'`)
- **options.endings** - line ending transformation (`'transparent'` or `'native'`)

```javascript
// From string
let blob = new Blob(["<html>...</html>"], { type: "text/html" });

// From mixed sources
let hello = new Uint8Array([72, 101, 108, 108, 111]);
let blob = new Blob([hello, " ", "world"], { type: "text/plain" });
```

## Methods

### slice()

```javascript
blob.slice([byteStart], [byteEnd], [contentType]);
```

Creates a new blob from a portion of the original (immutable operation).

### arrayBuffer()

```javascript
let buffer = await blob.arrayBuffer();
```

Converts blob to `ArrayBuffer` for low-level processing.

### stream()

```javascript
let stream = blob.stream();
```

Returns `ReadableStream` for processing large blobs chunk by chunk.

### text()

```javascript
let text = await blob.text();
```

Converts blob directly to string (UTF-8).

## Blob as URL

### URL.createObjectURL()

Creates temporary URL for blob:

```javascript
let blob = new Blob(["Hello, world!"], { type: "text/plain" });

// Create download link
let link = document.createElement("a");
link.download = "hello.txt";
link.href = URL.createObjectURL(blob);
link.click();

// Clean up to free memory
URL.revokeObjectURL(link.href);
```

URL format: `blob:https://origin/uuid`

**Important:** Always revoke URLs when no longer needed to free memory.

### Data URL (Base64)

Alternative using `FileReader`:

```javascript
let blob = new Blob(["Hello, world!"], { type: "text/plain" });
let reader = new FileReader();

reader.readAsDataURL(blob);
reader.onload = function () {
  let dataUrl = reader.result; // data:text/plain;base64,SGVsbG8sIHdvcmxkIQ==
  link.href = dataUrl;
  link.click();
};
```

## Comparison

| Method                  | Pros                 | Cons                                                |
| ----------------------- | -------------------- | --------------------------------------------------- |
| `URL.createObjectURL()` | Simple, fast, direct | Must manually revoke                                |
| Data URL                | No cleanup needed    | Encoding overhead, memory intensive for large blobs |

## Common Operations

### Image to Blob

```javascript
let img = document.querySelector("img");
let canvas = document.createElement("canvas");
canvas.width = img.clientWidth;
canvas.height = img.clientHeight;

let ctx = canvas.getContext("2d");
ctx.drawImage(img, 0, 0);

// Callback style
canvas.toBlob((blob) => {
  // upload or download blob
  console.log(blob);
}, "image/png");

// Promise style
let blob = await new Promise((resolve) => canvas.toBlob(resolve, "image/png"));
```

### Blob to ArrayBuffer

```javascript
let blob = new Blob(["Hello"]);
let buffer = await blob.arrayBuffer();
let view = new Uint8Array(buffer);
console.log(view); // [72, 101, 108, 108, 111]
```

### Blob to Stream (Large Files)

```javascript
let blob = new Blob(["Very large content..."]);
let stream = blob.stream();
let reader = stream.getReader();

while (true) {
  let { done, value } = await reader.read();
  if (done) break;
  console.log("Chunk:", value); // Uint8Array chunks
}
```

### Fetch with Blob

```javascript
// Download as blob
let response = await fetch("image.png");
let blob = await response.blob();

// Upload blob
let formData = new FormData();
formData.append("file", blob, "filename.png");
await fetch("/upload", { method: "POST", body: formData });
```

## Summary

| Feature          | Description                                             |
| ---------------- | ------------------------------------------------------- |
| **Blob**         | Binary data with MIME type                              |
| **Immutable**    | Cannot modify directly, create new blobs with `slice()` |
| **URL creation** | `URL.createObjectURL()` for temporary URLs              |
| **Conversion**   | `arrayBuffer()` → low-level, `text()` → string          |
| **Streams**      | `stream()` for large data processing                    |
| **Common use**   | File upload/download, image processing, fetch responses |
