# BigInt

## Overview

**BigInt** is a numeric type for integers of arbitrary length. It's useful when numbers exceed the safe limit of regular numbers (2⁵³ - 1).

## Creating BigInts

```javascript
// Append 'n' to integer literal
const bigint = 1234567890123456789012345678901234567890n;

// Using BigInt() function
const sameBigint = BigInt("1234567890123456789012345678901234567890");
const fromNumber = BigInt(10); // 10n
```

## Math Operations

```javascript
// Operations with bigints
alert(1n + 2n); // 3n
alert(5n / 2n); // 2n (rounded toward zero)

// Cannot mix bigint and number
alert(1n + 2); // Error: Cannot mix BigInt and other types

// Explicit conversion needed
let bigint = 1n;
let number = 2;

// Number → BigInt
alert(bigint + BigInt(number)); // 3n

// BigInt → Number (be careful with large values)
alert(Number(bigint) + number); // 3
```

### Unary Plus Not Supported

```javascript
let bigint = 1n;
alert(+bigint); // Error: Cannot convert BigInt to number
// Use Number() instead
alert(Number(bigint)); // 1
```

## Comparisons

```javascript
// Works with numbers
alert(2n > 1n); // true
alert(2n > 1); // true

// Equality vs Strict equality
alert(1 == 1n); // true (value equality)
alert(1 === 1n); // false (different types)
```

## Boolean Operations

```javascript
// 0n is falsy, others are truthy
if (0n) {
  // never executes
}

// Logical operators work like numbers
alert(1n || 2); // 1n (truthy)
alert(0n || 2); // 2  (0n is falsy)
```

## Browser Support

Check support at: [caniuse.com/bigint](https://caniuse.com/bigint)

## Polyfills

No good polyfill exists because operators behave differently. Alternative approach using JSBI library:

```javascript
// Instead of native BigInt
let a = BigInt(789);
let c = a + b;

// Use JSBI (writes code that can be converted later)
let a = JSBI.BigInt(789);
let c = JSBI.add(a, b);
```

Then use a Babel plugin to convert JSBI calls to native bigints for supported browsers.

## Summary

| Feature          | Behavior                                 |
| ---------------- | ---------------------------------------- |
| Creation         | Append `n` or `BigInt()`                 |
| Mixed operations | ❌ Not allowed – must convert explicitly |
| Division         | Rounds toward zero, returns bigint       |
| Unary `+`        | ❌ Not supported                         |
| `==`             | Works (value equality)                   |
| `===`            | False (different types)                  |
| Boolean          | `0n` is falsy, others truthy             |

**Key Points:**

- BigInts are for arbitrarily large integers
- Cannot mix with regular numbers without explicit conversion
- Use `BigInt()` or `Number()` to convert between types
- Be careful converting large bigints to numbers (precision loss)
- No good polyfill exists; consider JSBI library for compatibility
