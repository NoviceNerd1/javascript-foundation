# FormData Summary

`FormData` objects represent HTML form data for sending via `fetch()` or other network methods. They automatically use `multipart/form-data` encoding.

## Constructor

```javascript
let formData = new FormData([form]);
```

If a form element is provided, it automatically captures all fields.

## Sending a Simple Form

```html
<form id="formElem">
  <input type="text" name="name" value="John" />
  <input type="text" name="surname" value="Smith" />
  <input type="submit" />
</form>

<script>
  formElem.onsubmit = async (e) => {
    e.preventDefault();

    let response = await fetch("/api/user", {
      method: "POST",
      body: new FormData(formElem),
    });

    let result = await response.json();
    alert(result.message);
  };
</script>
```

## FormData Methods

| Method                         | Description                                        |
| ------------------------------ | -------------------------------------------------- |
| `append(name, value)`          | Add field with name and value                      |
| `append(name, blob, fileName)` | Add file field (like `<input type="file">`)        |
| `set(name, value)`             | Set field, removing existing fields with same name |
| `set(name, blob, fileName)`    | Set file field, removing existing                  |
| `delete(name)`                 | Remove field with given name                       |
| `get(name)`                    | Get value of field                                 |
| `has(name)`                    | Check if field exists                              |

### append vs set

```javascript
let fd = new FormData();

fd.append("color", "red");
fd.append("color", "blue");
fd.getAll("color"); // ['red', 'blue']

fd.set("color", "green");
fd.getAll("color"); // ['green'] (replaces all)
```

### Iterating FormData

```javascript
let formData = new FormData();
formData.append("key1", "value1");
formData.append("key2", "value2");

// for...of loop
for (let [name, value] of formData) {
  console.log(`${name} = ${value}`);
}

// Using entries()
for (let [name, value] of formData.entries()) {
  console.log(name, value);
}
```

## Sending Forms with Files

```html
<form id="formElem">
  <input type="text" name="firstName" value="John" />
  <input type="file" name="picture" accept="image/*" />
  <input type="submit" />
</form>

<script>
  formElem.onsubmit = async (e) => {
    e.preventDefault();

    let response = await fetch("/api/user-avatar", {
      method: "POST",
      body: new FormData(formElem), // Includes the file automatically
    });

    let result = await response.json();
    alert(result.message);
  };
</script>
```

## Sending Blob Data as File

```javascript
async function submit() {
  // Get blob from canvas
  let canvas = document.getElementById("canvasElem");
  let imageBlob = await new Promise((resolve) =>
    canvas.toBlob(resolve, "image/png"),
  );

  // Create form with additional fields
  let formData = new FormData();
  formData.append("firstName", "John");
  formData.append("image", imageBlob, "image.png"); // 3rd arg = filename

  let response = await fetch("/api/image-form", {
    method: "POST",
    body: formData,
  });

  let result = await response.json();
  alert(result.message);
}
```

## Manual FormData Construction

```javascript
// Create without form element
let formData = new FormData();

formData.append("username", "john_doe");
formData.append("age", 30);
formData.append("avatar", fileBlob, "avatar.jpg");

// Check fields
console.log(formData.has("username")); // true
console.log(formData.get("username")); // 'john_doe'

// Remove field
formData.delete("age");

// Send
let response = await fetch("/api/profile", {
  method: "POST",
  body: formData,
});
```

## Multiple Values with Same Name

```javascript
let formData = new FormData();

formData.append("hobby", "reading");
formData.append("hobby", "gaming");
formData.append("hobby", "coding");

// Get all values (append preserves duplicates)
console.log(formData.getAll("hobby")); // ['reading', 'gaming', 'coding']

// set removes all existing
formData.set("hobby", "music");
console.log(formData.getAll("hobby")); // ['music']
```

## Complete Example: Multi-File Upload

```html
<input type="file" id="fileInput" multiple />
<button onclick="uploadFiles()">Upload</button>

<script>
  async function uploadFiles() {
    let files = document.getElementById("fileInput").files;
    let formData = new FormData();

    // Append all files
    for (let file of files) {
      formData.append("files", file, file.name);
    }

    // Add metadata
    formData.append("uploader", "John");
    formData.append("timestamp", Date.now());

    let response = await fetch("/upload", {
      method: "POST",
      body: formData,
    });

    console.log(await response.json());
  }
</script>
```

## Summary

| Feature            | Description                                            |
| ------------------ | ------------------------------------------------------ |
| **Purpose**        | Capture and send HTML form data                        |
| **Encoding**       | `multipart/form-data` automatically                    |
| **Files**          | Supported natively, just append with 3-argument syntax |
| **Methods**        | `append()`, `set()`, `get()`, `has()`, `delete()`      |
| **Iteration**      | Supports `for...of` loop                               |
| **Use with fetch** | Pass as `body` parameter                               |

**Key Points:**

- `append()` adds, `set()` replaces existing fields
- File uploads require 3 arguments: `append(name, blob, filename)`
- No need to set `Content-Type` header; browser sets it automatically with boundary
- Works with `fetch`, `XMLHttpRequest`, and other network APIs
