# Object Methods and `this`

## Overview

Methods are functions stored as object properties. They allow objects to "act" using `this` to access the object itself.

## Method Examples

```javascript
let user = {
  name: "John",
  age: 30,
};

// Add method using function expression
user.sayHi = function () {
  alert("Hello!");
};

user.sayHi(); // Hello!

// Or use pre-declared function
function sayHi() {
  alert("Hello!");
}
user.sayHi = sayHi;
user.sayHi(); // Hello!
```

## Method Shorthand

```javascript
let user = {
  sayHi: function () {
    // Regular syntax
    alert("Hello");
  },
};

// Shorthand (preferred)
let user = {
  sayHi() {
    // Same as above
    alert("Hello");
  },
};
```

## The `this` Keyword

`this` refers to the object **before the dot** at call time.

```javascript
let user = {
  name: "John",
  sayHi() {
    alert(this.name); // Accesses current object
  },
};

user.sayHi(); // John (this = user)
```

### Why Not Use the Object Variable?

```javascript
let user = {
  name: "John",
  sayHi() {
    alert(user.name); // Uses variable, not this
  },
};

let admin = user;
user = null;
admin.sayHi(); // Error! user is null
// With this.name, it would work correctly
```

## `this` is Runtime-Evaluated

The same function can have different `this` depending on call context:

```javascript
let user = { name: "John" };
let admin = { name: "Admin" };

function sayHi() {
  alert(this.name);
}

user.f = sayHi;
admin.f = sayHi;

user.f(); // John (this = user)
admin.f(); // Admin (this = admin)
```

## Calling Without an Object

```javascript
function sayHi() {
  alert(this);
}

sayHi(); // undefined (in strict mode)
// In non-strict: global object (window)
```

## Arrow Functions Have No `this`

Arrow functions take `this` from the outer context:

```javascript
let user = {
  firstName: "Ilya",
  sayHi() {
    let arrow = () => alert(this.firstName);
    arrow(); // Takes this from sayHi (user)
  },
};

user.sayHi(); // Ilya
```

## Summary

| Concept             | Behavior                           |
| ------------------- | ---------------------------------- |
| **Method**          | Function stored as object property |
| **`this`**          | Object before the dot at call time |
| **Binding**         | Runtime-determined, not fixed      |
| **Arrow functions** | No own `this`; uses outer context  |

## Tasks

### 1. Using `this` in Object Literal

```javascript
function makeUser() {
  return {
    name: "John",
    ref: this,
  };
}

let user = makeUser();
alert(user.ref.name); // Error! this = global/window, not user
```

**Explanation:** `this` in `makeUser()` is the global object, not the returned object.

### 2. Create a Calculator

```javascript
let calculator = {
  read() {
    this.a = +prompt("Enter first number:", 0);
    this.b = +prompt("Enter second number:", 0);
  },
  sum() {
    return this.a + this.b;
  },
  mul() {
    return this.a * this.b;
  },
};

calculator.read();
alert(calculator.sum());
alert(calculator.mul());
```

### 3. Chaining

```javascript
let ladder = {
  step: 0,
  up() {
    this.step++;
    return this; // Return object for chaining
  },
  down() {
    this.step--;
    return this;
  },
  showStep() {
    alert(this.step);
    return this;
  },
};

// Chainable calls
ladder.up().up().down().showStep().down().showStep();
```
