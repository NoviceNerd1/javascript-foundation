# Strings in JavaScript

## Overview

Strings store textual data in UTF-16 format. No separate type for single characters.

## Quotes

```javascript
let single = "single-quoted";
let double = "double-quoted";
let backticks = `backticks`;
```

### Backticks Features

**Embed expressions with `${}`:**

```javascript
function sum(a, b) {
  return a + b;
}
alert(`1 + 2 = ${sum(1, 2)}.`); // 1 + 2 = 3.
```

**Multiline strings:**

```javascript
let guestList = `Guests:
 * John
 * Pete
 * Mary`;
```

## Special Characters

| Character            | Description     |
| -------------------- | --------------- |
| `\n`                 | New line        |
| `\r`                 | Carriage return |
| `\'`, `\"`, `` \` `` | Quotes          |
| `\\`                 | Backslash       |
| `\t`                 | Tab             |

```javascript
alert("I'm the Walrus!"); // I'm the Walrus! (escaping)
alert("I'm the Walrus!"); // Better: use different quotes
alert(`The backslash: \\`); // The backslash: \
alert(`My\n`.length); // 3 (\n is one character)
```

## String Length

```javascript
let str = "Hello";
alert(str.length); // 5 (property, not method)
```

## Accessing Characters

```javascript
let str = "Hello";
alert(str[0]); // H
alert(str.at(0)); // H
alert(str.at(-1)); // o (negative counts from end)
alert(str[-1]); // undefined (brackets don't support negatives)

// Iteration
for (let char of "Hello") {
  alert(char); // H,e,l,l,o
}
```

## Strings Are Immutable

```javascript
let str = "Hi";
str[0] = "h"; // Error (cannot change)
str = "h" + str[1]; // Workaround: create new string
alert(str); // hi
```

## Changing Case

```javascript
alert("Interface".toUpperCase()); // INTERFACE
alert("Interface".toLowerCase()); // interface
alert("Interface"[0].toLowerCase()); // i
```

## Searching for Substring

### `indexOf` and `lastIndexOf`

```javascript
let str = "Widget with id";

alert(str.indexOf("Widget")); // 0
alert(str.indexOf("id")); // 1
alert(str.indexOf("id", 2)); // 12 (search from position 2)
alert(str.indexOf("widget")); // -1 (case-sensitive)

// Find all occurrences
let target = "as";
let pos = -1;
while ((pos = str.indexOf(target, pos + 1)) != -1) {
  alert(pos);
}

// Check existence correctly
if (str.indexOf("Widget") != -1) {
  // Not just if(...)
  alert("Found");
}
```

### `includes`, `startsWith`, `endsWith`

```javascript
alert("Widget with id".includes("Widget")); // true
alert("Widget".includes("id", 3)); // false (from position 3)
alert("Widget".startsWith("Wid")); // true
alert("Widget".endsWith("get")); // true
```

## Getting Substring

| Method                  | Description                           | Negatives        |
| ----------------------- | ------------------------------------- | ---------------- |
| `slice(start, end)`     | From start to end (not including end) | Yes              |
| `substring(start, end)` | Between start and end                 | Treated as 0     |
| `substr(start, length)` | From start, get length characters     | Yes (start only) |

```javascript
let str = "stringify";

alert(str.slice(0, 5)); // "strin"
alert(str.slice(2)); // "ringify"
alert(str.slice(-4, -1)); // "gif"

alert(str.substring(2, 6)); // "ring"
alert(str.substring(6, 2)); // "ring" (swaps if start > end)
alert(str.substring(-4)); // "stringify" (negative = 0)

alert(str.substr(2, 4)); // "ring"
alert(str.substr(-4, 2)); // "gi"
```

**Recommendation:** Use `slice()` for most cases.

## Comparing Strings

### Character Code Comparison

```javascript
alert("a" > "Z"); // true (97 > 90)
alert("Österreich" > "Zealand"); // true (special characters)
```

### Getting Character Codes

```javascript
alert("Z".codePointAt(0)); // 90
alert("z".codePointAt(0)); // 122
alert(String.fromCodePoint(90)); // Z
alert(String.fromCodePoint(0x5a)); // Z
```

### Language-Aware Comparison

```javascript
alert("Österreich".localeCompare("Zealand")); // -1 (negative means less)
// Returns: negative if less, positive if greater, 0 if equal
```

## Other Useful Methods

```javascript
str.trim(); // Removes spaces from start and end
str.repeat(n); // Repeats string n times
```

## Tasks

### 1. Uppercase First Character

```javascript
function ucFirst(str) {
  if (!str) return str;
  return str[0].toUpperCase() + str.slice(1);
}
```

### 2. Check for Spam

```javascript
function checkSpam(str) {
  let lower = str.toLowerCase();
  return lower.includes("viagra") || lower.includes("xxx");
}
```

### 3. Truncate Text

```javascript
function truncate(str, maxlength) {
  if (str.length <= maxlength) return str;
  return str.slice(0, maxlength - 1) + "…";
}
```

### 4. Extract Currency Value

```javascript
function extractCurrencyValue(str) {
  return +str.slice(1); // Remove '$' and convert to number
}
```

## Summary

| Operation        | Method                                                        |
| ---------------- | ------------------------------------------------------------- |
| Quotes           | `'`, `"`, `` ` `` (backticks allow expressions and multiline) |
| Character access | `[pos]`, `.at(pos)`                                           |
| Length           | `.length` (property)                                          |
| Case conversion  | `.toUpperCase()`, `.toLowerCase()`                            |
| Search           | `.indexOf()`, `.includes()`, `.startsWith()`, `.endsWith()`   |
| Substring        | `.slice()` (preferred), `.substring()`, `.substr()`           |
| Compare          | `.localeCompare()` (language-aware)                           |
