# Decorators and Forwarding, call/apply

## Overview

A **decorator** is a wrapper function that adds behavior to another function without changing its code. Decorators are reusable and can be combined.

## Caching Decorator (Transparent Caching)

```javascript
function slow(x) {
  // CPU-heavy work
  alert(`Called with ${x}`);
  return x;
}

function cachingDecorator(func) {
  let cache = new Map();
  return function (x) {
    if (cache.has(x)) return cache.get(x);
    let result = func(x);
    cache.set(x, result);
    return result;
  };
}

slow = cachingDecorator(slow);
alert(slow(1)); // Computed
alert(slow(1)); // From cache
```

## func.call – Setting `this`

```javascript
function sayHi() {
  alert(this.name);
}

let user = { name: "John" };
let admin = { name: "Admin" };

sayHi.call(user); // John
sayHi.call(admin); // Admin

// With arguments
function say(phrase) {
  alert(this.name + ": " + phrase);
}
say.call(user, "Hello"); // John: Hello
```

### Fixing Context in Decorator

```javascript
let worker = {
  someMethod() {
    return 1;
  },
  slow(x) {
    return x * this.someMethod();
  },
};

function cachingDecorator(func) {
  let cache = new Map();
  return function (x) {
    if (cache.has(x)) return cache.get(x);
    let result = func.call(this, x); // Pass context!
    cache.set(x, result);
    return result;
  };
}

worker.slow = cachingDecorator(worker.slow);
worker.slow(2); // Works!
```

## Multi-Argument Decorator

```javascript
function cachingDecorator(func, hash) {
  let cache = new Map();
  return function () {
    let key = hash(arguments);
    if (cache.has(key)) return cache.get(key);
    let result = func.call(this, ...arguments);
    cache.set(key, result);
    return result;
  };
}

function hash(args) {
  return args[0] + "," + args[1];
}

worker.slow = cachingDecorator(worker.slow, hash);
worker.slow(3, 5); // Computed
worker.slow(3, 5); // Cached
```

## func.apply

`func.apply(context, args)` is similar to `func.call(context, ...args)` but takes array-like arguments.

```javascript
func.call(context, ...args);
func.apply(context, args); // args is array-like
```

### Call Forwarding

```javascript
let wrapper = function () {
  return original.apply(this, arguments);
};
```

## Method Borrowing

Borrow a method from another object to use with array-like objects:

```javascript
function hash() {
  alert([].join.call(arguments)); // Borrow array's join method
}
hash(1, 2); // "1,2"
```

## Summary

| Method       | Purpose                                            | Syntax                                |
| ------------ | -------------------------------------------------- | ------------------------------------- |
| `func.call`  | Call with explicit `this` and arguments list       | `func.call(context, arg1, arg2, ...)` |
| `func.apply` | Call with explicit `this` and array-like arguments | `func.apply(context, args)`           |
| Forwarding   | Pass all arguments and context                     | `original.apply(this, arguments)`     |

**Key Points:**

- Decorators wrap functions to add behavior (caching, logging, delays)
- Use `call`/`apply` to preserve `this` context
- `call` expects argument list; `apply` expects array-like
- Method borrowing allows using array methods on array-like objects

## Tasks

### 1. Spy Decorator

```javascript
function spy(func) {
  function wrapper(...args) {
    wrapper.calls.push(args);
    return func.apply(this, args);
  }
  wrapper.calls = [];
  return wrapper;
}

function work(a, b) {
  return a + b;
}
work = spy(work);
work(1, 2);
work(4, 5);
// work.calls = [[1,2], [4,5]]
```

### 2. Delaying Decorator

```javascript
function delay(f, ms) {
  return function (...args) {
    setTimeout(() => f.apply(this, args), ms);
  };
}

function f(x) {
  alert(x);
}
let f1000 = delay(f, 1000);
f1000("test"); // Shows after 1000ms
```

### 3. Debounce Decorator

```javascript
function debounce(f, ms) {
  let timeout;
  return function (...args) {
    clearTimeout(timeout);
    timeout = setTimeout(() => f.apply(this, args), ms);
  };
}
// Waits for pause in calls, then runs once with last arguments
```

### 4. Throttle Decorator

```javascript
function throttle(f, ms) {
  let isThrottled = false;
  let savedArgs, savedThis;

  function wrapper(...args) {
    if (isThrottled) {
      savedArgs = args;
      savedThis = this;
      return;
    }
    f.apply(this, args);
    isThrottled = true;
    setTimeout(() => {
      isThrottled = false;
      if (savedArgs) {
        wrapper.apply(savedThis, savedArgs);
        savedArgs = savedThis = null;
      }
    }, ms);
  }
  return wrapper;
}
// Runs at most once per ms milliseconds
```
