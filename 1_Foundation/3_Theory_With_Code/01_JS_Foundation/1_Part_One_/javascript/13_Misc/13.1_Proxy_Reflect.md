# Proxy and Reflect

## Overview

A **Proxy** wraps another object and intercepts operations (reading/writing properties, function calls, etc.). It allows customizing object behavior.

## Basic Syntax

```javascript
let proxy = new Proxy(target, handler);
```

- `target` – the object to wrap
- `handler` – object with traps (methods intercepting operations)

## Simple Proxy (No Traps)

```javascript
let target = {};
let proxy = new Proxy(target, {}); // Empty handler

proxy.test = 5;
alert(target.test); // 5 (writing forwarded)
alert(proxy.test); // 5 (reading forwarded)
```

## Common Traps

| Internal Method       | Handler Method   | Trigger                  |
| --------------------- | ---------------- | ------------------------ |
| `[[Get]]`             | `get`            | Reading a property       |
| `[[Set]]`             | `set`            | Writing a property       |
| `[[HasProperty]]`     | `has`            | `in` operator            |
| `[[Delete]]`          | `deleteProperty` | `delete` operator        |
| `[[Call]]`            | `apply`          | Function call            |
| `[[Construct]]`       | `construct`      | `new` operator           |
| `[[OwnPropertyKeys]]` | `ownKeys`        | `Object.keys`, `for..in` |

## `get` Trap – Default Values

```javascript
let numbers = [0, 1, 2];

numbers = new Proxy(numbers, {
  get(target, prop) {
    return prop in target ? target[prop] : 0; // Default for missing
  },
});

alert(numbers[1]); // 1
alert(numbers[123]); // 0 (no error)
```

### Dictionary Example

```javascript
let dictionary = {
  Hello: "Hola",
  Bye: "Adiós",
};

dictionary = new Proxy(dictionary, {
  get(target, phrase) {
    return phrase in target ? target[phrase] : phrase; // Return original if missing
  },
});

alert(dictionary["Hello"]); // Hola
alert(dictionary["Welcome"]); // Welcome (no translation)
```

## `set` Trap – Validation

```javascript
let numbers = [];

numbers = new Proxy(numbers, {
  set(target, prop, val) {
    if (typeof val === "number") {
      target[prop] = val;
      return true; // Must return true on success
    }
    return false; // Triggers TypeError
  },
});

numbers.push(1); // Works
numbers.push(2); // Works
numbers.push("test"); // TypeError
```

## `ownKeys` Trap – Filter Properties

```javascript
let user = {
  name: "John",
  age: 30,
  _password: "***",
};

user = new Proxy(user, {
  ownKeys(target) {
    return Object.keys(target).filter((key) => !key.startsWith("_"));
  },

  getOwnPropertyDescriptor(target, prop) {
    return {
      enumerable: true,
      configurable: true,
    };
  },
});

for (let key in user) alert(key); // name, age
alert(Object.keys(user)); // name, age
```

## Protected Properties with Multiple Traps

```javascript
let user = {
  name: "John",
  _password: "***",
};

user = new Proxy(user, {
  get(target, prop) {
    if (prop.startsWith("_")) throw new Error("Access denied");
    let value = target[prop];
    // Bind methods to original target to preserve this
    return typeof value === "function" ? value.bind(target) : value;
  },

  set(target, prop, val) {
    if (prop.startsWith("_")) throw new Error("Access denied");
    target[prop] = val;
    return true;
  },

  deleteProperty(target, prop) {
    if (prop.startsWith("_")) throw new Error("Access denied");
    delete target[prop];
    return true;
  },

  ownKeys(target) {
    return Object.keys(target).filter((key) => !key.startsWith("_"));
  },
});

alert(user.name); // John
alert(user._password); // Error
user._password = "new"; // Error
```

## `has` Trap – `in` Operator

```javascript
let range = { start: 1, end: 10 };

range = new Proxy(range, {
  has(target, prop) {
    return prop >= target.start && prop <= target.end;
  },
});

alert(5 in range); // true
alert(50 in range); // false
```

## `apply` Trap – Function Wrapping

```javascript
function delay(f, ms) {
  return new Proxy(f, {
    apply(target, thisArg, args) {
      setTimeout(() => target.apply(thisArg, args), ms);
    },
  });
}

function sayHi(user) {
  alert(`Hello, ${user}!`);
}

sayHi = delay(sayHi, 3000);
alert(sayHi.length); // 1 (works! proxy forwards property access)
sayHi("John"); // Hello, John! (after 3 seconds)
```

## Reflect – Forwarding Operations

`Reflect` provides methods matching all proxy traps, allowing easy forwarding:

```javascript
let user = { name: "John" };

user = new Proxy(user, {
  get(target, prop, receiver) {
    alert(`GET ${prop}`);
    return Reflect.get(target, prop, receiver); // Forward correctly
  },
  set(target, prop, val, receiver) {
    alert(`SET ${prop}=${val}`);
    return Reflect.set(target, prop, val, receiver);
  },
});

let name = user.name; // GET name
user.name = "Pete"; // SET name=Pete
```

### Why `Reflect` Matters – Preserving `this` for Getters

```javascript
let user = {
  _name: "Guest",
  get name() {
    return this._name;
  },
};

let userProxy = new Proxy(user, {
  get(target, prop, receiver) {
    return Reflect.get(target, prop, receiver); // Preserves this
  },
});

let admin = {
  __proto__: userProxy,
  _name: "Admin",
};

alert(admin.name); // Admin (correct, not Guest)
```

## Proxy Limitations

### 1. Built-in Objects with Internal Slots (Map, Set, Date, Promise)

```javascript
let map = new Map();
let proxy = new Proxy(map, {});
proxy.set("test", 1); // Error! (internal slot [[MapData]] missing)

// Fix: bind methods to target
let proxy = new Proxy(map, {
  get(target, prop, receiver) {
    let value = Reflect.get(...arguments);
    return typeof value === "function" ? value.bind(target) : value;
  },
});
proxy.set("test", 1); // Works
```

### 2. Private Class Fields (similar issue)

```javascript
class User {
  #name = "Guest";
  getName() {
    return this.#name;
  }
}

let user = new User();
let proxy = new Proxy(user, {});
proxy.getName(); // Error

// Fix: same binding solution
```

### 3. Proxy ≠ Target (different objects)

```javascript
let allUsers = new Set();
let user = { name: "John" };
allUsers.add(user);

let proxy = new Proxy(user, {});
allUsers.has(proxy); // false (different object)
```

### 4. `===` Cannot Be Intercepted

```javascript
let obj = {};
let proxy = new Proxy(obj, {});
alert(obj === proxy); // false (cannot be changed)
```

## Revocable Proxy

```javascript
let { proxy, revoke } = Proxy.revocable(target, {});

// Use proxy...
proxy.data = "value";

// Later: disable the proxy
revoke();

alert(proxy.data); // Error (proxy revoked)
```

## Summary

| Feature       | Description                              |
| ------------- | ---------------------------------------- |
| **Proxy**     | Wraps object, intercepts operations      |
| **Handler**   | Object with traps (get, set, etc.)       |
| **Reflect**   | Provides matching methods for forwarding |
| **Revocable** | Can be disabled with `revoke()`          |

**Key Points:**

- Proxies intercept operations like property access, assignment, deletion, function calls
- Use `Reflect` to safely forward operations while preserving `this`
- Built-in objects with internal slots (Map, Set) need special handling
- Proxy and target are different objects (affects `===` and Set/Map lookups)

## Tasks

### 1. Error on Reading Non-Existent Property

```javascript
function wrap(target) {
  return new Proxy(target, {
    get(target, prop) {
      if (prop in target) {
        return target[prop];
      } else {
        throw new ReferenceError(`Property doesn't exist: "${prop}"`);
      }
    },
  });
}

let user = { name: "John" };
user = wrap(user);
alert(user.name); // John
alert(user.age); // ReferenceError
```

### 2. Accessing array[-1]

```javascript
let array = [1, 2, 3];

array = new Proxy(array, {
  get(target, prop) {
    let index = parseInt(prop);
    if (index < 0) {
      index = target.length + index;
    }
    return target[index];
  },
});

alert(array[-1]); // 3
alert(array[-2]); // 2
```

### 3. Observable Object

```javascript
function makeObservable(target) {
  let handlers = [];

  target.observe = function (handler) {
    handlers.push(handler);
  };

  return new Proxy(target, {
    set(target, prop, value, receiver) {
      let success = Reflect.set(target, prop, value, receiver);
      if (success) {
        handlers.forEach((handler) => handler(prop, value));
      }
      return success;
    },
  });
}

let user = {};
user = makeObservable(user);

user.observe((key, value) => {
  alert(`SET ${key}=${value}`);
});

user.name = "John"; // alerts: SET name=John
```
