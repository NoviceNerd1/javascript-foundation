# CSS Animations Summary

CSS animations allow smooth transitions of CSS properties without JavaScript, using either `transition` for property changes or `@keyframes` for complex animations.

## CSS Transitions

Transitions animate property changes automatically when the property value changes.

### Basic Syntax

```css
.animated {
  transition-property: background-color;
  transition-duration: 3s;
  transition-timing-function: ease;
  transition-delay: 0s;
}

/* Shorthand: property duration timing-function delay */
.animated {
  transition: background-color 3s ease 0s;
}

/* Multiple properties */
.animated {
  transition:
    font-size 3s,
    color 2s;
}

/* All properties */
.animated {
  transition: all 3s;
}
```

### Transition Properties

| Property                     | Description                                              |
| ---------------------------- | -------------------------------------------------------- |
| `transition-property`        | Properties to animate (or `all`)                         |
| `transition-duration`        | Duration (e.g., `3s`, `500ms`)                           |
| `transition-timing-function` | Speed curve                                              |
| `transition-delay`           | Delay before start (negative values start mid-animation) |

```javascript
// JavaScript trigger
element.style.backgroundColor = "red"; // Transition starts automatically
element.classList.add("animate"); // CSS class triggers transition
```

## Transition Timing Functions

### Bezier Curves (`cubic-bezier()`)

Controls animation speed over time (x=time, y=completion).

```css
/* Built-in curves */
transition: all 3s linear; /* cubic-bezier(0,0,1,1) - uniform speed */
transition: all 3s ease; /* cubic-bezier(0.25,0.1,0.25,1) - default */
transition: all 3s ease-in; /* cubic-bezier(0.42,0,1,1) - slow start */
transition: all 3s ease-out; /* cubic-bezier(0,0,0.58,1) - slow end */
transition: all 3s ease-in-out; /* cubic-bezier(0.42,0,0.58,1) */

/* Custom curve */
transition: all 3s cubic-bezier(0.68, -0.55, 0.265, 1.55);
/* y can exceed 0-1 range for bounce effects */
```

### Step Functions (`steps()`)

Splits animation into discrete steps.

```css
/* 9 steps, change at start of each step */
transition: transform 9s steps(9, start);

/* 9 steps, change at end of each step */
transition: transform 9s steps(9, end);

/* Shorthands */
transition: all 1s step-start; /* steps(1, start) */
transition: all 1s step-end; /* steps(1, end) */
```

## transitionend Event

Triggers when animation completes:

```javascript
element.addEventListener("transitionend", (event) => {
  console.log("Property:", event.propertyName);
  console.log("Duration:", event.elapsedTime);
  console.log("Finished!");
});
```

## @keyframes Animation

Define complex multi-step animations:

```css
@keyframes slide {
  from {
    left: 0px;
  }
  to {
    left: calc(100% - 50px);
  }
}

@keyframes bounce {
  0% {
    transform: translateY(0);
  }
  50% {
    transform: translateY(-50px);
  }
  100% {
    transform: translateY(0);
  }
}

@keyframes fade-in-out {
  0% {
    opacity: 0;
    transform: translateY(-60px);
  }
  50% {
    opacity: 1;
    transform: none;
  }
  100% {
    opacity: 0;
    transform: translateX(230px) rotateZ(90deg) scale(0.5);
  }
}

/* Apply animation */
.element {
  animation: slide 3s infinite alternate;
  /* animation: name duration timing-function delay iteration-count direction fill-mode; */
}
```

### Animation Properties

| Property                    | Description                                   |
| --------------------------- | --------------------------------------------- |
| `animation-name`            | @keyframes name                               |
| `animation-duration`        | Total duration                                |
| `animation-timing-function` | Speed curve                                   |
| `animation-delay`           | Start delay                                   |
| `animation-iteration-count` | Number of repeats (`infinite`)                |
| `animation-direction`       | `normal`, `reverse`, `alternate`              |
| `animation-fill-mode`       | Styles before/after (`forwards`, `backwards`) |

## Performance Optimization

### Animation Cost (Layout → Paint → Composite)

| Property Type                    | Steps                      | Performance  |
| -------------------------------- | -------------------------- | ------------ |
| `transform`, `opacity`           | Composite only             | ⚡ Best      |
| `color`, `background`            | Paint + Composite          | ✓ Good       |
| `width`, `height`, `left`, `top` | Layout + Paint + Composite | ❌ Expensive |

### Use transform for Better Performance

```css
/* Bad - triggers Layout */
.move {
  left: 300px;
}

/* Good - Composite only */
.move {
  transform: translateX(300px);
}

/* Scale instead of width/height */
.grow {
  transform: scale(2);
}

/* Combine transforms */
.animate {
  transform: translateX(300px) rotate(90deg) scale(0.5);
  opacity: 0;
}
```

## Complete Examples

### Growing Circle with Callback

```html
<style>
  .circle {
    position: absolute;
    border-radius: 50%;
    background: red;
    opacity: 0;
    transform: scale(0);
    transition:
      transform 0.5s,
      opacity 0.5s;
  }
  .circle.show {
    transform: scale(1);
    opacity: 1;
  }
</style>

<script>
  function showCircle(cx, cy, radius, callback) {
    let div = document.createElement("div");
    div.className = "circle";
    div.style.width = radius * 2 + "px";
    div.style.height = radius * 2 + "px";
    div.style.left = cx - radius + "px";
    div.style.top = cy - radius + "px";

    document.body.append(div);

    div.addEventListener(
      "transitionend",
      () => {
        callback(div);
      },
      { once: true },
    );

    setTimeout(() => div.classList.add("show"), 0);
  }

  showCircle(150, 150, 100, (div) => {
    div.append("Hello, world!");
  });
</script>
```

### Animated Plane

```html
<img src="plane.png" id="plane" />

<style>
  #plane {
    width: 40px;
    transition: all 3s ease-out;
  }
  #plane.grow {
    width: 400px;
  }
</style>

<script>
  plane.onclick = function () {
    this.classList.add("grow");

    this.addEventListener(
      "transitionend",
      () => {
        alert("Done!");
      },
      { once: true },
    );
  };
</script>
```

### Bounce Animation

```html
<div class="ball"></div>

<style>
  .ball {
    width: 50px;
    height: 50px;
    border-radius: 50%;
    background: red;
    animation: bounce 1s ease-in-out infinite;
  }

  @keyframes bounce {
    0%,
    100% {
      transform: translateY(0);
    }
    50% {
      transform: translateY(-100px);
    }
  }
</style>
```

## CSS vs JavaScript Animation

| Aspect          | CSS               | JavaScript          |
| --------------- | ----------------- | ------------------- |
| **Complexity**  | Simple animations | Any logic possible  |
| **Performance** | GPU accelerated   | Manual optimization |
| **Control**     | Limited           | Full control        |
| **Events**      | `transitionend`   | Callbacks, promises |
| **Dynamic**     | Class toggles     | Programmatic        |

## Summary

1. **Transitions**: Animate property changes with `transition`
2. **Timing functions**: `cubic-bezier()` for smooth, `steps()` for discrete
3. **Keyframes**: Complex multi-step animations with `@keyframes`
4. **Events**: `transitionend` and `animationend` events
5. **Performance**: Prefer `transform` and `opacity` over layout-affecting properties
6. **JavaScript**: Can trigger and respond to CSS animations with minimal code
