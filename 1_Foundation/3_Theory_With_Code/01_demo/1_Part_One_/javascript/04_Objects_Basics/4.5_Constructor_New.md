# Constructor, Operator `new`

## Overview

Constructor functions create multiple similar objects using the `new` operator.

## Constructor Function Conventions

1. Named with **capital letter**
2. Executed only with **`new`**

```javascript
function User(name) {
  this.name = name;
  this.isAdmin = false;
}

let user = new User("Jack");
alert(user.name); // Jack
alert(user.isAdmin); // false
```

## What `new` Does

```javascript
function User(name) {
  // this = {};  (created implicitly)

  this.name = name;
  this.isAdmin = false;

  // return this;  (implicitly)
}
```

`new User("Jack")` creates same as:

```javascript
let user = {
  name: "Jack",
  isAdmin: false,
};
```

## Adding Methods in Constructor

```javascript
function User(name) {
  this.name = name;

  this.sayHi = function () {
    alert("My name is: " + this.name);
  };
}

let john = new User("John");
john.sayHi(); // My name is: John
```

## Testing Constructor Mode: `new.target`

```javascript
function User() {
  alert(new.target);
}

User(); // undefined (regular call)
new User(); // function User { ... } (constructor call)

// Can make both new and regular calls work
function User(name) {
  if (!new.target) {
    return new User(name); // Add new automatically
  }
  this.name = name;
}

let john = User("John"); // Works without new
alert(john.name); // John
```

## Return from Constructors

- If `return` is called with an **object**, that object is returned instead of `this`
- If `return` is called with a **primitive**, it's ignored

```javascript
function BigUser() {
  this.name = "John";
  return { name: "Godzilla" }; // Overrides this
}
alert(new BigUser().name); // Godzilla

function SmallUser() {
  this.name = "John";
  return; // Returns this
}
alert(new SmallUser().name); // John
```

## Omitting Parentheses

```javascript
let user = new User(); // Allowed (no parentheses)
let user = new User(); // Preferred style
```

## One-Time Constructor

```javascript
let user = new (function () {
  this.name = "John";
  this.isAdmin = false;
  // Complex creation logic
})();
// Cannot be called again (not saved)
```

## Summary

| Concept            | Description                                    |
| ------------------ | ---------------------------------------------- |
| **Constructor**    | Regular function with capital letter           |
| **`new` operator** | Creates empty `this`, returns populated object |
| **`new.target`**   | Checks if called with `new`                    |
| **Return**         | Object overrides `this`; primitive ignored     |

## Tasks

### 1. Two Functions – One Object

```javascript
let obj = {};

function A() {
  return obj;
}
function B() {
  return obj;
}

let a = new A();
let b = new B();

alert(a == b); // true
```

### 2. Create new Calculator

```javascript
function Calculator() {
  this.read = function () {
    this.a = +prompt("Enter first number:", 0);
    this.b = +prompt("Enter second number:", 0);
  };
  this.sum = function () {
    return this.a + this.b;
  };
  this.mul = function () {
    return this.a * this.b;
  };
}

let calculator = new Calculator();
calculator.read();
alert("Sum=" + calculator.sum());
alert("Mul=" + calculator.mul());
```

### 3. Create new Accumulator

```javascript
function Accumulator(startingValue) {
  this.value = startingValue;
  this.read = function () {
    this.value += +prompt("Add number:", 0);
  };
}

let accumulator = new Accumulator(1);
accumulator.read();
accumulator.read();
alert(accumulator.value);
```
