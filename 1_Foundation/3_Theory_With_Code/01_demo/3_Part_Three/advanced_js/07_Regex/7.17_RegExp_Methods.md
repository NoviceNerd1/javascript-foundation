# Regexp and String Methods Summary

Comprehensive reference for all JavaScript methods that work with regular expressions.

## String Methods

### str.match(regexp)

Returns matches of `regexp` in the string.

```javascript
// Without 'g' flag: returns first match with groups
let str = "I love JavaScript";
let result = str.match(/Java(Script)/);

console.log(result[0]); // "JavaScript" (full match)
console.log(result[1]); // "Script" (first group)
console.log(result.index); // 7 (match position)
console.log(result.input); // "I love JavaScript"

// With 'g' flag: returns array of all matches
let result2 = str.match(/Java(Script)/g);
console.log(result2); // ["JavaScript"]

// No matches: returns null (important!)
let noMatch = str.match(/HTML/);
console.log(noMatch); // null
// console.log(noMatch.length); // Error! Use noMatch || [] to avoid
```

### str.matchAll(regexp)

Returns iterable of all matches with groups (requires `g` flag).

```javascript
let str = "<h1>Hello</h1> <h2>World</h2>";
let regexp = /<(.*?)>/g;

let matches = Array.from(str.matchAll(regexp));
console.log(matches[0]); // ["<h1>", "h1"]
console.log(matches[1]); // ["</h1>", "/h1"]
console.log(matches[2]); // ["<h2>", "h2"]

// Iterate directly
for (let match of str.matchAll(regexp)) {
  console.log(`Tag: ${match[1]}, at position: ${match.index}`);
}
// Tag: h1, at position: 0
// Tag: /h1, at position: 6
// Tag: h2, at position: 12
```

### str.split(regexp, limit)

Splits string by regexp delimiter.

```javascript
// Split by commas with optional spaces
console.log("12, 34, 56".split(/,\s*/)); // ["12", "34", "56"]

// Split by any non-word character
console.log("Hello, world! How?".split(/\W+/));
// ["Hello", "world", "How", ""]
```

### str.search(regexp)

Returns position of first match, or -1 if none.

```javascript
let str = "A drop of ink may make a million think";
console.log(str.search(/ink/i)); // 10
console.log(str.search(/HTML/)); // -1
```

### str.replace(regexp, replacement)

Replaces matches with string or function.

```javascript
// Basic replacement
console.log("12-34-56".replace(/-/g, ":")); // "12:34:56"

// Using $n for groups
console.log("John Smith".replace(/(\w+) (\w+)/, "$2, $1"));
// "Smith, John"

// Special replacement symbols
let str = "Hello, world!";
console.log(str.replace(/world/, "$` before")); // "Hello, Hello,  before!"
```

**Replacement symbols:**

| Symbol    | Action                |
| --------- | --------------------- |
| `$&`      | Whole match           |
| ``$` ``   | Text before match     |
| `$'`      | Text after match      |
| `$n`      | nth capturing group   |
| `$<name>` | Named capturing group |
| `$$`      | Literal `$`           |

### str.replaceAll(regexp, replacement)

Replaces all occurrences (unlike `replace` which only replaces first when using string).

```javascript
// String argument: replaces ALL occurrences
console.log("12-34-56".replaceAll("-", ":")); // "12:34:56"

// With regexp: requires 'g' flag
console.log("12-34-56".replaceAll(/-/g, ":")); // "12:34:56"

// Without 'g' throws error
console.log("12-34-56".replaceAll(/-/, ":")); // Error!
```

### Using Function as Replacement

```javascript
// Uppercase all matches
let str = "html and css";
let result = str.replace(/html|css/gi, (match) => match.toUpperCase());
console.log(result); // "HTML and CSS"

// Replace with position
console.log("Ho-Ho-ho".replace(/ho/gi, (match, offset) => offset));
// "0-3-6"

// With capturing groups
let result2 = "John Smith".replace(/(\w+) (\w+)/, (match, name, surname) => {
  return `${surname}, ${name}`;
});
console.log(result2); // "Smith, John"

// With named groups
let result3 = "John Smith".replace(
  /(?<name>\w+) (?<surname>\w+)/,
  (...match) => {
    let groups = match.pop();
    return `${groups.surname}, ${groups.name}`;
  },
);
console.log(result3); // "Smith, John"
```

## RegExp Methods

### regexp.exec(str)

Returns match with groups, updates `lastIndex` when `g` flag present.

```javascript
let str = "More about JavaScript at javascript.info";
let regexp = /javascript/gi;

// Repeated calls return matches one by one
let result = regexp.exec(str);
console.log(result[0]); // "JavaScript"
console.log(result.index); // 11
console.log(regexp.lastIndex); // 21

result = regexp.exec(str);
console.log(result[0]); // "javascript"
console.log(result.index); // 33
console.log(regexp.lastIndex); // 43

result = regexp.exec(str);
console.log(result); // null
console.log(regexp.lastIndex); // 0 (reset)
```

### Using exec in Loop

```javascript
let str = "More about JavaScript at javascript.info";
let regexp = /javascript/gi;
let result;

while ((result = regexp.exec(str))) {
  console.log(`Found "${result[0]}" at position ${result.index}`);
}
// Found "JavaScript" at position 11
// Found "javascript" at position 33
```

### Starting Search at Specific Position

```javascript
let str = "Hello, world!";
let regexp = /\w+/g;

regexp.lastIndex = 5; // Start from position 5 (comma)
console.log(regexp.exec(str)); // ["world"] (finds word after comma)
```

### sticky flag `y` with exec

```javascript
let str = "Hello, world!";
let regexp = /\w+/y;

regexp.lastIndex = 5;
console.log(regexp.exec(str)); // null (no word exactly at position 5)

regexp.lastIndex = 0;
console.log(regexp.exec(str)); // ["Hello"] (word at position 0)
```

### regexp.test(str)

Returns `true` if match exists, updates `lastIndex` with `g` flag.

```javascript
let regexp = /love/gi;
let str = "I love JavaScript";

console.log(regexp.test(str)); // true
console.log(regexp.lastIndex); // 6

console.log(regexp.test(str)); // false (search from position 6)
console.log(regexp.lastIndex); // 0 (reset)
```

### Warning: Global Regexp Reuse

```javascript
let regexp = /javascript/g;

console.log(regexp.test("javascript")); // true (lastIndex = 10)
console.log(regexp.test("javascript")); // false (lastIndex != 0)

// Fix: reset lastIndex
regexp.lastIndex = 0;
console.log(regexp.test("javascript")); // true

// Better: use string methods
console.log("javascript".match(/javascript/g)); // ["javascript"]
```

## Method Comparison

| Method           | Called on | Returns        | Groups            | Global            |
| ---------------- | --------- | -------------- | ----------------- | ----------------- |
| `str.match`      | string    | Array or null  | Yes (without `g`) | Yes               |
| `str.matchAll`   | string    | Iterable       | Always            | Required `g`      |
| `str.split`      | string    | Array          | -                 | -                 |
| `str.search`     | string    | Number (index) | -                 | -                 |
| `str.replace`    | string    | String         | Yes               | Yes               |
| `str.replaceAll` | string    | String         | Yes               | Required `g`      |
| `regexp.exec`    | regexp    | Array or null  | Always            | Updates lastIndex |
| `regexp.test`    | regexp    | Boolean        | -                 | Updates lastIndex |

## Summary Table

| Method                         | When to Use                                                        |
| ------------------------------ | ------------------------------------------------------------------ |
| `str.match(regexp)`            | Get first match (without `g`) or all matches as strings (with `g`) |
| `str.matchAll(regexp)`         | Get all matches with groups and positions                          |
| `str.split(regexp)`            | Split string by regexp delimiter                                   |
| `str.search(regexp)`           | Find position of first match                                       |
| `str.replace(regexp, repl)`    | Replace matches (function or string)                               |
| `str.replaceAll(regexp, repl)` | Replace all matches (requires `g`)                                 |
| `regexp.exec(str)`             | Get match with groups, iterate with `g`                            |
| `regexp.test(str)`             | Check if match exists                                              |

**Key Points:**

- `match` returns `null` for no matches (use `|| []` to avoid errors)
- `matchAll` requires `g` flag and returns iterable
- `replace` with string replaces only first occurrence; use regexp with `g` or `replaceAll`
- `exec` and `test` update `lastIndex` when `g` flag is present
- Reset `lastIndex` when reusing global regexp on different strings
