# Unicode: Flag "u" and Class \p{...} Summary

JavaScript uses Unicode encoding. Some characters (like emojis, mathematical symbols) require 4 bytes (surrogate pairs). Flag `u` enables proper handling of these characters and allows searching by Unicode properties.

## The Problem: 4-Byte Characters

```javascript
// length treats 4-byte characters as 2 characters
console.log("😄".length); // 2 (incorrect)
console.log("𝒳".length); // 2 (incorrect)

// Without 'u' flag, regexp treats them as separate
console.log("😄".match(/./g)); // ['�', '�'] (broken)
console.log("😄".match(/./gu)); // ['😄'] (correct)
```

## Flag `u` (Unicode)

Enables correct handling of 4-byte characters and Unicode property search.

```javascript
// Without 'u' - fails on 4-byte characters
let str = "A😄B";
console.log(str.match(/./g)); // ['A', '�', '�', 'B'] (broken)
console.log(str.match(/./gu)); // ['A', '😄', 'B'] (correct)
```

## Unicode Properties: `\p{...}`

Search characters by their Unicode properties. **Requires flag `u`**.

```javascript
// Find all letters (any language)
let str = "A ბ ㄱ";
console.log(str.match(/\p{L}/gu)); // ['A', 'ბ', 'ㄱ']
console.log(str.match(/\p{L}/g)); // null (needs 'u' flag)
```

### Main Property Categories

| Category            | Description               | Aliases  |
| ------------------- | ------------------------- | -------- |
| **Letter (L)**      | Letters from any alphabet | `\p{L}`  |
| └ Ll                | Lowercase                 | `\p{Ll}` |
| └ Lu                | Uppercase                 | `\p{Lu}` |
| └ Lo                | Other letters             | `\p{Lo}` |
| **Number (N)**      | Numbers and digits        | `\p{N}`  |
| └ Nd                | Decimal digits            | `\p{Nd}` |
| └ Nl                | Letter numbers (e.g., Ⅻ)  | `\p{Nl}` |
| **Punctuation (P)** | Punctuation marks         | `\p{P}`  |
| **Mark (M)**        | Accents, diacritics       | `\p{M}`  |
| **Symbol (S)**      | Math, currency, etc.      | `\p{S}`  |
| └ Sc                | Currency symbols          | `\p{Sc}` |
| **Separator (Z)**   | Spaces, lines             | `\p{Z}`  |
| **Other (C)**       | Control characters, etc.  | `\p{C}`  |

### Property Examples

```javascript
// Find all digits (any script)
let text = "123 一二三 ٤٥٦";
console.log(text.match(/\p{Nd}/gu)); // ['1','2','3','٤','٥','٦']

// Find all punctuation
let quote = "Hello! How are you?";
console.log(quote.match(/\p{P}/gu)); // ['!', '?']

// Find currency symbols
let prices = "Price: $20, €15, ¥1000";
console.log(prices.match(/\p{Sc}/gu)); // ['$', '€', '¥']
```

## Script Property: `\p{sc=...}`

Find characters from specific writing systems.

```javascript
// Chinese characters (Han script)
let str = "Hello 你好 Привет";
console.log(str.match(/\p{sc=Han}/gu)); // ['你', '好']

// Cyrillic script
console.log(str.match(/\p{sc=Cyrillic}/gu)); // ['П', 'р', 'и', 'в', 'е', 'т']

// Greek script
let greek = "Alpha Αλφα Beta Βήτα";
console.log(greek.match(/\p{sc=Greek}/gu)); // ['Α', 'λ', 'φ', 'α', 'Β', 'ή', 'τ', 'α']
```

Common script values: `Han` (Chinese), `Cyrillic`, `Greek`, `Arabic`, `Hebrew`, `Hiragana`, `Katakana`, `Latin`.

## Practical Examples

### Find Hexadecimal Numbers

```javascript
let hex = "xAF xGZ x12";
let regexp = /x\p{Hex_Digit}\p{Hex_Digit}/gu;
console.log(hex.match(regexp)); // ['xAF', 'x12']
```

### Find Emojis

```javascript
let text = "Hello 😀 World 🌍!";
let emojis = text.match(/\p{Emoji}/gu);
console.log(emojis); // ['😀', '🌍']
```

### Validate Unicode Email (Simplified)

```javascript
function isValidUnicodeEmail(email) {
  // Letters, numbers, dots, @, etc. in any language
  let regexp = /^[\p{L}\p{N}\._%+-]+@[\p{L}\p{N}.-]+\.[\p{L}]{2,}$/u;
  return regexp.test(email);
}

console.log(isValidUnicodeEmail("用户@例子.中国")); // true
console.log(isValidUnicodeEmail("user@example.com")); // true
```

### Extract All Letters (Any Language)

```javascript
let mixed = "English Русский 中文 العربية";
let letters = mixed.match(/\p{L}/gu);
console.log(letters);
// ['E','n','g','l','i','s','h','Р','у','с','с','к','и','й','中','文','ا','ل','ع','ر','ب','ي','ة']
```

### Find All Numbers (Any Script)

```javascript
let numbers = "123 ١٢٣ 一二三";
let digits = numbers.match(/\p{N}/gu);
console.log(digits); // ['1','2','3','١','٢','٣','一','二','三']
```

## Property Reference

| Shortcut    | Full Property          | Matches            |
| ----------- | ---------------------- | ------------------ |
| `\p{L}`     | `\p{Letter}`           | Any letter         |
| `\p{N}`     | `\p{Number}`           | Any number         |
| `\p{P}`     | `\p{Punctuation}`      | Punctuation        |
| `\p{S}`     | `\p{Symbol}`           | Symbols            |
| `\p{Z}`     | `\p{Separator}`        | Spaces, separators |
| `\p{Sc}`    | `\p{Currency_Symbol}`  | Currency symbols   |
| `\p{Nd}`    | `\p{Decimal_Number}`   | Decimal digits     |
| `\p{Lu}`    | `\p{Uppercase_Letter}` | Uppercase letters  |
| `\p{Ll}`    | `\p{Lowercase_Letter}` | Lowercase letters  |
| `\p{Emoji}` | -                      | Emoji characters   |

## Summary

| Feature                 | Description                                         |
| ----------------------- | --------------------------------------------------- |
| **Flag `u`**            | Enables full Unicode support in regexp              |
| **4-byte characters**   | Handled correctly as single characters              |
| **`\p{...}`**           | Search by Unicode properties                        |
| **Property categories** | `L` (letter), `N` (number), `P` (punctuation), etc. |
| **Script property**     | `\p{sc=Han}`, `\p{sc=Cyrillic}`, etc.               |
| **Requirement**         | Must use `u` flag with `\p{...}`                    |

**Key Points:**

- Without `u` flag, 4-byte characters are treated as two separate characters
- Use `u` flag for correct Unicode handling
- `\p{...}` enables powerful Unicode-aware searches
- Properties like `L` (letter), `N` (number), `Sc` (currency) work across all languages
- `\p{sc=...}` filters by writing system (Han, Cyrillic, Greek, etc.)
