# Bezier Curves Summary

Bezier curves are mathematical curves defined by control points, used in computer graphics, vector editing, CSS animation, and font design.

## Control Points

A Bezier curve is defined by 2 or more control points:

| Points   | Curve Type                   |
| -------- | ---------------------------- |
| 2 points | Linear curve (straight line) |
| 3 points | Quadratic curve (parabolic)  |
| 4 points | Cubic curve                  |

**Key properties:**

- Only first and last points lie on the curve
- Curve stays inside the convex hull of control points
- Moving points changes curve in intuitive ways

## De Casteljau's Algorithm

The visual method for constructing Bezier curves using linear interpolation.

### For 3 Points (Quadratic)

```
Control points: 1 → 2 → 3

For t from 0 to 1:
1. Take points on segments 1→2 and 2→3 at distance t
2. Connect these points (blue segment)
3. Take point on blue segment at distance t (red point)
4. All red points form the curve
```

### For 4 Points (Cubic)

```
Control points: 1 → 2 → 3 → 4

For t from 0 to 1:
1. Take points on 1→2, 2→3, 3→4 at distance t (3 brown points)
2. Connect to get 2 green segments
3. Take points on green segments at distance t (2 points)
4. Connect to get 1 blue segment
5. Take point on blue segment at distance t (red point)
6. All red points form the curve
```

The algorithm is recursive and works for any number of control points.

## Mathematical Formulas

### 2 Points (Linear)

```
P = (1-t)P₁ + tP₂
```

### 3 Points (Quadratic)

```
P = (1-t)²P₁ + 2(1-t)tP₂ + t²P₃
```

### 4 Points (Cubic)

```
P = (1-t)³P₁ + 3(1-t)²tP₂ + 3(1-t)t²P₃ + t³P₄
```

### Coordinate Formulas (3-point example)

```
x = (1-t)²x₁ + 2(1-t)tx₂ + t²x₃
y = (1-t)²y₁ + 2(1-t)ty₂ + t²y₃
```

## Canvas Example

```javascript
let canvas = document.getElementById("canvas");
let ctx = canvas.getContext("2d");

// Quadratic Bezier
ctx.beginPath();
ctx.moveTo(0, 100); // Start point
ctx.quadraticCurveTo(100, 0, 200, 100); // Control, End
ctx.stroke();

// Cubic Bezier
ctx.beginPath();
ctx.moveTo(0, 150); // Start point
ctx.bezierCurveTo(50, 50, 150, 50, 200, 150); // Control1, Control2, End
ctx.stroke();
```

## JavaScript Bezier Calculator

```javascript
// Quadratic Bezier point calculation
function quadraticBezier(t, p0, p1, p2) {
  let mt = 1 - t;
  return {
    x: mt * mt * p0.x + 2 * mt * t * p1.x + t * t * p2.x,
    y: mt * mt * p0.y + 2 * mt * t * p1.y + t * t * p2.y,
  };
}

// Cubic Bezier point calculation
function cubicBezier(t, p0, p1, p2, p3) {
  let mt = 1 - t;
  return {
    x:
      mt * mt * mt * p0.x +
      3 * mt * mt * t * p1.x +
      3 * mt * t * t * p2.x +
      t * t * t * p3.x,
    y:
      mt * mt * mt * p0.y +
      3 * mt * mt * t * p1.y +
      3 * mt * t * t * p2.y +
      t * t * t * p3.y,
  };
}

// Generate curve points
function generateCurve(points, steps = 100) {
  let curve = [];
  for (let i = 0; i <= steps; i++) {
    let t = i / steps;
    curve.push(quadraticBezier(t, points[0], points[1], points[2]));
  }
  return curve;
}
```

## CSS Animation with Bezier Curves

Bezier curves define animation timing in CSS:

```css
/* Built-in timing functions */
.element {
  transition: all 1s ease; /* cubic-bezier(0.25, 0.1, 0.25, 1) */
  transition: all 1s linear; /* cubic-bezier(0, 0, 1, 1) */
  transition: all 1s ease-in; /* cubic-bezier(0.42, 0, 1, 1) */
  transition: all 1s ease-out; /* cubic-bezier(0, 0, 0.58, 1) */
  transition: all 1s ease-in-out; /* cubic-bezier(0.42, 0, 0.58, 1) */
}

/* Custom cubic-bezier */
.element {
  transition: all 1s cubic-bezier(0.68, -0.55, 0.265, 1.55);
}
```

## Key Properties

| Property        | Description                                   |
| --------------- | --------------------------------------------- |
| **Order**       | Number of points minus 1                      |
| **Convex hull** | Curve always inside polygon of control points |
| **Start/End**   | Only first and last points on curve           |
| **Tangents**    | Start tangent = P₁→P₂, End tangent = Pₙ₋₁→Pₙ  |

## Common Use Cases

1. **Vector graphics** - SVG paths, font outlines
2. **Canvas drawing** - `quadraticCurveTo()`, `bezierCurveTo()`
3. **CSS animations** - `cubic-bezier()` timing functions
4. **3D modeling** - Curve and surface generation
5. **Game development** - Smooth movement paths

## Summary

- Bezier curves are defined by **control points**
- **De Casteljau's algorithm** visualizes curve construction
- **Mathematical formulas** provide exact coordinates
- **Properties**: convex hull, only endpoints on curve
- **Applications**: SVG, Canvas, CSS animations, fonts, graphics
- **Complex shapes** created by joining multiple curves
