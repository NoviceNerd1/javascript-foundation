# Custom Elements Summary

Custom elements allow creating new HTML tags with custom behavior, methods, properties, and events using native browser APIs.

## Two Types of Custom Elements

| Type                    | Description              | Base Class                                    |
| ----------------------- | ------------------------ | --------------------------------------------- |
| **Autonomous**          | All-new elements         | `HTMLElement`                                 |
| **Customized Built-in** | Extend existing elements | `HTMLButtonElement`, `HTMLInputElement`, etc. |

## Autonomous Custom Elements

### Basic Structure

```javascript
class MyElement extends HTMLElement {
  constructor() {
    super();
    // Called when element created
  }

  connectedCallback() {
    // Called when element added to DOM
  }

  disconnectedCallback() {
    // Called when element removed from DOM
  }

  static get observedAttributes() {
    return ["attr1", "attr2"]; // Attributes to monitor
  }

  attributeChangedCallback(name, oldValue, newValue) {
    // Called when observed attributes change
  }

  adoptedCallback() {
    // Called when element moved to new document (rare)
  }
}

// Register element (must contain hyphen)
customElements.define("my-element", MyElement);
```

**Rules:**

- Custom element name **must contain a hyphen** (e.g., `my-element`, `time-formatted`)
- Cannot register same element twice

### Example: Time Formatted Element

```javascript
class TimeFormatted extends HTMLElement {
  render() {
    let date = new Date(this.getAttribute("datetime") || Date.now());

    this.innerHTML = new Intl.DateTimeFormat("default", {
      year: this.getAttribute("year") || undefined,
      month: this.getAttribute("month") || undefined,
      day: this.getAttribute("day") || undefined,
      hour: this.getAttribute("hour") || undefined,
      minute: this.getAttribute("minute") || undefined,
      second: this.getAttribute("second") || undefined,
    }).format(date);
  }

  connectedCallback() {
    if (!this.rendered) {
      this.render();
      this.rendered = true;
    }
  }

  static get observedAttributes() {
    return ["datetime", "year", "month", "day", "hour", "minute", "second"];
  }

  attributeChangedCallback() {
    this.render();
  }
}

customElements.define("time-formatted", TimeFormatted);
```

```html
<time-formatted datetime="2019-12-01" year="numeric" month="long" day="numeric">
</time-formatted>
```

### Element Lifecycle Methods

| Method                       | When Called                |
| ---------------------------- | -------------------------- |
| `constructor()`              | Element instance created   |
| `connectedCallback()`        | Added to DOM               |
| `disconnectedCallback()`     | Removed from DOM           |
| `attributeChangedCallback()` | Observed attribute changes |
| `adoptedCallback()`          | Moved to new document      |

## Customized Built-in Elements

Extend existing HTML elements:

```javascript
// Extend built-in button
class HelloButton extends HTMLButtonElement {
  constructor() {
    super();
    this.addEventListener("click", () => alert("Hello!"));
  }
}

// Register with {extends} option
customElements.define("hello-button", HelloButton, { extends: "button" });
```

```html
<!-- Use with is attribute -->
<button is="hello-button">Click me</button>
<button is="hello-button" disabled>Disabled button</button>
```

## Element Upgrade

Elements defined after being parsed are "upgraded":

```javascript
// Element exists before registration
customElements.whenDefined('time-formatted').then(() => {
  console.log('Element is defined');
});

// Check if defined
if (customElements.get('time-formatted')) {
  console.log('Registered');
}

// CSS selectors for state
my-element:not(:defined) { opacity: 0; }  /* Unregistered elements */
my-element:defined { opacity: 1; }         /* Registered elements */
```

## Important Notes

### Render in connectedCallback, Not constructor

```javascript
// ❌ Wrong - attributes not available yet
constructor() {
  super();
  let attr = this.getAttribute('name'); // null
}

// ✅ Correct - render when added to DOM
connectedCallback() {
  this.render();  // Attributes available
}
```

### Children Not Ready in connectedCallback

```javascript
connectedCallback() {
  console.log(this.innerHTML);  // Empty - children not parsed yet

  // Defer to get children
  setTimeout(() => {
    console.log(this.innerHTML);  // Has children
  });
}
```

### Output Order for Nested Elements

```html
<user-info id="outer">
  <user-info id="inner"></user-info>
</user-info>
```

Output order:

1. outer connected
2. inner connected
3. outer initialized (after setTimeout)
4. inner initialized

## Helper Methods

```javascript
// Get element class
let ElementClass = customElements.get("time-formatted");

// Wait for element to be defined
await customElements.whenDefined("time-formatted");

// Check if element is autonomous or customized
customElements.get("hello-button"); // returns class
customElements.get("button"); // returns undefined (built-in)
```

## Task: Live Timer Element

```javascript
class LiveTimer extends HTMLElement {
  constructor() {
    super();
    this.timer = null;
  }

  render() {
    let timeFormatted = this.querySelector("time-formatted");
    if (!timeFormatted) {
      timeFormatted = document.createElement("time-formatted");
      this.appendChild(timeFormatted);
    }

    let now = new Date();
    timeFormatted.setAttribute("datetime", now);
    timeFormatted.setAttribute("hour", "numeric");
    timeFormatted.setAttribute("minute", "numeric");
    timeFormatted.setAttribute("second", "numeric");

    // Dispatch tick event
    this.dispatchEvent(
      new CustomEvent("tick", {
        detail: now,
      }),
    );
  }

  connectedCallback() {
    this.render();
    this.timer = setInterval(() => this.render(), 1000);
  }

  disconnectedCallback() {
    if (this.timer) {
      clearInterval(this.timer);
      this.timer = null;
    }
  }

  static get observedAttributes() {
    return [];
  }
}

customElements.define("live-timer", LiveTimer);
```

```html
<live-timer id="timer"></live-timer>

<script>
  document.getElementById("timer").addEventListener("tick", (event) => {
    console.log("Tick:", event.detail);
  });
</script>
```

## Summary

1. **Autonomous**: `class MyElement extends HTMLElement` + `customElements.define('my-element', MyElement)`
2. **Customized built-in**: `class MyButton extends HTMLButtonElement` + `customElements.define('my-button', MyButton, {extends: 'button'})` + `<button is="my-button">`
3. **Lifecycle**: `connectedCallback`, `disconnectedCallback`, `attributeChangedCallback`
4. **Observe attributes**: Return array from `observedAttributes()`
5. **Render**: In `connectedCallback`, not constructor
6. **Children**: Not ready immediately in `connectedCallback` (use `setTimeout` or custom event)
