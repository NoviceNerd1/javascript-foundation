# ArrayBuffer and Binary Arrays Summary

Binary data in JavaScript is handled through `ArrayBuffer` (raw memory) and view objects that interpret that memory.

## Core Concepts

### ArrayBuffer

Fixed-length contiguous memory area:

```javascript
let buffer = new ArrayBuffer(16); // 16 bytes, zero-filled
console.log(buffer.byteLength); // 16
```

- Cannot be resized
- Cannot directly read/write - needs a view

### TypedArray Views

Views interpret buffer bytes as specific data types:

```javascript
let buffer = new ArrayBuffer(16);

// Different views on same buffer
let uint8 = new Uint8Array(buffer); // each byte = 0-255
let uint16 = new Uint16Array(buffer); // every 2 bytes = 0-65535
let uint32 = new Uint32Array(buffer); // every 4 bytes = 0-4294967295
let float64 = new Float64Array(buffer); // every 8 bytes = floating point
```

## Creating TypedArrays

```javascript
// From ArrayBuffer
let view = new Uint8Array(buffer, 0, 4); // view first 4 bytes

// From array
let arr = new Uint8Array([0, 1, 2, 3]);

// From another TypedArray (converts values)
let arr16 = new Uint16Array([1, 1000]);
let arr8 = new Uint8Array(arr16); // 1000 → 232 (overflow)

// From length
let arr = new Uint16Array(4); // 4 elements (8 bytes)

// Empty
let arr = new Uint8Array();
```

## Available TypedArrays

| Type                | Bytes | Range                     | Description                      |
| ------------------- | ----- | ------------------------- | -------------------------------- |
| `Uint8Array`        | 1     | 0 to 255                  | Unsigned 8-bit                   |
| `Uint8ClampedArray` | 1     | 0 to 255                  | Clamps values (image processing) |
| `Int8Array`         | 1     | -128 to 127               | Signed 8-bit                     |
| `Uint16Array`       | 2     | 0 to 65535                | Unsigned 16-bit                  |
| `Int16Array`        | 2     | -32768 to 32767           | Signed 16-bit                    |
| `Uint32Array`       | 4     | 0 to 4294967295           | Unsigned 32-bit                  |
| `Int32Array`        | 4     | -2147483648 to 2147483647 | Signed 32-bit                    |
| `Float32Array`      | 4     | ~1.2e-38 to 3.4e38        | 32-bit float                     |
| `Float64Array`      | 8     | ~5e-324 to 1.8e308        | 64-bit float                     |

## Overflow Behavior

```javascript
let uint8 = new Uint8Array(1);
uint8[0] = 256; // 100000000 → stores only 00000000
console.log(uint8[0]); // 0

uint8[0] = 257; // 100000001 → stores 00000001
console.log(uint8[0]); // 1

// Uint8ClampedArray clamps instead
let clamped = new Uint8ClampedArray(1);
clamped[0] = 300;
console.log(clamped[0]); // 255
clamped[0] = -10;
console.log(clamped[0]); // 0
```

## TypedArray Methods

```javascript
let arr = new Uint8Array([1, 2, 3, 4, 5]);

// Regular array methods work
arr.map((x) => x * 2); // [2, 4, 6, 8, 10]
arr.reduce((a, b) => a + b); // 15

// Additional methods
arr.set([6, 7], 3); // [1, 2, 3, 6, 7]

let sub = arr.subarray(1, 3); // view of [2, 3] (no copy)
```

**Limitations:**

- No `splice()` (can't delete, only assign zero)
- No `concat()`

## DataView

Flexible view for mixed data types in same buffer:

```javascript
let buffer = new Uint8Array([255, 255, 255, 255]).buffer;
let dv = new DataView(buffer);

console.log(dv.getUint8(0)); // 255 (1 byte)
console.log(dv.getUint16(0)); // 65535 (2 bytes)
console.log(dv.getUint32(0)); // 4294967295 (4 bytes)

dv.setUint32(0, 0); // sets all 4 bytes to 0
```

**DataView methods:**

- `getUint8(offset)`, `getUint16(offset)`, `getUint32(offset)`
- `getInt8(offset)`, `getInt16(offset)`, `getInt32(offset)`
- `getFloat32(offset)`, `getFloat64(offset)`
- Corresponding `set` methods

## Common Terms

- **ArrayBufferView** - umbrella for all typed arrays and DataView
- **BufferSource** - umbrella for ArrayBuffer or ArrayBufferView

## Task: Concatenate Typed Arrays

```javascript
function concat(arrays) {
  // Calculate total length
  let totalLength = arrays.reduce((sum, arr) => sum + arr.length, 0);

  // Create result array
  let result = new Uint8Array(totalLength);

  // Copy each array
  let offset = 0;
  for (let arr of arrays) {
    result.set(arr, offset);
    offset += arr.length;
  }

  return result;
}

// Usage
let arr1 = new Uint8Array([1, 2]);
let arr2 = new Uint8Array([3, 4]);
let arr3 = new Uint8Array([5, 6]);

let combined = concat([arr1, arr2, arr3]);
console.log(combined); // Uint8Array [1, 2, 3, 4, 5, 6]
```

## Cheatsheet

```
ArrayBuffer (raw bytes)
    ↓
    ├── TypedArray (uniform format)
    │    ├── Uint8Array
    │    ├── Uint16Array
    │    ├── Float64Array
    │    └── ...
    │
    └── DataView (flexible format)
         └── getUint8(offset), getUint16(offset), etc.
```
