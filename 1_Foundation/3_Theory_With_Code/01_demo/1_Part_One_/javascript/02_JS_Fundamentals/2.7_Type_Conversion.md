# Type Conversions

Most operators and functions automatically convert values to the right type. We can also explicitly convert values.

> **Note:** This chapter covers primitives only. Objects are covered later in "Object to primitive conversion."

## String Conversion

Occurs when we need the string form of a value (e.g., `alert()`).

```javascript
let value = true;
alert(typeof value); // boolean

value = String(value); // now value is "true"
alert(typeof value); // string
```

**Rules:** `false` → `"false"`, `null` → `"null"`, etc. (mostly obvious)

## Numeric Conversion

Occurs automatically in mathematical operations.

```javascript
alert("6" / "2"); // 3 (strings converted to numbers)
```

**Explicit conversion with `Number(value)`:**

```javascript
let str = "123";
let num = Number(str); // becomes number 123

let age = Number("an arbitrary string"); // NaN (conversion failed)
```

### Numeric Conversion Rules

| Value            | Becomes                                                               |
| ---------------- | --------------------------------------------------------------------- |
| `undefined`      | `NaN`                                                                 |
| `null`           | `0`                                                                   |
| `true` / `false` | `1` / `0`                                                             |
| `string`         | Whitespaces removed from start/end. Empty string → `0`. Error → `NaN` |

```javascript
alert(Number("   123   ")); // 123
alert(Number("123z")); // NaN (error at "z")
alert(Number(true)); // 1
alert(Number(false)); // 0
```

**Important:** `null` becomes `0`, `undefined` becomes `NaN` (they behave differently).

## Boolean Conversion

Occurs in logical operations. Can be explicit with `Boolean(value)`.

```javascript
alert(Boolean(1)); // true
alert(Boolean(0)); // false
alert(Boolean("hello")); // true
alert(Boolean("")); // false
```

### Boolean Conversion Rules

| Value                                                | Becomes |
| ---------------------------------------------------- | ------- |
| `0`, `null`, `undefined`, `NaN`, `""` (empty string) | `false` |
| Any other value                                      | `true`  |

### Important Edge Cases

```javascript
alert(Boolean("0")); // true (non-empty string is true, unlike PHP)
alert(Boolean(" ")); // true (space is non-empty string)
```

## Summary Table

| Conversion  | Method           | Notes                                        |
| ----------- | ---------------- | -------------------------------------------- |
| **String**  | `String(value)`  | Obvious for primitives                       |
| **Number**  | `Number(value)`  | `null`→`0`, `undefined`→`NaN`, `""`→`0`      |
| **Boolean** | `Boolean(value)` | Falsy: `0`, `null`, `undefined`, `NaN`, `""` |

### Common Mistakes

1. `undefined` becomes `NaN` as a number, **not** `0`
2. `"0"` is `true` as a boolean
3. `" "` (space-only string) is `true` as a boolean
