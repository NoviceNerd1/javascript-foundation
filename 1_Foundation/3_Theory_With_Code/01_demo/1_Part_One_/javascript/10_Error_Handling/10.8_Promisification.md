# Promisification

## Overview

**Promisification** is converting a callback-based function into a function that returns a promise. This is useful because promises are more convenient (chaining, async/await).

## Basic Example: Promisifying loadScript

### Callback-based Version

```javascript
function loadScript(src, callback) {
  let script = document.createElement("script");
  script.src = src;

  script.onload = () => callback(null, script);
  script.onerror = () => callback(new Error(`Script load error for ${src}`));

  document.head.append(script);
}

// Usage
loadScript("path/script.js", (err, script) => {
  if (err) {
    // handle error
  } else {
    // use script
  }
});
```

### Manual Promisification

```javascript
let loadScriptPromise = function (src) {
  return new Promise((resolve, reject) => {
    loadScript(src, (err, script) => {
      if (err) reject(err);
      else resolve(script);
    });
  });
};

// Usage
loadScriptPromise("path/script.js")
  .then((script) => {
    /* use script */
  })
  .catch((err) => {
    /* handle error */
  });
```

## Generic Promisify Helper

### Basic Version (Single Result)

```javascript
function promisify(f) {
  return function(...args) {
    return new Promise((resolve, reject) => {
      function callback(err, result) {
        if (err) reject(err);
        else resolve(result);
      }
      args.push(callback);
      f.call(this, ...args);
    });
  };
}

// Usage
let loadScriptPromise = promisify(loadScript);
loadScriptPromise('path/script.js').then(...);
```

### Advanced Version (Multiple Results)

Handles callbacks with multiple arguments like `callback(err, res1, res2, ...)`:

```javascript
function promisify(f, manyArgs = false) {
  return function(...args) {
    return new Promise((resolve, reject) => {
      function callback(err, ...results) {
        if (err) reject(err);
        else resolve(manyArgs ? results : results[0]);
      }
      args.push(callback);
      f.call(this, ...args);
    });
  };
}

// Usage: returns single result
let loadScriptPromise = promisify(loadScript);

// Usage: returns array of all results
let someFunctionPromise = promisify(someFunction, true);
someFunctionPromise(...).then(arrayOfResults => ...);
```

## Important Notes

- **One result only:** Promises can only resolve once. Promisify only works for functions that call the callback **once**. Multiple callback calls are ignored.

- **Callback convention:** Works best with error-first callbacks `(err, result)`. For other formats, manual promisification may be needed.

- **Built-in solutions:** Node.js has `util.promisify`:

```javascript
const util = require("util");
const loadScriptPromise = util.promisify(loadScript);
```

## Summary

| Concept             | Description                                      |
| ------------------- | ------------------------------------------------ |
| **Promisification** | Converting callback functions to return promises |
| **Why**             | Enables promise chaining and async/await         |
| **Convention**      | Error-first callback: `(err, result)`            |
| **Limitation**      | Works only for functions that call callback once |

**Key Points:**

- Promisify wraps a callback function to return a promise
- Use a generic helper to promisify multiple functions
- Node.js has built-in `util.promisify`
- Only suitable for functions that call callback once
- Enables using modern async patterns with legacy callback-based code
