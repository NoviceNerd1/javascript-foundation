# Loops: while and for

## Overview

Loops repeat code multiple times. JavaScript provides `while`, `do...while`, and `for` loops.

## `while` Loop

```javascript
while (condition) {
  // loop body
}

let i = 0;
while (i < 3) {
  alert(i); // 0, 1, 2
  i++;
}
```

### Single-line Body (Curly braces optional)

```javascript
let i = 3;
while (i) alert(i--); // 3, 2, 1
```

## `do...while` Loop

Executes body **at least once**, then checks condition.

```javascript
do {
  // loop body
} while (condition);

let i = 0;
do {
  alert(i);
  i++;
} while (i < 3); // 0, 1, 2
```

## `for` Loop

```javascript
for (begin; condition; step) {
  // loop body
}

for (let i = 0; i < 3; i++) {
  alert(i); // 0, 1, 2
}
```

### Parts Explained

| Part      | Example     | When Executed             |
| --------- | ----------- | ------------------------- |
| begin     | `let i = 0` | Once at loop start        |
| condition | `i < 3`     | Before each iteration     |
| body      | `alert(i)`  | While condition is truthy |
| step      | `i++`       | After each body execution |

### Inline Variable Declaration

```javascript
for (let i = 0; i < 3; i++) {
  alert(i); // 0, 1, 2
}
// alert(i); // Error: i not defined
```

### Skipping Parts

```javascript
let i = 0;
for (; i < 3; i++) {} // begin omitted

for (; i < 3; ) {
  // begin and step omitted
  alert(i++);
}

for (;;) {} // Infinite loop (semicolons required)
```

## `break` Directive

Exits loop immediately.

```javascript
let sum = 0;
while (true) {
  let value = +prompt("Enter a number", "");
  if (!value) break; // Exit on empty input
  sum += value;
}
alert("Sum: " + sum);
```

## `continue` Directive

Skips current iteration, continues with next.

```javascript
for (let i = 0; i < 10; i++) {
  if (i % 2 == 0) continue; // Skip even numbers
  alert(i); // 1, 3, 5, 7, 9
}
```

> **Note:** `break`/`continue` cannot be used in ternary operator `?`.

## Labels for Nested Loops

Label allows breaking out of outer loops.

```javascript
outer: for (let i = 0; i < 3; i++) {
  for (let j = 0; j < 3; j++) {
    let input = prompt(`Value at (${i},${j})`, "");
    if (!input) break outer; // Exits both loops
  }
}
alert("Done!");
```

## Tasks

### 1. Last Loop Value

```javascript
let i = 3;
while (i) {
  alert(i--); // Shows: 3, 2, 1 (stops when i=0)
}
```

### 2. Prefix vs Postfix in `while`

```javascript
// Prefix: shows 1,2,3,4
let i = 0;
while (++i < 5) alert(i);

// Postfix: shows 1,2,3,4,5
let i = 0;
while (i++ < 5) alert(i);
```

### 3. Prefix vs Postfix in `for`

```javascript
// Both show 0,1,2,3,4 (identical)
for (let i = 0; i < 5; i++) alert(i);
for (let i = 0; i < 5; ++i) alert(i);
```

### 4. Output Even Numbers

```javascript
for (let i = 2; i <= 10; i += 2) {
  alert(i);
}
```

### 5. Replace `for` with `while`

```javascript
let i = 0;
while (i < 3) {
  alert(`number ${i}!`);
  i++;
}
```

### 6. Repeat Until Input > 100

```javascript
let num;
do {
  num = prompt("Enter a number greater than 100", 0);
} while (num <= 100 && num !== null);
```

### 7. Output Prime Numbers (2 to n)

```javascript
let n = 10;
outer: for (let i = 2; i <= n; i++) {
  for (let j = 2; j < i; j++) {
    if (i % j == 0) continue outer;
  }
  alert(i); // 2, 3, 5, 7
}
```

## Summary

| Loop         | Description                              |
| ------------ | ---------------------------------------- |
| `while`      | Checks condition before each iteration   |
| `do...while` | Executes at least once, checks after     |
| `for`        | Most common, has begin/condition/step    |
| `break`      | Exits loop entirely                      |
| `continue`   | Skips to next iteration                  |
| `label:`     | Targets specific loop for break/continue |
