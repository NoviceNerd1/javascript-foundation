# Logical Operators in JavaScript

## Overview

Four logical operators: `||` (OR), `&&` (AND), `!` (NOT), `??` (Nullish Coalescing). They work with any data type and return values of any type.

## `||` (OR)

### Boolean Behavior

```javascript
true || true; // true
false || true; // true
true || false; // true
false || false; // false
```

### Finds First Truthy Value

Returns first truthy value; if none, returns last value.

```javascript
alert(1 || 0); // 1
alert(null || 1); // 1
alert(null || 0 || 1); // 1
alert(undefined || null || 0); // 0
```

### Short-circuit Evaluation

Stops at first truthy value; subsequent operands aren't evaluated.

```javascript
true || alert("not printed"); // alert not executed
false || alert("printed"); // alert executes
```

### Practical Usage

```javascript
let firstName = "";
let lastName = "";
let nickName = "SuperCoder";
alert(firstName || lastName || nickName || "Anonymous"); // SuperCoder
```

## `&&` (AND)

### Boolean Behavior

```javascript
true && true; // true
false && true; // false
true && false; // false
false && false; // false
```

### Finds First Falsy Value

Returns first falsy value; if all truthy, returns last value.

```javascript
alert(1 && 0); // 0
alert(1 && 5); // 5
alert(null && 5); // null
alert(1 && 2 && null && 3); // null
alert(1 && 2 && 3); // 3
```

### Precedence

`&&` has higher precedence than `||`:

```javascript
// a && b || c && d is equivalent to:
(a && b) || (c && d);
```

## `!` (NOT)

### Behavior

Converts to boolean and returns inverse.

```javascript
alert(!true); // false
alert(!0); // true
alert(!"hello"); // false
```

### Double NOT `!!` - Convert to Boolean

```javascript
alert(!!"non-empty string"); // true
alert(!!null); // false
// Equivalent to:
alert(Boolean("non-empty string")); // true
```

**Precedence:** `!` has highest precedence among logical operators.

## Tasks

### 1. OR Result

```javascript
alert(null || 2 || undefined); // 2
```

### 2. OR with Alerts

```javascript
alert(alert(1) || 2 || alert(3));
// Shows: 1, then 2
```

### 3. AND Result

```javascript
alert(1 && null && 2); // null
```

### 4. AND with Alerts

```javascript
alert(alert(1) && alert(2));
// Shows: 1, then undefined
```

### 5. OR AND OR

```javascript
alert(null || (2 && 3) || 4); // 3 (&& runs first)
```

### 6. Check Range (Between 14-90)

```javascript
if (age >= 14 && age <= 90) {
  // within range
}
```

### 7. Check Range Outside

```javascript
// With NOT
if (!(age >= 14 && age <= 90)) {
}

// Without NOT
if (age < 14 || age > 90) {
}
```

### 8. Question About "if"

```javascript
if (-1 || 0) alert("first"); // Executes (-1 is truthy)
if (-1 && 0) alert("second"); // Doesn't execute (0 is falsy)
if (null || (-1 && 1)) alert("third"); // Executes (-1 && 1 = 1, truthy)
```

let username = prompt("Who's there?", "");
