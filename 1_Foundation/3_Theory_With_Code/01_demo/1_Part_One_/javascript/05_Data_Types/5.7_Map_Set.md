# Map and Set

## Map

A collection of keyed data items where **keys can be any type** (unlike Objects where keys are strings/symbols).

### Methods and Properties

| Method                | Description                                       |
| --------------------- | ------------------------------------------------- |
| `new Map()`           | Creates a map                                     |
| `map.set(key, value)` | Stores value by key (returns map for chaining)    |
| `map.get(key)`        | Returns value by key, or `undefined` if not found |
| `map.has(key)`        | Returns `true` if key exists                      |
| `map.delete(key)`     | Removes element, returns `true` if existed        |
| `map.clear()`         | Removes all elements                              |
| `map.size`            | Returns element count                             |

### Examples

```javascript
let map = new Map();

// Different key types
map.set("1", "str1"); // string key
map.set(1, "num1"); // number key
map.set(true, "bool1"); // boolean key

alert(map.get(1)); // 'num1'
alert(map.get("1")); // 'str1'
alert(map.size); // 3

// Objects as keys
let john = { name: "John" };
let visitsCountMap = new Map();
visitsCountMap.set(john, 123);
alert(visitsCountMap.get(john)); // 123

// Chaining
map.set("1", "str1").set(1, "num1").set(true, "bool1");
```

### Iteration

```javascript
let recipeMap = new Map([
  ["cucumber", 500],
  ["tomatoes", 350],
  ["onion", 50],
]);

// Iterate over keys
for (let vegetable of recipeMap.keys()) {
  alert(vegetable); // cucumber, tomatoes, onion
}

// Iterate over values
for (let amount of recipeMap.values()) {
  alert(amount); // 500, 350, 50
}

// Iterate over entries
for (let entry of recipeMap) {
  alert(entry); // cucumber,500 etc
}

// forEach
recipeMap.forEach((value, key, map) => {
  alert(`${key}: ${value}`);
});
```

### Object ↔ Map Conversion

```javascript
// Object → Map
let obj = { name: "John", age: 30 };
let map = new Map(Object.entries(obj));
alert(map.get("name")); // John

// Map → Object
let obj2 = Object.fromEntries(map);
alert(obj2.name); // John
```

## Set

A collection of **unique values** (no keys, each value appears only once).

### Methods and Properties

| Method                | Description                                      |
| --------------------- | ------------------------------------------------ |
| `new Set([iterable])` | Creates set, optionally copies values            |
| `set.add(value)`      | Adds value (does nothing if exists), returns set |
| `set.delete(value)`   | Removes value, returns `true` if existed         |
| `set.has(value)`      | Returns `true` if value exists                   |
| `set.clear()`         | Removes all elements                             |
| `set.size`            | Returns element count                            |

### Examples

```javascript
let set = new Set();

let john = { name: "John" };
let pete = { name: "Pete" };
let mary = { name: "Mary" };

set.add(john);
set.add(pete);
set.add(mary);
set.add(john); // Duplicate - ignored
set.add(mary); // Duplicate - ignored

alert(set.size); // 3

for (let user of set) {
  alert(user.name); // John, Pete, Mary
}
```

### Iteration

```javascript
let set = new Set(["oranges", "apples", "bananas"]);

for (let value of set) alert(value);

// forEach (note: value appears twice for Map compatibility)
set.forEach((value, valueAgain, set) => {
  alert(value);
});
```

### Set Iterators

- `set.keys()` – returns values (for Map compatibility)
- `set.values()` – same as `keys()`
- `set.entries()` – returns `[value, value]` pairs

## Comparison

| Feature           | Map                           | Set                    | Object                    |
| ----------------- | ----------------------------- | ---------------------- | ------------------------- |
| Key types         | Any                           | Values (unique)        | Strings/Symbols           |
| Size property     | `map.size`                    | `set.size`             | `Object.keys(obj).length` |
| Iteration order   | Insertion order               | Insertion order        | Not guaranteed            |
| Default iteration | `[key, value]`                | Values                 | Not iterable              |
| Methods           | `get`, `set`, `has`, `delete` | `add`, `has`, `delete` | Property access           |

## Tasks

### 1. Filter Unique Array Members

```javascript
function unique(arr) {
  return Array.from(new Set(arr));
}

let values = [
  "Hare",
  "Krishna",
  "Hare",
  "Krishna",
  "Krishna",
  "Krishna",
  "Hare",
  "Hare",
  ":-O",
];

alert(unique(values)); // Hare, Krishna, :-O
```

### 2. Filter Anagrams

```javascript
function aclean(arr) {
  let map = new Map();

  for (let word of arr) {
    let sorted = word.toLowerCase().split("").sort().join("");
    map.set(sorted, word);
  }

  return Array.from(map.values());
}

let arr = ["nap", "teachers", "cheaters", "PAN", "ear", "era", "hectares"];
alert(aclean(arr)); // nap,teachers,ear or PAN,cheaters,era
```

### 3. Iterable Keys

```javascript
let map = new Map();
map.set("name", "John");

let keys = Array.from(map.keys()); // Convert to array
keys.push("more");
alert(keys); // name, more

// Why map.keys() doesn't work directly:
// map.keys() returns an iterable, not an array
// Use Array.from() to convert to real array
```

## Summary

| Collection | Purpose                            | Key Features                                       |
| ---------- | ---------------------------------- | -------------------------------------------------- |
| **Map**    | Keyed collection with any key type | `set`, `get`, `has`, `size`, insertion order       |
| **Set**    | Collection of unique values        | `add`, `has`, `delete`, `size`, auto-deduplication |

**Key Points:**

- Map keys can be objects, functions, any type
- Set automatically eliminates duplicates
- Both maintain insertion order
- Use `Array.from()` to convert iterables to arrays
- Objects are not suitable for object keys (they convert to `[object Object]`)
