# Destructuring Assignment

## Overview

Destructuring assignment "unpacks" arrays or objects into variables.

## Array Destructuring

### Basic Syntax

```javascript
let arr = ["John", "Smith"];
let [firstName, surname] = arr;
alert(firstName); // John
alert(surname); // Smith

// With split
let [firstName, surname] = "John Smith".split(" ");
```

### Ignore Elements with Commas

```javascript
let [firstName, , title] = ["Julius", "Caesar", "Consul"];
alert(title); // Consul (second element skipped)
```

### Works with Any Iterable

```javascript
let [a, b, c] = "abc"; // ["a", "b", "c"]
let [one, two] = new Set([1, 2]); // [1, 2]
```

### Assign to Any Assignable

```javascript
let user = {};
[user.name, user.surname] = "John Smith".split(" ");
alert(user.name); // John
```

### Loop with `.entries()`

```javascript
let user = { name: "John", age: 30 };
for (let [key, value] of Object.entries(user)) {
  alert(`${key}:${value}`); // name:John, age:30
}
```

### Swap Variables

```javascript
let guest = "Jane",
  admin = "Pete";
[guest, admin] = [admin, guest];
alert(`${guest} ${admin}`); // Pete Jane
```

### Rest Pattern `...`

```javascript
let [name1, name2, ...rest] = ["Julius", "Caesar", "Consul", "of Rome"];
alert(rest[0]); // Consul
alert(rest[1]); // of Rome
```

### Default Values

```javascript
let [name = "Guest", surname = "Anonymous"] = ["Julius"];
alert(name); // Julius
alert(surname); // Anonymous (default used)
```

## Object Destructuring

### Basic Syntax

```javascript
let options = { title: "Menu", width: 100, height: 200 };
let { title, width, height } = options;
alert(title); // Menu
// Order doesn't matter
let { height, width, title } = options; // Same result
```

### Mapping to Different Variable Names

```javascript
let { width: w, height: h, title } = options;
alert(w); // 100
alert(h); // 200
alert(title); // Menu
```

### Default Values

```javascript
let { width = 100, height = 200, title } = { title: "Menu" };
alert(width); // 100 (default)
alert(height); // 200 (default)
```

### Combine Colon and Default

```javascript
let { width: w = 100, height: h = 200, title } = { title: "Menu" };
alert(w); // 100
alert(h); // 200
```

### Rest Pattern

```javascript
let { title, ...rest } = { title: "Menu", height: 200, width: 100 };
alert(title); // Menu
alert(rest.height); // 200
alert(rest.width); // 100
```

### Gotcha: Parentheses Needed Without `let`

```javascript
let title, width, height;
// Error: {title, width, height} = {...} treated as code block
({ title, width, height } = { title: "Menu", width: 200, height: 100 });
alert(title); // Menu
```

### Nested Destructuring

```javascript
let options = {
  size: { width: 100, height: 200 },
  items: ["Cake", "Donut"],
  extra: true,
};

let {
  size: { width, height },
  items: [item1, item2],
  title = "Menu",
} = options;

alert(width); // 100
alert(height); // 200
alert(item1); // Cake
alert(item2); // Donut
alert(title); // Menu (default)
```

## Smart Function Parameters

### Object Parameter Destructuring

```javascript
function showMenu({
  title = "Untitled",
  width = 200,
  height = 100,
  items = [],
}) {
  alert(`${title} ${width} ${height}`);
  alert(items);
}

let options = { title: "My Menu", items: ["Item1", "Item2"] };
showMenu(options);
// My Menu 200 100
// Item1, Item2
```

### With Nested Destructuring

```javascript
function showMenu({
  title = "Untitled",
  width: w = 100,
  height: h = 200,
  items: [item1, item2],
}) {
  alert(`${title} ${w} ${h}`);
  alert(item1);
  alert(item2);
}

showMenu({ title: "My Menu", items: ["Cake", "Donut"] });
```

### Default for Entire Object

```javascript
function showMenu({ title = "Menu", width = 100, height = 200 } = {}) {
  alert(`${title} ${width} ${height}`);
}

showMenu(); // Menu 100 200 (works!)
showMenu({}); // Menu 100 200
```

## Summary

| Context         | Syntax                     |
| --------------- | -------------------------- |
| Array           | `[var1, var2] = array`     |
| Array with rest | `[var1, ...rest] = array`  |
| Array default   | `[var1 = default] = array` |
| Object          | `{prop1, prop2} = obj`     |
| Object rename   | `{prop: var} = obj`        |
| Object default  | `{prop = default} = obj`   |
| Object rest     | `{prop, ...rest} = obj`    |

## Tasks

### 1. Destructuring Assignment

```javascript
let user = { name: "John", years: 30 };

let { name, years: age, isAdmin = false } = user;

alert(name); // John
alert(age); // 30
alert(isAdmin); // false
```

### 2. The Maximal Salary

```javascript
function topSalary(salaries) {
  let maxSalary = 0;
  let maxName = null;

  for (let [name, salary] of Object.entries(salaries)) {
    if (salary > maxSalary) {
      maxSalary = salary;
      maxName = name;
    }
  }

  return maxName;
}

let salaries = { John: 100, Pete: 300, Mary: 250 };
alert(topSalary(salaries)); // Pete
```
