# Objects in JavaScript

## Overview

Objects store keyed collections of data. Unlike primitives (strings, numbers, etc.), objects can hold multiple values as properties.

## Creating Objects

```javascript
let user = new Object(); // Constructor syntax
let user = {}; // Object literal syntax (preferred)
```

## Properties

Properties are `key: value` pairs. Keys are strings (or symbols), values can be any type.

```javascript
let user = {
  name: "John", // key "name", value "John"
  age: 30, // key "age", value 30
};
```

### Accessing Properties

```javascript
// Dot notation
alert(user.name); // John
alert(user.age); // 30

// Square bracket notation (for multiword or dynamic keys)
user["likes birds"] = true;
alert(user["likes birds"]); // true

// With variables
let key = "age";
alert(user[key]); // 30
```

### Adding, Modifying, Deleting

```javascript
user.isAdmin = true; // Add
user.age = 31; // Modify
delete user.age; // Delete
```

### Multiword Property Names

Must be quoted:

```javascript
let user = {
  name: "John",
  "likes birds": true, // Quotes required
};
```

## Computed Properties

Use square brackets for dynamic property names in object literals:

```javascript
let fruit = prompt("Which fruit?", "apple");
let bag = {
  [fruit]: 5, // Property name from variable
};
alert(bag.apple); // 5 if fruit = "apple"

// Expressions also work
let bag = {
  [fruit + "Computers"]: 5, // bag.appleComputers = 5
};
```

## Property Value Shorthand

When variable name matches property name:

```javascript
function makeUser(name, age) {
  return {
    name, // same as name: name
    age, // same as age: age
  };
}
```

## Property Name Limitations

- No reserved word restrictions (`for`, `let`, `return` can be keys)
- Any string or symbol allowed
- Other types auto-convert to strings

```javascript
let obj = {
  for: 1,
  let: 2,
  return: 3,
  0: "test", // becomes "0": "test"
};
alert(obj[0]); // test (number 0 converts to "0")
```

> **Note:** `__proto__` is special; assigning a non-object value is ignored.

## Checking Property Existence

### Using `undefined` comparison

```javascript
let user = {};
alert(user.noSuchProp === undefined); // true (doesn't exist)
```

### Using `in` operator

```javascript
let user = { name: "John", age: 30 };
alert("age" in user); // true
alert("blabla" in user); // false

// With variable
let key = "age";
alert(key in user); // true
```

**Why `in`?** For properties that exist but store `undefined`:

```javascript
let obj = { test: undefined };
alert(obj.test === undefined); // true (misleading)
alert("test" in obj); // true (correct)
```

## Looping with `for...in`

```javascript
let user = {
  name: "John",
  age: 30,
  isAdmin: true,
};

for (let key in user) {
  alert(key); // name, age, isAdmin
  alert(user[key]); // John, 30, true
}
```

## Property Order

- **Integer properties** (strings that convert to integers) are sorted ascending
- **Non-integer properties** appear in creation order

```javascript
let codes = {
  49: "Germany",
  41: "Switzerland",
  1: "USA",
};

for (let code in codes) {
  alert(code); // 1, 41, 49 (sorted!)
}

// Fix: make keys non-integer
let codes = {
  "+49": "Germany",
  "+41": "Switzerland",
  "+1": "USA",
};

for (let code in codes) {
  alert(+code); // 49, 41, 1 (original order)
}
```

## Summary

| Operation       | Syntax                      |
| --------------- | --------------------------- |
| Create object   | `let obj = {};`             |
| Access property | `obj.prop` or `obj["prop"]` |
| Add/update      | `obj.prop = value;`         |
| Delete          | `delete obj.prop;`          |
| Check existence | `"prop" in obj`             |
| Loop            | `for (let key in obj)`      |

## Tasks

### 1. Hello, Object

```javascript
let user = {};
user.name = "John";
user.surname = "Smith";
user.name = "Pete";
delete user.name;
```

### 2. Check for Emptiness

```javascript
function isEmpty(obj) {
  for (let key in obj) {
    return false; // Has at least one property
  }
  return true;
}
```

### 3. Sum Object Properties

```javascript
let salaries = {
  John: 100,
  Ann: 160,
  Pete: 130,
};

let sum = 0;
for (let key in salaries) {
  sum += salaries[key];
}
alert(sum); // 390
```

### 4. Multiply Numeric Properties

```javascript
function multiplyNumeric(obj) {
  for (let key in obj) {
    if (typeof obj[key] === "number") {
      obj[key] *= 2;
    }
  }
}

let menu = {
  width: 200,
  height: 300,
  title: "My menu",
};
multiplyNumeric(menu);
// menu = { width: 400, height: 600, title: "My menu" }
```
