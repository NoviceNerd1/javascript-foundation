# Array Methods

## Add/Remove Items

### `splice` ‚Äì Remove, Insert, Replace

```javascript
let arr = ["I", "study", "JavaScript"];
arr.splice(1, 1); // Remove 1 at index 1 ‚Üí ["I", "JavaScript"]

arr = ["I", "study", "JavaScript", "right", "now"];
arr.splice(0, 3, "Let's", "dance"); // Replace 3 items ‚Üí ["Let's", "dance", "right", "now"]

arr = ["I", "study", "JavaScript"];
arr.splice(2, 0, "complex", "language"); // Insert at index 2 ‚Üí ["I", "study", "complex", "language", "JavaScript"]

// Negative indexes allowed
arr = [1, 2, 5];
arr.splice(-1, 0, 3, 4); // ‚Üí [1, 2, 3, 4, 5]
```

### `slice` ‚Äì Copy Subarray

```javascript
let arr = ["t", "e", "s", "t"];
arr.slice(1, 3); // ["e", "s"]
arr.slice(-2); // ["s", "t"]
arr.slice(); // ["t", "e", "s", "t"] (full copy)
```

### `concat` ‚Äì Merge Arrays/Values

```javascript
let arr = [1, 2];
arr.concat([3, 4]); // [1, 2, 3, 4]
arr.concat([3, 4], [5, 6]); // [1, 2, 3, 4, 5, 6]
arr.concat([3, 4], 5, 6); // [1, 2, 3, 4, 5, 6]
```

## Searching

### `indexOf` / `lastIndexOf` / `includes`

```javascript
let arr = [1, 0, false];
arr.indexOf(0); // 1
arr.indexOf(false); // 2
arr.indexOf(null); // -1
arr.includes(1); // true

let fruits = ["Apple", "Orange", "Apple"];
fruits.indexOf("Apple"); // 0
fruits.lastIndexOf("Apple"); // 2

// includes handles NaN correctly
[NaN].indexOf(NaN); // -1
[NaN].includes(NaN); // true
```

### `find` / `findIndex` / `findLastIndex`

```javascript
let users = [
  { id: 1, name: "John" },
  { id: 2, name: "Pete" },
  { id: 3, name: "Mary" },
];

users.find((item) => item.id == 1); // {id:1, name:"John"}
users.findIndex((item) => item.id == 1); // 0
users.findLastIndex((item) => item.name == "John"); // 0 (searches from right)
```

### `filter` ‚Äì Get All Matching

```javascript
users.filter((item) => item.id < 3); // [{id:1, name:"John"}, {id:2, name:"Pete"}]
```

## Iteration

### `forEach`

```javascript
["Bilbo", "Gandalf"].forEach((item, index, array) => {
  alert(`${item} at ${index} in ${array}`);
});
```

## Transformation

### `map` ‚Äì Transform Each Element

```javascript
let lengths = ["Bilbo", "Gandalf", "Nazgul"].map((item) => item.length);
// [5, 7, 6]
```

### `sort` ‚Äì Sort In-Place

```javascript
let arr = [1, 2, 15];
arr.sort(); // [1, 15, 2] (string comparison!)
arr.sort((a, b) => a - b); // [1, 2, 15] (numeric)

// For strings
let countries = ["√ñsterreich", "Andorra"];
countries.sort((a, b) => a.localeCompare(b)); // Correct language order
```

### `reverse`

```javascript
[1, 2, 3, 4, 5].reverse(); // [5, 4, 3, 2, 1]
```

### `split` / `join`

```javascript
let names = "Bilbo, Gandalf, Nazgul";
let arr = names.split(", "); // ['Bilbo', 'Gandalf', 'Nazgul']
arr.join(";"); // 'Bilbo;Gandalf;Nazgul'
"hello".split(""); // ['h','e','l','l','o']
```

### `reduce` / `reduceRight` ‚Äì Accumulate

```javascript
let arr = [1, 2, 3, 4, 5];
arr.reduce((sum, current) => sum + current, 0); // 15

// Without initial (uses first element as start)
arr.reduce((sum, current) => sum + current); // 15

// Error if empty array without initial
[].reduce((a, b) => a + b); // Error!
```

## Utility

### `Array.isArray`

```javascript
Array.isArray([]); // true
Array.isArray({}); // false
```

### `thisArg` Parameter

```javascript
let army = {
  minAge: 18,
  maxAge: 27,
  canJoin(user) {
    return user.age >= this.minAge && user.age < this.maxAge;
  },
};

let users = [{ age: 20 }, { age: 30 }];
let soldiers = users.filter(army.canJoin, army); // thisArg = army
```

## Tasks

### 1. Camelize

```javascript
function camelize(str) {
  return str
    .split("-")
    .map((word, index) =>
      index == 0 ? word : word[0].toUpperCase() + word.slice(1),
    )
    .join("");
}
```

### 2. Filter Range

```javascript
function filterRange(arr, a, b) {
  return arr.filter((item) => item >= a && item <= b);
}
```

### 3. Filter Range In Place

```javascript
function filterRangeInPlace(arr, a, b) {
  for (let i = 0; i < arr.length; i++) {
    if (arr[i] < a || arr[i] > b) {
      arr.splice(i, 1);
      i--;
    }
  }
}
```

### 6. Extendable Calculator

```javascript
function Calculator() {
  this.methods = {
    "+": (a, b) => a + b,
    "-": (a, b) => a - b,
  };

  this.calculate = function (str) {
    let [a, op, b] = str.split(" ");
    return this.methods[op](+a, +b);
  };

  this.addMethod = function (name, func) {
    this.methods[name] = func;
  };
}
```

### 10. Shuffle Array

```javascript
function shuffle(arr) {
  for (let i = arr.length - 1; i > 0; i--) {
    let j = Math.floor(Math.random() * (i + 1));
    [arr[i], arr[j]] = [arr[j], arr[i]];
  }
}
```

### 11. Get Average Age

```javascript
function getAverageAge(users) {
  return users.reduce((sum, user) => sum + user.age, 0) / users.length;
}
```

### 12. Filter Unique

```javascript
function unique(arr) {
  return arr.filter((item, index) => arr.indexOf(item) === index);
}
```

### 13. Group by Id

```javascript
function groupById(users) {
  return users.reduce((obj, user) => {
    obj[user.id] = user;
    return obj;
  }, {});
}
```
