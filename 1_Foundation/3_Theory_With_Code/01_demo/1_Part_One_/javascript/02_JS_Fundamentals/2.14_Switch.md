# The `switch` Statement

## Overview

`switch` provides a descriptive way to compare a value with multiple variants using strict equality (`===`).

## Syntax

```javascript
switch (x) {
  case "value1":
    // code
    break;
  case "value2":
    // code
    break;
  default:
    // code
    break;
}
```

## Basic Example

```javascript
let a = 2 + 2;

switch (a) {
  case 3:
    alert("Too small");
    break;
  case 4:
    alert("Exactly!"); // Executes
    break;
  case 5:
    alert("Too big");
    break;
  default:
    alert("I don't know such values");
}
```

## Important: Fall-Through Behavior

Without `break`, execution continues to the next case:

```javascript
let a = 2 + 2;

switch (a) {
  case 3:
    alert("Too small");
  case 4:
    alert("Exactly!"); // Executes
  case 5:
    alert("Too big"); // Executes (no break)
  default:
    alert("I don't know such values"); // Executes
}
// Shows: "Exactly!" → "Too big" → "I don't know such values"
```

## Any Expression Allowed

```javascript
let a = "1";
let b = 0;

switch (
  +a // +a converts to number 1
) {
  case b + 1: // 0 + 1 = 1, matches
    alert("This runs");
    break;
}
```

## Grouping Cases

Multiple cases can share the same code:

```javascript
let a = 3;

switch (a) {
  case 4:
    alert("Right!");
    break;
  case 3:
  case 5: // Both 3 and 5 run this code
    alert("Wrong!");
    alert("Take a math class!");
    break;
  default:
    alert("Strange result");
}
```

## Strict Equality (Type Matters)

```javascript
let arg = prompt("Enter a value?"); // User enters 3 → string "3"

switch (arg) {
  case "0":
  case "1":
    alert("One or zero");
    break;
  case "2":
    alert("Two");
    break;
  case 3: // Won't match (string "3" vs number 3)
    alert("Never executes!");
    break;
  default:
    alert("An unknown value"); // Executes
}
```

## Tasks

### 1. Rewrite `switch` to `if`

```javascript
if (browser === "Edge") {
  alert("You've got the Edge!");
} else if (
  browser === "Chrome" ||
  browser === "Firefox" ||
  browser === "Safari" ||
  browser === "Opera"
) {
  alert("Okay we support these browsers too");
} else {
  alert("We hope that this page looks ok!");
}
```

### 2. Rewrite `if` to `switch`

```javascript
let a = +prompt("a?", "");

switch (a) {
  case 0:
    alert(0);
    break;
  case 1:
    alert(1);
    break;
  case 2:
  case 3:
    alert("2,3");
    break;
}
```

## Key Points

| Feature     | Behavior                                                  |
| ----------- | --------------------------------------------------------- |
| Comparison  | Strict equality (`===`)                                   |
| `break`     | Exits switch block; without it, execution "falls through" |
| `default`   | Optional, runs if no case matches                         |
| Grouping    | Multiple `case` lines can share same code                 |
| Expressions | Both `switch` and `case` accept expressions               |
